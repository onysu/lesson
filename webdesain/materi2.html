Dokumen HTML

Semua dokumen HTML harus dimulai dengan deklarasi tipe dokumen: <!DOCTYPE html>. Dokumen HTML itu sendiri dimulai dengan <html> dan diakhiri dengan </html>. Bagian yang terlihat dari dokumen HTML adalah antara <body> dan </body>.

Contoh

<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link href="#" rel="stylesheet">
  <title>Judul Halaman</title>
</head>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
</body>

</html>

Cobalah sendiri

Deklarasi <!DOCTYPE>

deklarasi <!DOCTYPE> mewakili jenis dokumen, dan membantu browser untuk tampilan halaman web dengan benar. Itu hanya boleh muncul sekali, di bagian atas halaman (sebelum tag HTML apa pun).

<!DOCTYPE> deklarasi ini tidak case sensitive.

Deklarasi HTML5 adalah :
<!DOCTYPE html>

Bagaimana Cara Melihat Sumber HTML?

Pernahkah Anda melihat halaman Web dan bertanya-tanya "Hei! Bagaimana mereka melakukannya?"

Lihat Kode Sumber HTML:

Klik kanan di halaman HTML dan pilih "Lihat Sumber Halaman" (di Chrome) atau "Lihat Sumber" (di Edge), atau yang serupa di browser lain. Ini akan membuka jendela yang berisi kode sumber HTML halaman.

Periksa Elemen HTML:

Klik kanan pada elemen (atau area kosong), dan pilih Inspect atau Inspect Element untuk melihat elemen apa yang dibuat (Anda akan melihat HTML dan CSS). Anda juga dapat mengedit HTML atau CSS dengan cepat di panel Elemen atau style yang terbuka.

Elemen HTML

Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir.
Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:

Syntax :
<tagname>Konten berada di sini...</tagname>
Contoh beberapa elemen HTML:
<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>
Mulai tag Konten elemen Tag akhir
<h1> Judul Pertama Saya </h1>
<p> Paragraf pertama saya. </p>
<br> none none

Catatan: Beberapa elemen HTML tidak memiliki konten (seperti elemen <br>). Elemen-elemen ini disebut elemen kosong. Elemen kosong tidak memiliki tag akhir!

Nested HTML Elements

Elemen HTML dapat bersarang (ini berarti elemen dapat berisi elemen lain). Semua dokumen HTML terdiri dari elemen HTML bersarang. Contoh berikut berisi empat elemen HTML ( <html>, <body>, <h1> dan <p>):

Contoh :

<!DOCTYPE html>
<html>
<body>

  <h1>Judul Pertama Saya</h1>
  <p>Paragraf pertama saya.</p>

</body>
</html>

Penjelasannya :

  • <html> adalah elemen root elemen dan mendefinisikan dokumen HTML secara keseluruhan. Ini memiliki tag awal <html> dan tag akhir </html>. Kemudian, di dalam <html> elemen ada <body> elemen:

    <body>

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

    </body>

  • <body> mendefinisikan tubuh dokumen. Ini memiliki tag awal <body> dan tag akhir </body>. Kemudian, di dalam <body> elemen ada dua elemen lainnya: <h1> dan <p>:

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

  • <h1> elemen mendefinisikan heading (judul). Ini memiliki tag awal <h1> dan tag akhir </h1>:

    <h1>Judul Pertama Saya</h1>

  • <p> elemen mendefinisikan sebuah paragraf. Ini memiliki tag awal <p> dan tag akhir </p>:

    <p>Paragraf pertama saya.</p>

Jangan abaikan tag akhir

Beberapa elemen HTML akan ditampilkan dengan benar, bahkan jika Anda lupa tag akhir:
Contoh : 

<html>
<body>

  <p>Ini adalah paragraf
  <p>Ini adalah paragraf

</body>
</html>

Cobalah sendiri

Warning!!!, jangan pernah mengandalkan ini! Hasil dan kesalahan yang tidak terduga dapat terjadi jika Anda lupa tag akhir!

src atribut

<img> tag digunakan untuk menanamkan gambar di halaman HTML. src menspesifikasikan atribut path ke gambar yang akan ditampilkan:

Contoh :

<img src="https://onysu.github.io/lesson/webdesain/files/img/img_girl.jpg">

Cobalah sendiri

Ada dua cara untuk menentukan URL dalam src atribut:

Atribut Keterangan
Absolute URL link ke gambar eksternal yang dihosting di situs web lain. Contoh: src="https://onysu.github.io/lesson/assets/img/icons/html5.png".

Catatan: Gambar eksternal mungkin berada di bawah hak cipta. Jika Anda tidak mendapatkan izin untuk menggunakannya, Anda mungkin melanggar undang-undang hak cipta. Selain itu, Anda tidak dapat mengontrol gambar eksternal; itu bisa tiba-tiba dihapus atau diubah.
Relative URL link ke gambar yang dihosting di dalam situs web. Di sini, URL tidak menyertakan nama domain. Jika URL dimulai tanpa garis miring, itu akan relatif terhadap halaman saat ini. Contoh: src="img_girl.jpg. Jika URL dimulai dengan garis miring, itu akan menjadi relatif terhadap domain. Contoh: src="img_girl.jpg.

Tip: Hampir selalu yang terbaik adalah menggunakan URL relatif. Mereka tidak akan rusak jika Anda mengubah domain.

lebar & tinggi atribut

<img> tag juga harus berisi width dan height atribut, yang menentukan lebar dan tinggi gambar (dalam pixel):

Contoh :

<img src="https://onysu.github.io/lesson/webdesain/files/img/img_girl.jpg" width="225" height="300">

lang atribut

Kode Bahasa

Anda harus selalu menyertakan lang atribut di dalam <html> tag, untuk mendeklarasikan bahasa halaman Web. Ini dimaksudkan untuk membantu mesin pencari dan browser.

Contoh berikut menentukan bahasa Inggris sebagai bahasa:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

Kode Bahasa ISO 639-1 mendefinisikan singkatan untuk bahasa. Dalam HTML dan XHTML mereka dapat digunakan dalam lang dan xml:lang atribut.

Daftar Kode Bahasa

Kode Negara

Kode Negara ISO mendefinisikan singkatan untuk negara. Dalam HTML mereka dapat digunakan sebagai tambahan nilai bahasa di lang atribut. Dua karakter pertama dari kode bahasa mendefinisikan bahasa. dua terakhir mendefinisikan Negara.

Contoh berikut menentukan bahasa Inggris sebagai bahasa dan Amerika Serikat sebagai negara:

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

Kutipan Tunggal atau Ganda?

Tanda kutip ganda   di sekitar nilai atribut adalah yang paling umum dalam HTML, tetapi tanda kutip tunggal '' juga dapat digunakan. Dalam beberapa situasi, ketika nilai atribut itu sendiri berisi tanda kutip ganda, perlu menggunakan tanda kutip tunggal:

Contoh :

Judul HTML didefinisikan dengan tag <h1> - <h6>. <h1> mendefinisikan judul yang paling penting. <h6> mendefinisikan heading yang bisa dibilang tidak penting. 

<h1>Heading 1</h1>

Fungsi Judul / Heading

Mesin pencari menggunakan judul untuk mengindeks struktur dan konten halaman web Anda. Pengguna sering membaca sepintas halaman berdasarkan judulnya. Penting untuk menggunakan heading untuk menunjukkan struktur dokumen. <h1> heading sebaiknya digunakan untuk heading utama, diikuti <h2> heading, kemudian heading yang kurang penting <h2>, dan seterusnya.

Catatan: Gunakan heading HTML untuk heading saja. Jangan gunakan heading untuk membuat teks menjadi BESAR atau tebal.


Judul bisa lebih besar

Setiap heading HTML memiliki ukuran default. Namun, Anda dapat menentukan ukuran untuk setiap heading dengan style atribut tersebut, menggunakan font-size properti CSS :
Contoh :

Heading 1

<h1 style="font-size:60px;">Heading 1</h1>

Contoh :

Teks ini penting!

<i> & <em> elemen

<i> Elemen HTML mendefinisikan bagian teks alternatif. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

Tip: <i> tag sering digunakan untuk menunjukkan istilah teknis, sebuah frase dari bahasa lain, pikiran, nama kapal, dll

Contoh :

Ini teks miring
<i>Ini teks miring</i>

Cobalah sendiri

<em> Elemen HTML mendefinisikan teks yang ditekankan. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

Tip: Pembaca layar akan mengucapkan kata-kata <em> dengan penekanan, menggunakan tekanan verbal.

Contoh :

Ini teks emphasized
<em>Ini teks emphasized</em>

Cobalah sendiri

Contoh :

Ini adalah beberapa teks yang lebih kecil.
<small>Ini adalah beberapa teks yang lebih kecil.</small>

<mark> elemen

<mark> Elemen HTML mendefinisikan teks yang harus ditandai atau disorot:

Contoh :

Jangan lupa beli susu hari ini.

<p>Jangan lupa beli <mark>susu</mark> hari ini.</p>

Cobalah sendiri

<del> elemen

<del> Elemen HTML mendefinisikan teks yang telah dihapus dari dokumen. Peramban biasanya akan menyerang garis melalui teks yang dihapus:

Contoh :

Warna favorit saya adalah biru merah.

<p>Warna favorit saya adalah <del>biru</del> merah.</p>

Cobalah sendiri

<ins> elemen

<ins> Elemen HTML mendefinisikan teks yang telah dimasukkan ke dalam dokumen. Browser biasanya akan menggarisbawahi teks yang disisipkan:

Contoh :

Warna favorit saya adalah biru merah.

<p>Warna favorit saya adalah <del>biru</del> <ins>merah</ins>.</p>

Cobalah sendiri

<sub> elemen

<sub> Elemen HTML mendefinisikan teks subscript. Teks subskrip muncul setengah karakter di bawah garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks subscript dapat digunakan untuk rumus kimia, seperti H2O.

Contoh :

Ini adalah subscripted text.

<p>Ini adalah <sub>subscripted</sub> text.</p>

Cobalah sendiri

<sup> elemen

<sup> Elemen HTML mendefinisikan teks superscript. Teks superskrip muncul setengah karakter di atas garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks superscript dapat digunakan untuk catatan kaki, seperti WWW[1] :

Contoh :

Ini adalah superscripted text.

<p>Ini adalah <sup>superscripted</sup> text.</p>

Quote/Kutipan elemen HTML

Dalam bab ini kita akan membahas elemen <blockquote>, <q>, <abbr>, <address>, <cite> dan <bdo> HTML.

Contoh Blockquote :

Berikut adalah kutipan dari situs WWF:

Selama 50 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka di dunia, WWF bekerja di 100 negara dan didukung oleh 1,2 juta anggota di Amerika Serikat dan hampir 5 juta secara global.

<blockquote> untuk Quote/Kutipan

<blockquote> Elemen HTML mendefinisikan bagian yang dikutip dari sumber lain. Browser biasanya membuat indentasi <blockquote> elemen.

Contoh :

Browser biasanya membuat indentasi elemen blockquote.

Selama hampir 60 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka dunia, WWF bekerja di 100 negara dan didukung oleh lebih dari satu juta anggota di Amerika Serikat dan hampir lima juta di seluruh dunia.
<p>Browser biasanya membuat indentasi elemen blockquote.</p>

<blockquote cite="http://www.worldwildlife.org/who/index.html">Selama hampir 60 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka dunia, WWF bekerja di 100 negara dan didukung oleh lebih dari satu juta anggota di Amerika Serikat dan hampir lima juta di seluruh dunia.</blockquote>

Cobalah sendiri

<q> untuk Kutipan Singkat

<q> tag HTML mendefinisikan kutipan singkat. Browser biasanya menyisipkan tanda kutip di sekitar elemen q.

Contoh :

Tujuan WWF adalah untuk: Membangun masa depan di mana orang hidup selaras dengan alam.

<p>Tujuan WWF adalah untuk: <q>Membangun masa depan di mana orang hidup selaras dengan alam.</q></p>

Cobalah sendiri

<abbr> untuk Singkatan

<abbr> tag HTML mendefinisikan singkatan atau akronim, seperti HTML, CSS, Mr., Dr., ASAP, ATM. Menandai singkatan dapat memberikan informasi yang berguna bagi peramban, sistem terjemahan, dan search-engines.

Tip: Gunakan atribut judul global untuk menampilkan deskripsi singkatan/akronim saat Anda mengarahkan mouse ke elemen.

Contoh :

WHO didirikan pada tahun 1948.
<p><abbr title="World Health Organization">WHO</abbr> didirikan pada tahun 1948.</p>

Cobalah sendiri

<address> untuk Informasi Kontak

<address> Tag HTML mendefinisikan informasi kontak untuk penulis/pemilik dokumen atau artikel. Informasi kontak dapat berupa alamat email, URL, alamat fisik, nomor telepon, pegangan media sosial, dll. Teks dalam <address> elemen biasanya dirender dalam huruf miring, dan browser akan selalu menambahkan jeda baris sebelum dan sesudah <address> elemen.

Contoh :

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
<address>
Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
</address>

Cobalah sendiri

<cite> untuk Judul Karya

<cite> tag HTML mendefinisikan judul karya kreatif (misalnya buku, puisi, lagu, film, lukisan, patung dll.).

Catatan: Nama seseorang bukanlah gelar sebuah karya.

Teks dalam <cite> elemen biasanya dirender dalam huruf miring.

Contoh :

The Scream oleh Edvard Munch. Dilukis pada tahun 1893.
<p><cite>The Scream</cite> oleh Edvard Munch. Dilukis pada tahun 1893.</p>

Cobalah sendiri

<bdo> untuk merubah Arah teks

BDO adalah singkatan dari Bi-Directional Override. <bdo> Tag digunakan untuk mengganti arah teks saat ini.

Contoh :

Teks ini akan ditulis dari kanan ke kiri
<bdo dir="rtl">Teks ini akan ditulis dari kanan ke kiri</bdo>

Cobalah sendiri

Tag Komentar HTML

Komentar HTML tidak ditampilkan di browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda. Anda dapat menambahkan komentar ke sumber HTML Anda dengan menggunakan Syntax berikut:
Perhatikan bahwa ada tanda seru ! di tag awal, tetapi tidak di tag akhir.

Syntax :
<!-- Tulis komentar anda disini -->

Catatan: Komentar tidak ditampilkan oleh browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.


Tambahkan Komentar

Dengan komentar Anda dapat menempatkan pemberitahuan dan pengingat dalam kode HTML Anda.
Contoh :

Ini adalah sebuah paragraf.

<!-- Ini adalah komentar -->
<p>Ini adalah sebuah paragraf.</p>
<!-- Komentar tidak ditampilkan di browser -->

Cobalah sendiri


Sembunyikan Konten

Komentar dapat digunakan untuk menyembunyikan konten. Yang dapat membantu jika Anda menyembunyikan konten untuk sementara.
Contoh :

Ini adalah sebuah paragraf.

Ini juga paragraf.

<p>Ini adalah sebuah paragraf.</p>

<!-- <p>Ini adalah paragraf lain </p> -->

<p>Ini juga paragraf.</p>

Cobalah sendiri


Anda juga dapat menyembunyikan lebih dari satu baris, semua yang ada di antara <!-- dan --> akan disembunyikan dari tampilan.
Contoh :


Sembunyikan bagian kode HTML:

Ini adalah sebuah paragraf.

Ini juga paragraf.

<p>Ini adalah sebuah paragraf.</p>
<!--
<p>Lihat gambar keren ini:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>Ini juga paragraf.</p>

Cobalah sendiri


Komentar juga bagus untuk men-debug HTML, karena Anda dapat mengomentari baris kode HTML, satu per satu, untuk mencari kesalahan.


Sembunyikan Konten Sebaris

Komentar dapat digunakan untuk menyembunyikan bagian di tengah kode HTML.
Contoh :

Sembunyikan bagian dari paragaph:

Ini adalah sebuah paragraf.

<p>Ini <!-- teks yang bagus --> adalah sebuah paragraf.</p>
