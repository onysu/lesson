Semua child langsung dari container grid secara otomatis menjadi item grid.

Grid Column / Kolom

Garis vertikal item grid disebut kolom.

Grid Row / Baris

Garis horizontal item grid disebut row.

Grid Gap/ Jarak

Ruang antara setiap kolom / baris disebut gap.

Anda dapat menyesuaikan ukuran celah dengan menggunakan salah satu dari Properti yang berikut ini :

  • grid-column-gap
  • grid-row-gap
  • grid-gap

Contoh :

grid-column-gap property mengatur celah antara kolom:

Semua Properties Grid CSS

Properti Keterangan
column-gap Menentukan celah antara kolom
gap Properti singkatan untuk celah-baris dan sifat-sifat gap kolom
grid Properti steno untuk baris grid-template, kolom grid-template, area template grid, grid-auto-baris, kolom grid-auto, dan properti aliran-otomatis
grid-area Baik menentukan nama untuk item grid, atau properti ini adalah properti singkat untuk Grid-Bill-Start, Grid-Column-Start, Grid-Row-End, dan Grid-Column-End Properti.
grid-auto-columns Menentukan ukuran kolom default
grid-auto-flow Menentukan bagaimana item yang ditempatkan secara otomatis dimasukkan ke dalam grid
grid-auto-rows Menentukan ukuran baris default
grid-column Properti Singkatan untuk Properti Grid-Column-Marrow dan Grid-Column-End
grid-column-end Menentukan di mana untuk mengakhiri item grid
grid-column-gap Menentukan ukuran celah antara kolom
grid-column-start Menentukan di mana untuk memulai item grid
grid-gap Properti Singkatan untuk Properti Grid-Row-Gap dan Grid-Column-Gap
grid-row Properti Singkatan untuk Start-Row-Start dan properti Grid-Row-End
grid-row-end Menentukan di mana untuk mengakhiri item grid
grid-row-gap Menentukan ukuran celah antara baris
grid-row-start Menentukan di mana untuk memulai item grid
grid-template Properti singkatan untuk baris-templat-grid, kolom grid-template dan Properti area grid
grid-template-areas Menentukan cara menampilkan kolom dan baris, menggunakan item grid bernama
grid-template-columns Menentukan ukuran kolom, dan berapa banyak kolom dalam tata letak grid
grid-template-rows Menentukan ukuran baris dalam tata letak grid
row-gap Menentukan celah antara baris grid
