Tabel HTML - Padding Sel

Cell padding adalah ruang antara tepi sel dan isi sel.

Secara default padding diatur ke 0.

Untuk menambahkan padding pada sel tabel, gunakan paddingproperti CSS :

Contoh :

th, td {
padding: 15px;
}

Cobalah sendiri "

Untuk menambahkan padding hanya di atas konten, gunakan padding-topproperti.

Dan sisi lain dengan padding-bottom, padding-left, dan padding-rightproperti:

Contoh :

th, td {
padding-top: 10px;
padding-bottom: 20px;
padding-left: 30px;
padding-right: 40px;
}

Cobalah sendiri "

Tabel HTML - Spasi Sel

Jarak antar sel adalah jarak antar sel.

Secara default, ruang diatur ke 2 piksel.

Untuk mengubah spasi antar sel tabel, gunakan border-spacingproperti CSS pada table elemen:

Contoh :

table {
border-spacing: 30px;
}

Properti CSS Legal

Hanya ada pilihan properti CSS yang sangat terbatas yang diizinkan untuk digunakan dalam colgroup:

widthproperti
visibilityproperti
backgroundproperti
borderproperti

Semua properti CSS lainnya tidak akan berpengaruh pada tabel Anda.

Colgroup kosong

Jika Anda ingin menata kolom di tengah tabel, sisipkan <col>elemen "kosong" (tanpa gaya) untuk kolom sebelumnya:

Contoh :

<table>
<colgroup>
<col span="3">
<col span="2" style="background-color: pink">
</colgroup>
<tr>
<th>MON</th>
<th>TUE</th>
<th>WED</th>
<th>THU</th>
...

Sembunyikan Kolom

Anda dapat menyembunyikan kolom dengan visibility: collapseproperti:

Contoh :

<table>
<colgroup>
<col span="2">
<col span="3" style="visibility: collapse">
</colgroup>
<tr>
<th>MON</th>
<th>TUE</th>
<th>WED</th>
<th>THU</th>
...
