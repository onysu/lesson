Paste | Paste Link

Pelajari cara berbagi data Excel dengan dokumen Word dan file lainnya.

Paste

Sebagian besar waktu, Anda hanya perlu menempelkan data Excel statis dalam dokumen Word.

1. Pilih data Excel.

2. Klik kanan, lalu klik Copy (atau tekan Ctrl+C).

3. Open dokumen Word.

4. Tekan Ctrl+V.

5. Klik ikon di sudut kiri atas tabel dan tambahkan batas.

Hasil.

Catatan: Untuk menghapus tabel, klik kanan ikon di sudut kiri atas, lalu klik Hapus Tabel.

Paste Link

Anda juga dapat menautkan data sumber di Excel dengan data tujuan di Word. Jika Anda mengubah data di Excel, data di Word diperbarui secara otomatis.

1. Ulangi langkah 1 sampai 3 di atas.

2. Di Word, pada tab Home, di grup Clipboard, klik Paste.

3. Klik Paste Special.

Kotak dialog Paste Special muncul.

4. Klik tautan Paste, Format HTML.

5. Klik Oke.

6. Klik ikon di sudut kiri atas tabel dan tambahkan batas.

Hasil.

7. Ubah data Excel.

Hasil.

Catatan: Di Word, klik kanan ikon di sudut kiri atas tabel, lalu klik Objek Lembar Kerja Tertaut, Tautan untuk meluncurkan kotak dialog Tautan. Di sini, Anda dapat memutuskan tautan, mengubah lokasi file Excel, dll.

Menyematkan (Embed)

Sematkan lembar kerja Excel dalam dokumen Word dan bekerja dengan Excel di Word. Anda juga dapat menyematkan file di Excel.

1. Pilih data Excel.

2. Klik kanan, lalu klik Copy (atau tekan Ctrl+C).

3. Open dokumen Word.

4. Di Word, pada tab Home, di grup Clipboard, klik Paste.

5. Klik Paste Special.

Kotak dialog Paste Special muncul.

6. Klik Paste, Objek Lembar Kerja Microsoft Excel.

7. Klik Oke.

8. Klik dua kali objek untuk bekerja dengannya. Misalnya, format tabel dan sisipkan fungsi SUM.

Catatan: objek yang disematkan adalah bagian dari file Word. Tidak ada tautan ke file Excel asli. Untuk membuat tautan, pada langkah 6, klik Paste Link, Objek Lembar Kerja Microsoft Excel. Sekarang, jika Anda mengklik dua kali objek tersebut, Excel membuka file Excel. Untuk menyematkan file di Excel, pada tab Insert, di grup Text, klik Objek.

PDF

Halaman ini mengajarkan Anda bagaimana untuk mengkonversi sebuah berkas Excel ke PDF berkas.

1. Open buku kerja.

2. Pada tab File, klik Save As.

3. Klik Browse.

4. Pilih PDF dari daftar drop-down.

5. Klik Options.

6. Anda dapat menerbitkan pilihan, lembar aktif atau seluruh buku kerja.

7. Klik OK lalu Klik Save.

Bagikan Buku Kerja (Share Workbooks)

Sorot Changes | Terima/Tolak Changes

Jika Anda berbagi sebuah buku kerja di Excel 2016 atau versi yang lebih tua, Anda dapat bekerja dengan orang lain pada buku kerja yang sama pada waktu yang sama. Buku kerja harus disimpan ke lokasi jaringan di mana orang lain bisa membukanya. Anda dapat melacak perubahan yang dilakukan orang lain dan menerima atau menolak perubahan tersebut.

Untuk berbagi buku kerja, jalankan langkah-langkah berikut.

1. Open buku kerja.

2. Pada tab Review, di grup Changes, klik Share Workbook.

Kotak dialog Share Workbook muncul.

3. Pada tab Editing, klik kotak centang dan klik OK.

Excel menyimpan buku kerja. Anda sekarang dapat bekerja dengan orang lain di buku kerja yang sama secara bersamaan. Layar di atas menunjukkan kepada Anda siapa yang membuka buku kerja.

Soroti Perubahan

Anda dapat menyorot perubahan di layar yang dibuat pengguna lain.

1. Pada tab Review, di grup Changes, klik Track Changes.

2. Klik Highlight Changes.

Kotak dialog Highlight Changes muncul.

3. Klik Oke.

Excel menyoroti perubahan di layar.

Rupanya, Aiden telah mengubah Penjualan Smith dari $16.753,00 menjadi $26.753,00.

Menerima/Menolak Perubahan

Anda sekarang dapat menerima atau menolak perubahan ini.

1. Pada tab Review, di grup Changes, klik Track Changes.

2. Klik Accept/Reject Changes.

3. Klik Oke.

4. Klik Accept untuk menerima perubahan atau Reject untuk menolak perubahan.

Catatan: tidak semua fitur tersedia di buku kerja bersama.

OneDrive

Save file Excel Anda ke lokasi OneDrive Anda dan akses file Anda dari komputer mana pun atau bagikan dengan orang lain.

1. Open buku kerja.

2. Pada tab File, klik Save As.

3. Klik OneDrive, Sign In.

Catatan: jika Anda tidak memiliki akun Microsoft, klik Sign up untuk membuat akun. Jika Anda menggunakan Outlook atau Xbox live, Anda sudah memiliki akun Microsoft.

4. Ketik alamat email Anda dan klik Next.

5. Masukkan password Anda dan klik Sign in.

6. Pilih folder.

7. Klik Save.

Anda sekarang dapat mengedit file Excel ini dengan Excel Online dari mana saja tanpa menginstal Excel di komputer Anda. Untuk berbagi file Excel ini dengan orang lain, jalankan langkah-langkah berikut.

8. Open onedrive.live.com dan masuk dengan akun Microsoft Anda.

9. Klik folder yang dipilih pada langkah 6.

10. Klik kanan file Excel dan klik Share (Bagikan).

11. Masukkan alamat email, tambahkan pesan dan klik Send.

Penerima menerima tautan dan dapat mengedit file Excel. Anda sekarang dapat menggunakan Excel Online untuk bekerja dengan penerima di buku kerja ini secara bersamaan.

On line

Anda dapat mengedit file Excel dengan Excel Online dari mana saja tanpa menginstal Excel di komputer Anda.

1. Pertama, simpan file Excel ke lokasi OneDrive Anda.

2. Open onedrive.live.com dan masuk dengan akun Microsoft Anda.

3. Klik file Excel.

Excel Online membuka buku kerja di browser.

4. Edit file Excel Anda.

Catatan: tidak perlu menyimpan file Anda karena semua perubahan disimpan secara otomatis. Tidak semua fitur tersedia di Excel Online.

Import Access Data

Contoh ini mengajarkan Anda cara mengimpor data dari database Microsoft Access. Di Excel, saat Anda mengimpor data, Anda membuat koneksi permanen yang bisa di-refresh.

1. Pada tab Data, dalam grup Get & Transform Data, klik Get Data.

2. Klik From Database, From Microsoft Access Database.

3. Pilih file Access.

4. Klik Import.

5. Pilih tabel di sisi kiri window Navigator dan klik Load.

Hasil. Catatan database Anda di Excel.

6. Saat data Access Anda berubah, Anda dapat dengan mudah me-refresh data di Excel. Pertama, pilih sel di dalam tabel. Selanjutnya, pada tab Design, dalam grup External Table Data, klik Refresh.

Microsoft Query

Contoh ini mengajarkan Anda cara mengimpor data dari database Microsoft Access dengan menggunakan Microsoft Query Wizard. Dengan Microsoft Query, Anda bisa memilih kolom data yang Anda inginkan dan hanya mengimpor data tersebut ke Excel.

1. Pada tab Data, dalam grup Get & Transform Data, klik Get Data.

2. Klik From Other Sources, Dari Microsoft Query.

Kotak dialog Choose Data Source muncul.

3. Pilih MS Access Database* dan centang Use the Query Wizard to create/edit queries.

4. Klik Oke.

5. Pilih database dan klik OK.

Database Access ini terdiri dari beberapa tabel. Anda dapat memilih tabel dan kolom yang ingin Anda sertakan dalam query Anda.

6. Pilih Customers dan klik simbol >.

7. Klik Next.

Untuk hanya mengimpor kumpulan rekaman tertentu, filter data.

8. Klik City dari daftar Column to filter dan hanya sertakan baris di mana City sama dengan New York.

9. Klik Next.

Anda dapat mengurutkan data Anda jika Anda mau (kami tidak melakukannya di sini).

10. Klik Next.

11. Klik Finish untuk mengembalikan data ke Microsoft Excel.

12. Pilih bagaimana Anda ingin melihat data ini, di mana Anda ingin meletakkannya, dan klik OK.

Hasil:

13. Saat data Access Anda berubah, Anda dapat dengan mudah me-refresh data di Excel. Pertama, pilih sel di dalam tabel. Selanjutnya, pada tab Design, dalam grup External Table Data, klik Refresh.

Impor/Ekspor File Teks

Impor | Ekspor

Artikel ini menjelaskan cara mengimpor atau mengekspor file teks. File teks dapat dipisahkan koma (.csv ) atau dipisahkan tab (.txt )

unduh data-set.xlsx unduh data-set.txt unduh data-set.csv

Import

Untuk mengimpor file teks, jalankan langkah-langkah berikut.

1. Pada tab File, klik Open.

2. Klik Browse.

3. Pilih Text Files dari daftar drop-down.

4a. Untuk mengimpor file.csv, pilih Microsoft Excel Comma Separated Values File dan klik Open. Itu saja.

4b. Untuk mengimpor file.txt, pilih Text Document dan klik Open. Excel meluncurkan Text Import Wizard.

5. Pilih Delimited dan klik Next.

6. Kosongkan semua kotak centang di bawah Pembatas kecuali kotak centang Tab dan klik Next.

7. Klik Finish.

Hasil:

Export

Untuk mengekspor file teks, jalankan langkah-langkah berikut.

1. Open file Excel.

2. Pada tab File, klik Save As.

3. Klik Browse.

4. Pilih Text (Tab delimited) atau CSV (Comma delimited) dari daftar drop-down.

5. Klik Save.

Hasil. file .csv (dipisahkan koma) atau file .txt (dipisahkan tab).

XML

Anda dapat mengonversi file Excel menjadi file data XML, atau sebaliknya. Ini memungkinkan Anda untuk bertukar data antara aplikasi yang berbeda. Pertama, aktifkan Developer Tab.

unduh xml.xlsx unduh schema.xml unduh data-set.xlsx

Berikut data yang ingin kami konversi ke file data XML.

Pertama, kita akan membuat skema berdasarkan data sumber XML. Skema mendefinisikan struktur file XML. Kita harus melakukan ini di luar Excel.

1. Misalnya, buka Notepad dan tambahkan baris berikut.

<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<data-set xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <record>
       <LastName>Smith</LastName>
       <Sales>16753</Sales>
       <Country>UK</Country>
       <Quarter>Qtr 3</Quarter>
    </record>
    <record>
       <LastName>Johnson</LastName>
       <Sales>14808</Sales>
       <Country>USA</Country>
       <Quarter>Qtr 4</Quarter>
    </record>
</data-set>

Catatan: tag merujuk ke nama kolom, tetapi Anda dapat memberi mereka nama apa pun yang Anda inginkan! Misalnya, alih-alih Nama Belakang gunakan Nama.

2. Save file sebagai schema.xml.

3. Open data-set.xlsx.

4. Pada Tab Developer, dalam grup XML, klik Source untuk membuka panel tugas Source XML.

5. Untuk menambahkan peta XML, klik XML Maps.

Kotak dialog XML Maps muncul.

6. Klik Add.

7. Pilih schema.xml dan klik OK dua kali.

8. Sekarang cukup seret (petakan) 4 elemen dari pohon ke lembar kerja (baris 1).

9. Pada Tab Developer, di grup XML, klik Export.

10. Save file Anda sebagai data-set.xml dan tekan Enter.

Hasil:

Itu menghemat banyak waktu!

Catatan: untuk mengimpor file XML, buka buku kerja kosong, pada tab Developer, dalam grup XML, klik Import, lalu pilih file XML.
