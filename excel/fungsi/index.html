Fungsi yang paling banyak digunakan di Excel adalah fungsi count dan sum. Anda dapat menghitung dan menjumlahkan berdasarkan satu kriteria atau beberapa kriteria.

unduh file latihan

Count

Untuk menghitung jumlah sel yang berisi angka, gunakan fungsi COUNT.

Catatan: untuk menghitung sel kosong dan tidak kosong di Excel, gunakan COUNTBLANK dan COUNTA.

Countif

Untuk menghitung sel berdasarkan satu kriteria (misalnya, lebih besar dari 9), gunakan fungsi COUNTIF berikut.

Catatan: kunjungi halaman kami tentang fungsi COUNTIF untuk lebih banyak contoh.

Countifs

Untuk menghitung baris berdasarkan beberapa kriteria (misalnya, hijau dan lebih besar dari 9), gunakan fungsi COUNTIFS berikut.

Sum

Untuk menjumlahkan range sel, gunakan fungsi SUM.

Catatan: kunjungi halaman kami tentang fungsi SUM untuk lebih banyak contoh.

Sumif

Untuk menjumlahkan sel berdasarkan satu kriteria (misalnya, lebih besar dari 9), gunakan fungsi SUMIF berikut (dua argumen).

Untuk menjumlahkan sel berdasarkan satu kriteria (misalnya, hijau), gunakan fungsi SUMIF berikut (tiga argumen, argumen terakhir adalah range yang akan dijumlahkan).

Catatan: kunjungi halaman kami tentang fungsi SUMIF untuk lebih banyak contoh.

Sumifs

Untuk menjumlahkan sel berdasarkan beberapa kriteria (misalnya, lingkaran dan merah), gunakan fungsi SUMIFS berikut (argumen pertama adalah range yang akan dijumlahkan).

Catatan umum: dengan cara yang sama, Anda dapat menggunakan fungsi AVERAGEIF untuk rata-rata sel berdasarkan satu kriteria dan fungsi AVERAGEIFS untuk rata-rata sel berdasarkan beberapa kriteria.

Sumproduct

unduh file latihan

Untuk menghitung jumlah produk dari angka yang sesuai dalam satu atau beberapa range, gunakan fungsi SUMPRODUCT.

Penggunaan Dasar

  1. Misalnya, fungsi SUMPRODUCT di bawah ini menghitung jumlah total yang dibelanjakan.

    Penjelasan: fungsi SUMPRODUCT melakukan perhitungan ini: (2 * 1000) + (4 * 250) + (4 * 100) + (2 * 50) = 3500.

  2. Range harus memiliki dimensi yang sama atau Excel akan menampilkan #VALUE! error.

  3. Fungsi SUMPRODUCT memperlakukan setiap entri yang bukan numerik seolah-olah mereka adalah nol.

  4. Jika Anda menyediakan range tunggal, fungsi SUMPRODUCT menghasilkan hasil yang sama persis dengan fungsi SUM.

Jika Kosong

  1. Fungsi IF di bawah ini mengalikan nilai input dengan 2 jika nilai input tidak sama dengan string kosong (dua tanda kutip ganda tanpa apa-apa di antaranya), selain itu mengembalikan string kosong.

Fungsi ABS di Excel mengembalikan nilai absolut dari sebuah angka. Dengan kata lain: fungsi ABS menghilangkan tanda minus (-) dari angka negatif, menjadikannya positif.

  1. Misalnya, fungsi ABS di sel B1 di bawah ini mengembalikan nilai absolut dari angka negatif.

    Fungsi ABS tidak berpengaruh pada angka 0 (nol) atau positif.

  2. Nilai mutlak 0 adalah 0.

  3. Nilai mutlak suatu bilangan positif adalah bilangan positif yang sama.

    Kapan kita membutuhkan fungsi ABS di Excel?

  4. Misalnya, menghitung kesalahan ramalan (selisih antara nilai aktual dan perkiraan) untuk setiap bulan.

    Catatan: jika kita melihat jumlah kesalahan ini (nol), model perkiraan ini tampaknya sempurna, tetapi tidak!

  5. Cukup gunakan fungsi ABS untuk menghitung kesalahan absolut setiap bulannya.

  6. Jika Anda tidak ingin menampilkan kesalahan perkiraan pada lembar, gunakan SUMPRODUCT dan ABS.

  7. Gunakan fungsi ABS untuk menghitung nilai absolut dari setiap perbedaan.

  8. Tambahkan fungsi IF untuk menguji apakah nilainya berada dalam toleransi.

    Penjelasan: jika selisihnya kurang dari atau sama dengan 3, fungsi IF mengembalikan 1, selain itu mengembalikan 0.

  9. Tambahkan ikon.

  10. Ubah toleransi.

Hari ini dan Sekarang (Today and Now)

  1. Untuk memasukkan tanggal hari ini di Excel, gunakan fungsi TODAY.

    Catatan: fungsi TODAY tidak membutuhkan argumen. Tanggal ini akan diperbarui secara otomatis saat Anda membuka buku kerja pada tanggal lain.

  2. Untuk memasukkan tanggal dan waktu sekarang, gunakan fungsi NOW.

    Catatan: fungsi NOW tidak membutuhkan argumen. Waktu ini akan diperbarui secara otomatis setiap kali lembar dihitung ulang. Ini terjadi saat Anda membuat perubahan ke sel mana pun atau saat Anda membuka buku kerja. Tekan F9 untuk menghitung ulang buku kerja secara manual.

  3. Untuk memasukkan waktu saat ini saja, gunakan NOW()-TODAY() dan terapkan format waktu.

    Catatan: tanggal disimpan sebagai angka di Excel dan menghitung jumlah hari sejak 0, Januari 1900. Waktu ditangani secara internal sebagai angka antara 0 dan 1. Kunjungi halaman kami tentang format tanggal dan waktu untuk informasi selengkapnya.

Tanggal dan Waktu Statis (Static Date and Time)

  1. Untuk memasukkan tanggal saat ini sebagai nilai statis, tekan Ctrl+; (titik koma).

  1. Misalnya, fungsi XLOOKUP di bawah ini tidak dapat menemukan nilai 28 dalam range B3:B9.

  2. Gunakan argumen keempat dari fungsi XLOOKUP untuk mengganti kesalahan #N/A dengan pesan ramah.
