Dokumen HTML

Semua dokumen HTML harus dimulai dengan deklarasi tipe dokumen: <!DOCTYPE html>. Dokumen HTML itu sendiri dimulai dengan <html> dan diakhiri dengan </html>. Bagian yang terlihat dari dokumen HTML adalah antara <body> dan </body>.

Contoh

<!DOCTYPE html>
<html>
<body>

<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>

</body>
</html>

Cobalah sendiri

Deklarasi <!DOCTYPE>

deklarasi <!DOCTYPE> mewakili jenis dokumen, dan membantu browser untuk tampilan halaman web dengan benar. Itu hanya boleh muncul sekali, di bagian atas halaman (sebelum tag HTML apa pun).

<!DOCTYPE> deklarasi ini tidak case sensitive.

Deklarasi HTML5 adalah :
<!DOCTYPE html>

Judul HTML

Judul HTML didefinisikan dengan tag <h1> ~ <h6>. <h1> mendefinisikan judul yang paling penting.
<h6> mendefinisikan judul yang paling tidak penting:

Contoh :

Ini adalah judul 1

Ini adalah judul 2

Ini adalah judul 3

<h1>Ini adalah judul 1</h1>
<h2>Ini adalah judul 2</h2>
<h3>Ini adalah judul 3</h3>

Cobalah sendiri

Paragraf HTML

Paragraf HTML didefinisikan dengan <p> tag.

Contoh :

Ini adalah sebuah paragraf.

Ini adalah paragraf lain.

<p>Ini adalah sebuah paragraf.</p>
<p>Ini adalah paragraf lain.</p>

Link HTML

Link HTML didefinisikan dengan <a> tag.

Contoh :

ini adalah link

Contoh :

Bagaimana Cara Melihat Sumber HTML?

Pernahkah Anda melihat halaman Web dan bertanya-tanya "Hei! Bagaimana mereka melakukannya?"

Lihat Kode Sumber HTML:

Klik kanan di halaman HTML dan pilih "Lihat Sumber Halaman" (di Chrome) atau "Lihat Sumber" (di Edge), atau yang serupa di browser lain. Ini akan membuka jendela yang berisi kode sumber HTML halaman.

Periksa Elemen HTML:

Klik kanan pada elemen (atau area kosong), dan pilih Inspect atau Inspect Element untuk melihat elemen apa yang dibuat (Anda akan melihat HTML dan CSS). Anda juga dapat mengedit HTML atau CSS dengan cepat di panel Elemen atau style yang terbuka.

Elemen HTML

Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir.
Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:

Syntax :
<tagname>Konten berada di sini...</tagname>
Contoh beberapa elemen HTML:
<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>
Mulai tag Konten elemen Tag akhir
<h1> Judul Pertama Saya </h1>
<p> Paragraf pertama saya. </p>
<br> none none

Catatan: Beberapa elemen HTML tidak memiliki konten (seperti elemen <br>). Elemen-elemen ini disebut elemen kosong. Elemen kosong tidak memiliki tag akhir!

Elemen HTML Bersarang

Elemen HTML dapat bersarang (ini berarti elemen dapat berisi elemen lain). Semua dokumen HTML terdiri dari elemen HTML bersarang. Contoh berikut berisi empat elemen HTML ( <html>, <body>, <h1> dan <p>):

Contoh :

<!DOCTYPE html>
<html>
<body>

<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>

</body>
</html>

Penjelasannya :

  • <html> adalah elemen root elemen dan mendefinisikan dokumen HTML secara keseluruhan. Ini memiliki tag awal <html> dan tag akhir </html>. Kemudian, di dalam <html> elemen ada <body> elemen:

    <body>

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

    </body>

  • <body> mendefinisikan tubuh dokumen. Ini memiliki tag awal <body> dan tag akhir </body>. Kemudian, di dalam <body> elemen ada dua elemen lainnya: <h1> dan <p>:

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

  • <h1> elemen mendefinisikan heading (judul). Ini memiliki tag awal <h1> dan tag akhir </h1>:

    <h1>Judul Pertama Saya</h1>

  • <p> elemen mendefinisikan sebuah paragraf. Ini memiliki tag awal <p> dan tag akhir </p>:

    <p>Paragraf pertama saya.</p>

Jangan abaikan Tag Akhir

Beberapa elemen HTML akan ditampilkan dengan benar, bahkan jika Anda lupa tag akhir:
Contoh :

<html>
<body>

<p>Ini adalah paragraf
<p>Ini adalah paragraf

</body>
</html>

Cobalah sendiri

Warning!!!, jangan pernah mengandalkan ini! Hasil dan kesalahan yang tidak terduga dapat terjadi jika Anda lupa tag akhir!

Elemen HTML Kosong

Elemen HTML tanpa konten disebut elemen kosong. <br> tag mendefinisikan satu baris, dan merupakan elemen kosong tanpa tag penutup.
Contoh :

Ini adalah
paragraf dengan jeda baris.

<p>Ini adalah <br> paragraf dengan jeda baris.</p>

Cobalah sendiri

<br> adalah Break atau bisa disebut Line Break, untuk membuat Baris Baru.

HTML tidak sensitif huruf Besar/Kecil

<P> artinya sama dengan <p>. Standar HTML tidak memerlukan tag huruf kecil, kami merekomendasikan huruf kecil dalam HTML, dan menuntut huruf kecil untuk jenis dokumen yang lebih ketat seperti XHTML.

Kami selalu menggunakan nama tag huruf kecil.

Referensi Tag HTML

Referensi tag berisi informasi tambahan tentang tag ini dan atributnya.

Tag Keterangan
<html> Mendefinisikan akar dari dokumen HTML
<body> Mendefinisikan badan dokumen
<h1> ~ <h6> Mendefinisikan judul HTML

Atribut HTML

  1. Semua elemen HTML dapat memiliki attributes
  2. Atribut memberikan informasi tambahan tentang elemen
  3. Atribut selalu ditentukan di tag awal
  4. Atribut biasanya datang dalam pasangan nama/nilai seperti: name="value"

href atribut

<a> tag mendefinisikan hyperlink. The href atribut menentukan link URL untuk membuka sebuah halaman :

Contoh :

Kunjungi O-Learning
<a href="https://onysu.github.io/lesson/webdesain/">Kunjungi O-Learning</a>

Cobalah sendiri

src atribut

<img> tag digunakan untuk menanamkan gambar di halaman HTML. src menspesifikasikan atribut path ke gambar yang akan ditampilkan:

Contoh :

<img src="https://onysu.github.io/lesson/webdesain/files/img/img_girl.jpg">

Cobalah sendiri

Ada dua cara untuk menentukan URL dalam src atribut:

Absolute URL link ke gambar eksternal yang dihosting di situs web lain. Contoh: src="https://onysu.github.io/lesson/assets/img/icons/html5.png".

Catatan: Gambar eksternal mungkin berada di bawah hak cipta. Jika Anda tidak mendapatkan izin untuk menggunakannya, Anda mungkin melanggar undang-undang hak cipta. Selain itu, Anda tidak dapat mengontrol gambar eksternal; itu bisa tiba-tiba dihapus atau diubah.
Relative URL link ke gambar yang dihosting di dalam situs web. Di sini, URL tidak menyertakan nama domain. Jika URL dimulai tanpa garis miring, itu akan relatif terhadap halaman saat ini. Contoh: src="img_girl.jpg. Jika URL dimulai dengan garis miring, itu akan menjadi relatif terhadap domain. Contoh: src="img_girl.jpg.

Tip: Hampir selalu yang terbaik adalah menggunakan URL relatif. Mereka tidak akan rusak jika Anda mengubah domain.

lebar & tinggi atribut

<img> tag juga harus berisi width dan height atribut, yang menentukan lebar dan tinggi gambar (dalam pixel):

Contoh :

<img src="https://onysu.github.io/lesson/webdesain/files/img/img_girl.jpg" width="225" height="300">

Cobalah sendiri

alt atribut

alt Atribut yang diperlukan untuk <img> tag menentukan teks alternatif untuk gambar, jika gambar karena alasan tertentu tidak dapat ditampilkan. Hal ini dapat disebabkan oleh koneksi yang lambat, atau kesalahan pada src atribut, atau jika pengguna menggunakan pembaca layar.

Contoh :

<img src="https://onysu.github.io/lesson/webdesain/files/img/img_girl.jpg" alt="Gadis dengan jaket">

Cobalah sendiri

Lihat apa yang terjadi jika kami mencoba menampilkan gambar yang tidak ada:

<img src="img_typo.jpg" alt="Gadis dengan jaket">

Cobalah sendiri

style atribut

style atribut digunakan untuk menambah gaya untuk elemen, seperti warna, font, ukuran, dan banyak lagi.

Contoh :

Ini adalah paragraf merah.

<p style="color:red;">Ini adalah paragraf merah.</p>

Cobalah sendiri

title atribut

title atribut mendefinisikan beberapa informasi tambahan tentang elemen. Nilai atribut title akan ditampilkan sebagai tooltip saat Anda mengarahkan mouse ke elemen:

Contoh :

Ini adalah sebuah paragraf.

<p title="I'm a tooltip">Ini adalah sebuah paragraf.</p>

Cobalah sendiri

Kami Sarankan: Selalu Gunakan Atribut Huruf Kecil.

Standar HTML tidak memerlukan nama atribut huruf kecil. Atribut title (dan semua atribut lainnya) dapat ditulis dengan huruf besar atau kecil seperti title atau TITLE.
Namun, kami merekomendasikan atribut huruf kecil dalam HTML, dan menuntut atribut huruf kecil untuk jenis dokumen yang lebih ketat seperti XHTML.
Kami selalu menggunakan nama atribut huruf kecil.

Standar HTML tidak memerlukan tanda kutip di sekitar nilai atribut. Namun, kami merekomendasikan kutipan dalam HTML, dan meminta kutipan untuk jenis dokumen yang lebih ketat seperti XHTML.

Bagus:

<a href="https://onysu.github.io/lesson/webdesain/">Kunjungi tutorial HTML kami</a>

Contoh :

Tentang O-Learning

<p title=Tentang O-Learning>

Kutipan Tunggal atau Ganda?

Tanda kutip ganda   di sekitar nilai atribut adalah yang paling umum dalam HTML, tetapi tanda kutip tunggal ' ' juga dapat digunakan. Dalam beberapa situasi, ketika nilai atribut itu sendiri berisi tanda kutip ganda, perlu menggunakan tanda kutip tunggal:

<p title='John "ShotGun" Nelson'>

Atau sebaliknya:

<p title="John 'ShotGun' Nelson">

Cobalah sendiri

Ringkasan

  • Semua elemen HTML dapat memiliki atribut
  • href atribut <a> Menentukan URL halaman link ke alamat website.
  • src atribut <img> menentukan path ke gambar yang akan ditampilkan.
  • width dan height atribut <img> memberikan informasi ukuran untuk gambar.
  • alt atribut <img> menyediakan teks alternatif untuk gambar.
  • style atribut digunakan untuk menambah gaya untuk elemen, seperti warna, font, ukuran dsb.
  • lang atribut dari <html> tag menyatakan bahasa halaman Web.
  • title atribut mendefinisikan beberapa informasi tambahan tentang elemen.

Judul HTML

Judul HTML adalah judul atau subjudul yang ingin Anda tampilkan di halaman web.
Contoh :

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Judul HTML didefinisikan dengan tag <h1> - <h6>. <h1> mendefinisikan judul yang paling penting. <h6> mendefinisikan heading yang bisa dibilang tidak penting.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Cobalah sendiri

Catatan: Browser secara otomatis menambahkan beberapa spasi (margin) sebelum dan sesudah heading.

Fungsi Judul / Heading

Mesin pencari menggunakan judul untuk mengindeks struktur dan konten halaman web Anda. Pengguna sering membaca sepintas halaman berdasarkan judulnya. Penting untuk menggunakan heading untuk menunjukkan struktur dokumen. <h1> heading sebaiknya digunakan untuk heading utama, diikuti <h2> heading, kemudian heading yang kurang penting <h2>, dan seterusnya.

Catatan: Gunakan heading HTML untuk heading saja. Jangan gunakan heading untuk membuat teks menjadi BESAR atau tebal.

Judul bisa lebih besar

Setiap heading HTML memiliki ukuran default. Namun, Anda dapat menentukan ukuran untuk setiap heading dengan style atribut tersebut, menggunakan font-size properti CSS :

Tag Komentar HTML

Komentar HTML tidak ditampilkan di browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda. Anda dapat menambahkan komentar ke sumber HTML Anda dengan menggunakan Syntax berikut:
Perhatikan bahwa ada tanda seru ! di tag awal, tetapi tidak di tag akhir.

Syntax :
<!-- Tulis komentar anda disini -->

Catatan: Komentar tidak ditampilkan oleh browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.

Tambahkan Komentar

Dengan komentar Anda dapat menempatkan pemberitahuan dan pengingat dalam kode HTML Anda.
Contoh :

Ini adalah sebuah paragraf.

<!-- Ini adalah komentar -->
<p>Ini adalah sebuah paragraf.</p>
<!-- Komentar tidak ditampilkan di browser -->

Cobalah sendiri

Sembunyikan Konten

Komentar dapat digunakan untuk menyembunyikan konten. Yang dapat membantu jika Anda menyembunyikan konten untuk sementara.

Ini juga paragraf.

<p>Ini adalah sebuah paragraf.</p>

<!-- <p>Ini adalah paragraf lain </p> -->

<p>Ini juga paragraf.</p>
