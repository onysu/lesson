CSS Grid Item

Child Elemen (Items)

Grid container berisi item grid. Secara default, sebuah wadah/container memiliki satu item kisi untuk setiap kolom, di setiap baris, tetapi Anda dapat menata style item grid sehingga item tersebut akan menjangkau beberapa kolom dan/atau baris.

Properti grid-column

Properti grid-column yang ditentukan pada kolom mana untuk menempatkan item. Anda menentukan di mana item akan dimulai, dan di mana item akan berakhir.

Catatan: Properti grid-column adalah properti singkatan untuk properti grid-column-start dan grid-column-end.

Untuk menempatkan item, Anda dapat merujuk ke nomor baris, atau menggunakan kata kunci "span" untuk menentukan berapa banyak kolom item akan span.

Contoh :
Buat "item1" mulai pada kolom 1 dan berakhir sebelum kolom 5 :
Contoh Code :
1
2
3
4
5
 
<style>
  .item1 {
    grid-column: 1 / 5;
  }
</style>

Properti grid-row

Properti grid-row mendefinisikan pada baris mana untuk menempatkan item. Anda mendefinisikan di mana barang akan dimulai, dan di mana barang akan berakhir.

Catatan: grid-row Property adalah properti kependekan untuk grid-row-start dan grid-row-end properti.

Untuk menempatkan item, Anda dapat merujuk ke nomor baris, atau menggunakan kata kunci "span" untuk menentukan berapa banyak baris item yang akan terbentang:

Buat "item1" mulai pada baris-baris 1 dan berakhir pada baris-baris 4:
Contoh Code : grid-row: 1 / 4
1
2
3
4
5
 
<style>
  .item1 {
    grid-row: 1 / 4;
  }
</style>

Cobalah sendiri

Buat "item1" mulai pada baris 1 dan rentang 2 baris:
Contoh Code : grid-row: 1 / span 2

<style>
  .item1 {
    grid-row: 1 / span 2;
  }
</style>

Properti grid-area

Properti grid-area dapat digunakan sebagai properti untuk grid-start-start, grid-column-start, grid-row-end dan grid-kolom-end.

Contoh :
Buat "Item8" mulai pada baris-line 1 dan kolom-line 2, dan berakhir pada baris baris 5 dan jalur kolom 6:
Contoh Code : grid-area: 1 / 2 / 5 / 6
1
2
3
4
5
 
<style>
  .item8 {
    grid-area: 1 / 2 / 5 / 6;
  }
</style>

Cobalah sendiri

Buat "item8" mulai pada baris-line 2 dan kolom-line 1, dan span 2 baris dan 3 kolom:
Contoh Code : grid-area: 2 / 1 / span 2 / span 3
1
2
3
4
5
 
<style>
  .item8 {
    grid-area: 2 / 1 / span 2 / span 3;
  }
</style>

Penamaan Grid Items

Properti grid-area juga dapat digunakan untuk menetapkan nama ke item grid.

Penamaan item grid dapat dirujuk oleh properti grid-template-areas dari wadah grid.

Item1 mempunyai nama "myArea" dan mencakup semua 5 kolom dalam tata letak grid 5 kolom:
Contoh Code : grid-template-areas: 'myArea myArea myArea myArea myArea'
1
2
3
4
5
6
7
 
<style>
  .item1 {
    grid-area: myArea;
  }
  .grid-container {
    grid-template-areas: 'myArea myArea myArea myArea myArea';
  }
</style>

Cobalah sendiri

Setiap baris didefinisikan oleh apostrof (' ').
Kolom di setiap baris didefinisikan di dalam apostrof, dipisahkan oleh spasi.

Catatan: Tanda titik mewakili item grid tanpa nama.

Biarkan "myArea" menjangkau dua kolom dalam tata letak kotak lima kolom (tanda titik mewakili item tanpa nama):
Contoh Code : grid-template-areas: 'myArea myArea . . .'
1

<style>
  .item1 {
    grid-area: myArea;
  }
  .grid-container {
    grid-template-areas: 'myArea myArea . . .';
  }
</style>

Cobalah sendiri

Untuk mendefinisikan dua baris, tentukan kolom baris kedua di dalam satu set apostrof lainnya:

Buat "item1" menjangkau dua kolom dan dua baris :
Contoh Code : grid-template-areas: 'myArea myArea . . .' 'myArea myArea . . .'
