Apa Itu PHP?

PHP = PHP: Hypertext Preprocessor
→ Bahasa backend untuk membuat website dinamis

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang berjalan di sisi server (server-side).
Dikembangkan oleh Rasmus Lerdorf pada tahun 1994.
Digunakan untuk membuat halaman web dinamis dan interaktif.
File PHP memiliki ekstensi .php.

Kegunaan PHP:

  • Form Login, Register
  • CRUD (Create, Read, Update, Delete)
  • Upload file
  • Sistem Dashboard Admin

Keunggulan PHP:

  • Mudah
  • Banyak contoh di internet
  • Dipakai WordPress, Facebook (awal), dll
  • Bisa berjalan di hosting murah

Bagaimana PHP Bekerja?

Contoh Salah:
❌ double-click index.php di file explorer → langsung buka di browser
       Hasil: kode PHP tidak dijalankan, tampil apa adanya.

Contoh Benar:
✅ letakkan index.php di folder htdocs (XAMPP) atau www (Laragon) → buka di browser via http://localhost/namafolder/index.php
       Hasil: kode PHP dijalankan di server, tampil sesuai output.

Penjelasan:
1. Browser mengirim request ke server (XAMPP/Laragon)
2. Server memproses kode PHP di file .php
3. Server mengirim hasil (HTML/CSS/JS) kembali ke browser
4. Browser menampilkan hasilnya

Jadi, PHP dijalankan di server, bukan di browser. Itulah sebabnya kita perlu menggunakan local server seperti XAMPP atau Laragon untuk menjalankan file PHP.

Ekstensi file PHP selalu .php, bukan .html atau lainnya. Contoh: index.php, biodata.php.

Bagian Utama:

  • Apache = web server
  • PHP = interpreter
  • MySQL = database engine
  • phpMyAdmin = pengelola database via web

INSTALASI XAMPP

  1. Download XAMPP dari apachefriends.org
  2. Jalankan installer, ikuti langkah instalasi
  3. Tentukan lokasi: misalnya C:\xampp\ atau D:\xampp\ tunggu sampai instalasi selesai.

Menjalankan XAMPP Control Panel

  1. Buka XAMPP Control Panel dari menu Start atau dari folder instalasi XAMPP
  2. Di Control Panel, klik tombol "Start" pada module Apache dan MySQL
  3. Pastikan status berubah menjadi "Running" berwarna hijau

Menguji Apache

  1. Buka browser
  2. Ketik http://localhost/ di address bar
  3. Jika muncul halaman dashboard XAMPP, berarti Apache sudah berjalan dengan baik

Mengenal Folder htdocs

Folder htdocs adalah root directory untuk menyimpan file website di XAMPP.
Lokasi default: C:\xampp\htdocs\ atau D:\xampp\htdocs\

Semua file PHP, HTML, CSS, JS yang ingin dijalankan melalui localhost harus disimpan di dalam folder htdocs.

Membuat Project PHP Pertama

  1. Buka folder instalasi XAMPP, biasanya di C:\xampp\htdocs\ atau D:\xampp\htdocs\
  2. Buat folder baru untuk project, misalnya belajarphp
  3. Buka text editor (misalnya Notepad, VSCode)
  4. Buat file baru dengan nama index.php
  5. Isi file dengan kode berikut: 
    <?php
echo "Hello, World!";
?>
  6. Simpan file di dalam folder belajarphp yang sudah dibuat di langkah 2
  7. Buka browser, akses http://localhost/belajarphp/
  8. Jika muncul tulisan "Hello, World!", berarti Apache dan PHP sudah berjalan dengan baik

Penjelasan Tag PHP

Tag pembuka PHP adalah <?php dan tag penutupnya adalah ?>.
Semua kode PHP harus ditulis di antara tag ini.

Contoh: 

<?php
echo "Ini adalah kode PHP";
?>

Output di PHP

Ada dua cara utama untuk menampilkan output di PHP:

  • echo: digunakan untuk menampilkan satu atau lebih string. Contoh: 
    <?php
echo "Halo, Dunia!";
?>
  • print: mirip dengan echo, tetapi hanya dapat menampilkan satu string. Contoh: 
    <?php
print "Selamat datang di PHP!";
?>

Perbedaan Utama: Echo vs Print

  • ✔ A. echo → bisa mencetak beberapa nilai sekaligus 
    <?php
echo "Halo ", "dunia ", "PHP!";
?>
    Output:
    Halo dunia PHP!
        
  • ❌ B. print → tidak bisa mencetak beberapa nilai sekaligus 
    <?php
print "Halo ", "dunia ", "PHP!";
?>
    Output:
    Error: print hanya bisa mencetak satu nilai saja.
        
  • ✔ C. Kecepatan eksekusi:
    • echo → sedikit lebih cepat karena tidak mengembalikan nilai
    • print → sedikit lebih lambat karena mengembalikan nilai 1

Penggunaan dalam kondisi (conditional)

Karena print mengembalikan nilai 1, maka bisa dipakai dalam ekspresi:

<?php
if (print "Hai") {
echo " - Ini dieksekusi";
}
?>

Output

Hai - Ini dieksekusi

echo tidak bisa digunakan seperti ini.

Kesimpulan Sederhana

  • Gunakan echo → untuk menampilkan teks biasa (lebih cepat, fleksibel).
  • Gunakan print → bila kamu butuh nilai balik (jarang diperlukan).
  • Keduanya umum dipakai, jadi pilih sesuai kebutuhan.

Menulis Komentar di PHP

Komentar digunakan untuk memberikan penjelasan pada kode dan tidak akan dieksekusi oleh PHP. Ada tiga cara menulis komentar di PHP:

  • Komentar satu baris menggunakan // atau #: 
    <?php
// Ini adalah komentar satu baris
# Ini juga komentar satu baris
echo "Halo!";
?>
  • Komentar multi-baris menggunakan /* ... */: 
    <?php
/* Ini adalah komentar
   yang terdiri dari
    beberapa baris */
echo "Halo!";
?>

HTML + PHP (Mixed Code)

Kode PHP dapat disisipkan di dalam file HTML. Contoh:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contoh PHP dalam HTML</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <?php
        echo "<p>Ini adalah paragraf yang dihasilkan oleh PHP.</p>";
    ?>
</body>
</html>

Output di browser:

Selamat Datang
Ini adalah paragraf yang dihasilkan oleh PHP.

Error Umum Pemula

  • Lupa menulis tag pembuka <?php atau penutup ?>
  • Lupa menyimpan file dengan ekstensi .php
  • Folder tidak diletakkan di htdocs
  • Salah menulis URL
  • Apache belum aktif
  • Membuka file langsung di browser tanpa melalui localhost
  • Salah ketik pada perintah seperti echo atau print
  • Lupa menambahkan tanda titik koma (;) di akhir perintah

Jika error:

Parse error: syntax error...

Artinya ada kesalahan penulisan.
