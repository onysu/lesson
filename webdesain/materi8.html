Favicon adalah gambar kecil yang ditampilkan di sebelah judul halaman di tab browser. Cara Menambahkan Favicon di HTML. kalian dapat menggunakan gambar apa pun yang kalian suka sebagai favicon kalian. kalian juga dapat membuat favicon sendiri di situs seperti https://favicon.cc.

Tip: Favicon adalah gambar kecil, jadi harus gambar sederhana dengan kontras tinggi.

Gambar favicon ditampilkan di sebelah kiri judul halaman di tab browser, seperti ini:

Untuk menambahkan favicon ke situs web kalian, simpan gambar favicon kalian ke direktori root server web kalian, atau buat Folder di direktori root dengan nama-folder gambar/images/img, dan simpan gambar favicon di folder kalian. Nama umum untuk gambar favicon adalah favicon.ico

Selanjutnya, tambahkan elemen <link> ke file index.html, setelah elemen <title>, seperti ini:

Contoh :

<!doctype html>
<html lang="en">
  
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="icon" type="image/x-icon" href="favicon.ico"> 
  <title>Judul Halaman</title>
</head>

<body>

  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
  
</body>  
</html>

Sekarang, simpan file index.html dan muat ulang di browser kalian. Tab browser kalian sekarang akan menampilkan gambar favicon kalian di sebelah kiri judul halaman.
