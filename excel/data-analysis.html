Anda dapat mengurutkan data Excel Anda pada satu kolom atau beberapa kolom. Anda dapat mengurutkan dalam urutan menaik atau menurun.

Satu Kolom (One Column)

Untuk mengurutkan pada satu kolom, jalankan langkah-langkah berikut.

  1. Klik sel mana saja di kolom yang ingin Anda urutkan.

  2. Untuk mengurutkan dalam urutan menaik , pada tab Data, dalam grup Urutkan & Filter, klik AZ.

    Hasil:

    Catatan : untuk mengurutkan dalam urutan menurun, klik ZA.

Beberapa Kolom (Multiple Columns)

Untuk mengurutkan pada beberapa kolom, jalankan langkah-langkah berikut.

  1. Pada tab Data, di grup Urutkan & Filter, klik Urutkan.

    Kotak dialog Urutkan muncul.

  2. Pilih Nama Belakang dari daftar tarik-turun 'Sort by'.

  3. Klik Tambah Tingkat.

  4. Pilih Penjualan dari daftar tarik-turun 'Then by'.

  5. Klik Oke.

    Hasil. Catatan diurutkan berdasarkan Nama Belakang pertama dan Penjualan kedua.

Urutan Sortir Kustom (Custom Sort Order)

unduh file latihan

Anda dapat menggunakan Excel untuk mengurutkan data dalam urutan khusus. Pada contoh di bawah ini, kami ingin mengurutkan berdasarkan Prioritas (Tinggi, Normal, Rendah).

  1. Klik sel mana saja di dalam kumpulan data.

  2. Pada tab Data, di grup Urutkan & Filter, klik Urutkan.

    Kotak dialog Urutkan muncul.

  3. Pilih Prioritas dari daftar drop-down 'Sort by'.

  4. Pilih Daftar Kustom dari daftar drop-down 'Order'.

    Kotak dialog Daftar Kustom muncul.

  5. Ketik entri daftar.

  6. Klik Oke.

  7. Klik OK lagi.

    Hasil. Rekaman diurutkan berdasarkan Prioritas (Tinggi, Normal, Rendah).

Urutkan berdasarkan Warna (Sort by Color)

unduh file latihan

Contoh ini mengajarkan Anda cara mengurutkan data berdasarkan warna di Excel.

  1. Klik salah satu sel di dalam kumpulan data.

  2. Pada tab Data, di grup Urutkan & Filter, klik Urutkan.

    Kotak dialog Urutkan muncul.

  3. Urutkan berdasarkan Nama Belakang (atau kolom lainnya), urutkan pada Warna Sel (Anda juga dapat mengurutkan pada Warna Font dan Ikon Sel), dan pilih warna hijau untuk tingkat pertama.

  4. Klik 'Copy Level' dua kali dan pilih warna lain.

  5. Klik Oke.

    Hasil.

Daftar Terbalik (Reverse List)

unduh file latihan

Artikel ini mengajarkan Anda bagaimana untuk membalikkan sebuah daftar di Excel . Misalnya, kita ingin membalik daftar di kolom A di bawah ini.

  1. Masukkan nilai 1 ke dalam sel B1 dan nilai 2 ke dalam sel B2.

  2. Pilih rentang B1:B2, klik sudut kanan bawah rentang ini, dan seret ke bawah ke sel B8.

  3. Klik sembarang nomor dalam daftar di kolom B.

  4. Untuk mengurutkan dalam urutan menurun, pada tab Data, dalam grup Sort & Filter, klik ZA.

    Hasil. Tidak hanya daftar di kolom B, tetapi juga daftar di kolom A telah dibalik.

Daftar Acak (Randomize List)

unduh file latihan

Artikel ini mengajarkan Anda cara mengacak (mengacak) daftar di Excel . Misalnya, kami ingin mengacak daftar di kolom A di bawah ini.

  1. Pilih sel B1 dan masukkan fungsi RAND().

  2. Klik di sudut kanan bawah sel B1 dan seret ke bawah ke sel B8.

  3. Klik sembarang nomor dalam daftar di kolom B.

  4. Untuk mengurutkan dalam urutan menurun, pada tab Data, dalam grup Sort & Filter, klik ZA.

    Hasil. Daftar acak di kolom A (diurutkan berdasarkan angka acak di atas).

    Catatan : angka acak berubah setiap kali sel pada lembar dihitung. Jika Anda tidak menginginkan ini, cukup salin angka acak dan tempel sebagai nilai.

Fungsi SORT (SORT function)

unduh file latihan

Gunakan fungsi SORT ajaib di Excel 365 untuk mengurutkan data Excel Anda pada satu kolom atau beberapa kolom. Mari kita coba.

  1. Fungsi SORT sederhana di bawah ini hanya memiliki 1 argumen (A2:D15). Secara default, fungsi SORT mengurutkan berdasarkan kolom pertama, dalam urutan menaik.

    Catatan : fungsi larik dinamis ini, dimasukkan ke dalam sel F2, mengisi beberapa sel. Wow! Perilaku ini di Excel 365 disebut menumpahkan.

  2. Gunakan argumen kedua dari fungsi SORT untuk mengurutkan menurut kolom yang berbeda. Fungsi SORT di bawah ini mengurutkan berdasarkan kolom kedua.

    Catatan : hanya edit rumus di sel F2. Excel akan melakukan sisanya.

  3. Gunakan argumen ketiga dari fungsi SORT untuk mengurutkan dalam urutan yang berbeda. Fungsi SORT di bawah ini mengurutkan berdasarkan kolom pertama, dalam urutan menurun.

    Catatan : gunakan 1 untuk mengurutkan dalam urutan menaik, gunakan -1 untuk mengurutkan dalam urutan menurun.

  4. Untuk mengurutkan pada beberapa kolom, cukup berikan konstanta array. Misalnya, urutkan berdasarkan Nama Belakang pertama dan Penjualan kedua.

  5. Terakhir, fungsi SORT lanjutan di bawah ini mengurutkan berdasarkan Nama Belakang terlebih dahulu (dalam urutan menaik) dan berdasarkan Penjualan kedua (dalam urutan menurun). Cukup keren, ya?
