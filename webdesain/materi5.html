Apa itu CSS?

Cascading Style Sheets (CSS) digunakan untuk memformat tata letak halaman web. Dengan CSS, Anda dapat mengontrol warna, font, ukuran teks, jarak antar elemen, bagaimana elemen diposisikan dan ditata, gambar latar belakang atau warna latar apa yang akan digunakan, tampilan berbeda untuk perangkat dan ukuran layar yang berbeda, dan lebih banyak!

Tip: Kata cascading berarti bahwa style yang diterapkan ke elemen induk juga akan diterapkan ke semua elemen turunan di dalam induk. Jadi, jika Anda mengatur warna teks isi ke blue, semua judul, paragraf, dan elemen teks lain di dalam badan juga akan mendapatkan warna yang sama (kecuali jika Anda menentukan yang lain)!.

Menggunakan CSS

CSS dapat ditambahkan ke dokumen HTML dengan 3 cara:

  • Inline - dengan menggunakan <style> atribut di dalam elemen HTML
  • Internal - dengan menggunakan <style> elemen di bagian <head>
  • Eksternal - dengan menggunakan <link> elemen untuk menautkan ke file CSS eksternal.

Cara paling umum untuk menambahkan CSS, adalah dengan menyimpan <style> di file CSS eksternal. Namun, dalam tutorial ini kita akan menggunakan <style> inline dan internal, karena ini lebih mudah untuk didemonstrasikan, dan lebih mudah bagi Anda untuk mencobanya sendiri.

Inline CSS / CSS Sebaris

CSS sebaris digunakan untuk menerapkan <style> unik ke satu elemen HTML. CSS sebaris menggunakan <style> atribut elemen HTML.
Contoh berikut menyetel warna teks <h1> elemen menjadi biru, dan warna teks <p> elemen menjadi merah:

Contoh :

Judul Biru

Sebuah paragraf merah.

<h1 style="color:blue;">Judul Biru</h1>
<p style="color:red;">Sebuah paragraf merah.</p>

Internal CSS

Internal CSS digunakan untuk menentukan style untuk satu halaman HTML. Internal CSS didefinisikan di <head> bagian halaman HTML, di dalam <style> elemen. Contoh : berikut menyetel warna teks SEMUA <h1> elemen (pada halaman itu) menjadi biru, dan warna teks SEMUA <p> elemen menjadi merah. Selain itu, halaman akan ditampilkan dengan warna latar belakang orange.

Contoh :

Ini adalah judul

Ini adalah sebuah paragraf.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: orange;
    }
    h1 {
      color: blue;
    }
    p {
      color: red;
    }
  </style>  
</head>
<body>
  <h1>Ini adalah judul</h1>
  <p>Ini adalah sebuah paragraf.</p>
</body>
</html>

External CSS

Lembar style eksternal digunakan untuk menentukan style untuk banyak halaman HTML. Untuk menggunakan lembar style eksternal, tambahkan tautan ke dalamnya di <head> bagian setiap halaman HTML.

Latihan :

ada 2 file :

  1. index.html
  2. style.css

index.html

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <h1>Ini adalah judul</h1>
  <p>Ini adalah sebuah paragraf.</p>
</body>
</html>

Cobalah sendiri

Lembar style eksternal dapat ditulis dalam editor teks apa pun. File tidak boleh berisi kode HTML apa pun, dan harus disimpan dengan ekstensi.css. Berikut adalah tampilan file styles.css:

styles.css


body {
  background-color: orange;
}
h1 {
  color: blue;
}
p {
  color: red;
}

Tips: Dengan lembar style eksternal, Anda dapat mengubah tampilan seluruh situs web, dengan mengubah satu file!

Warna, Font & Ukuran CSS

Di sini, kami akan mendemonstrasikan beberapa properti CSS yang umum digunakan. Anda akan belajar lebih banyak tentang mereka nanti.
Properti Keterangan
color properti CSS mendefinisikan warna teks yang akan digunakan.
font-family properti CSS mendefinisikan font yang akan digunakan.
font-size properti CSS mendefinisikan ukuran teks yang akan digunakan.

Contoh :

Penggunaan warna CSS, font-family dan properti font-size:

Ini adalah judul

Ini adalah sebuah paragraf.

<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: blue;
      font-family: verdana;
      font-size: 300%;
    }
    p {
      color: red;
      font-family: courier;
      font-size: 160%;
    }
  </style>
</head>
<body>
  <h1>Ini adalah judul</h1>
  <p>Ini adalah sebuah paragraf.</p>
</body>
</html>

Link ke CSS Eksternal

Style eksternal dapat direferensikan dengan URL lengkap atau dengan jalur relatif ke halaman web saat ini.
Contoh ini menggunakan URL lengkap untuk menautkan ke lembar style:

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css">
</head>
<body>
  <h1> This is a heading</h1>
  <p> This is a paragraph.</p>
</body>
</html>

Contoh ini menautkan style yang terletak di situs web lain:

Ringkasan Bab

  • Gunakan style atribut HTML untuk penataan sebaris.
  • Gunakan <style> elemen HTML untuk mendefinisikan CSS internal.
  • Gunakan <link> elemen HTML untuk merujuk ke file CSS eksternal.
  • Gunakan <head> elemen HTML untuk menyimpan elemen <style> dan <link>.
  • Gunakan color properti CSS untuk warna teks.
  • Gunakan font-family properti CSS untuk font teks.
  • Gunakan font-size properti CSS untuk ukuran teks.
  • Gunakan border properti CSS untuk perbatasan.
  • Gunakan padding properti CSS untuk ruang di dalam perbatasan.
  • Gunakan margin properti CSS untuk ruang di luar perbatasan.
