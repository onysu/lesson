Style HTML

style Atribut HTML digunakan untuk menambahkan gaya ke elemen, seperti warna, font, ukuran dan lainnya.

Contoh :

Teks normal

Teks merah

Teks biru

Teks besar

<p>Teks normal</p>
<p style="color:red;">Teks merah</p>
<p style="color:blue;">Teks biru</p>
<p style="font-size:50px;">Teks besar</p>

Cobalah sendiri

Atribut Style HTML

Pengaturan style suatu elemen HTML, dapat dilakukan dengan style atribut. style Atribut HTML memiliki Syntax berikut:

Syntax :
<tagname style="property:value;">
property Properti CSS.
value Nilai CSS.

Anda akan belajar lebih banyak tentang CSS nanti dalam tutorial ini.

Pemformatan Teks HTML

HTML berisi beberapa elemen untuk mendefinisikan teks dengan arti khusus.

Contoh :

Ini teks tebal

Ini teks miring

Ini adalah subscript dan superscript

Elemen Pemformatan HTML

Elemen pemformatan dirancang untuk menampilkan jenis teks khusus:

Elemen Keterangan
<b> Teks tebal
<strong> Teks penting
<i> Teks miring
<em> Teks yang ditekankan
<mark> Teks yang ditandai
<small> Teks yang lebih kecil
<del> Teks yang dihapus
<ins> Teks yang disisipkan
<sub> Teks subskrip
<sup> Teks superskrip

<b> & <strong> elemen

<b> Elemen HTML mendefinisikan teks tebal, tanpa kepentingan ekstra.

Contoh :

Ini teks tebal
<b>Ini teks tebal</b>
Cobalah sendiri

<strong> Elemen HTML mendefinisikan teks dengan sangat penting. Konten di dalamnya biasanya ditampilkan dalam huruf tebal.

Contoh :

Teks ini penting!
<strong>Teks ini penting!</strong>
Cobalah sendiri

<i> & <em> elemen

<i> Elemen HTML mendefinisikan bagian teks alternatif. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

Tip: <i> tag sering digunakan untuk menunjukkan istilah teknis, sebuah frase dari bahasa lain, pikiran, nama kapal, dll

Contoh :

Ini teks miring
<i>Ini teks miring</i>
Cobalah sendiri

<em> Elemen HTML mendefinisikan teks yang ditekankan. Konten di dalamnya biasanya ditampilkan dalam huruf miring.

Tip: Pembaca layar akan mengucapkan kata-kata <em> dengan penekanan, menggunakan tekanan verbal.

Contoh :

Ini teks emphasized
<em>Ini teks emphasized</em>
Cobalah sendiri

<small> elemen

<small> Elemen HTML mendefinisikan teks yang lebih kecil:

Contoh :

Ini adalah beberapa teks yang lebih kecil.
<small>Ini adalah beberapa teks yang lebih kecil.</small>
Cobalah sendiri

<mark> elemen

<mark> Elemen HTML mendefinisikan teks yang harus ditandai atau disorot:

Contoh :

Jangan lupa beli susu hari ini.

<p>Jangan lupa beli <mark>susu</mark> hari ini.</p>
Cobalah sendiri

<del> elemen

<del> Elemen HTML mendefinisikan teks yang telah dihapus dari dokumen. Peramban biasanya akan menyerang garis melalui teks yang dihapus:

Contoh :

Warna favorit saya adalah biru merah.

<p>Warna favorit saya adalah <del>biru</del> merah.</p>
Cobalah sendiri

<ins> elemen

<ins> Elemen HTML mendefinisikan teks yang telah dimasukkan ke dalam dokumen. Browser biasanya akan menggarisbawahi teks yang disisipkan:

Contoh :

Warna favorit saya adalah biru merah.

<p>Warna favorit saya adalah <del>biru</del> <ins>merah</ins>.</p>
Cobalah sendiri

<sub> elemen

<sub> Elemen HTML mendefinisikan teks subscript. Teks subskrip muncul setengah karakter di bawah garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks subscript dapat digunakan untuk rumus kimia, seperti H2O.

Contoh :

Ini adalah subscripted text.

<p>Ini adalah <sub>subscripted</sub> text.</p>
Cobalah sendiri

<sup> elemen

<sup> Elemen HTML mendefinisikan teks superscript. Teks superskrip muncul setengah karakter di atas garis normal, dan terkadang ditampilkan dalam font yang lebih kecil. Teks superscript dapat digunakan untuk catatan kaki, seperti WWW[1] :

Contoh :

Ini adalah superscripted text.

<p>Ini adalah <sup>superscripted</sup> text.</p>
Cobalah sendiri

Quote/Kutipan elemen HTML

Dalam bab ini kita akan membahas elemen <blockquote>, <q>, <abbr>, <address>, <cite> dan <bdo> HTML.

Contoh Blockquote :

Berikut adalah kutipan dari situs WWF:

Selama 50 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka di dunia, WWF bekerja di 100 negara dan didukung oleh 1,2 juta anggota di Amerika Serikat dan hampir 5 juta secara global.

<blockquote> untuk Quote/Kutipan

<blockquote> Elemen HTML mendefinisikan bagian yang dikutip dari sumber lain. Browser biasanya membuat indentasi <blockquote> elemen.

Contoh :

Browser biasanya membuat indentasi elemen blockquote.

Selama hampir 60 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka dunia, WWF bekerja di 100 negara dan didukung oleh lebih dari satu juta anggota di Amerika Serikat dan hampir lima juta di seluruh dunia.
<p>Browser biasanya membuat indentasi elemen blockquote.</p>

<blockquote cite="http://www.worldwildlife.org/who/index.html">Selama hampir 60 tahun, WWF telah melindungi masa depan alam. Organisasi konservasi terkemuka dunia, WWF bekerja di 100 negara dan didukung oleh lebih dari satu juta anggota di Amerika Serikat dan hampir lima juta di seluruh dunia.</blockquote>
Cobalah sendiri

<q> untuk Kutipan Singkat

<q> tag HTML mendefinisikan kutipan singkat. Browser biasanya menyisipkan tanda kutip di sekitar elemen q.

Contoh :

Tujuan WWF adalah untuk: Membangun masa depan di mana orang hidup selaras dengan alam.

<p>Tujuan WWF adalah untuk: <q>Membangun masa depan di mana orang hidup selaras dengan alam.</q></p>
Cobalah sendiri

<abbr> untuk Singkatan

<abbr> tag HTML mendefinisikan singkatan atau akronim, seperti HTML, CSS, Mr., Dr., ASAP, ATM. Menandai singkatan dapat memberikan informasi yang berguna bagi peramban, sistem terjemahan, dan search-engines.

Tip: Gunakan atribut judul global untuk menampilkan deskripsi singkatan/akronim saat Anda mengarahkan mouse ke elemen.

Contoh :

WHO didirikan pada tahun 1948.
<p><abbr title="World Health Organization">WHO</abbr> didirikan pada tahun 1948.</p>
Cobalah sendiri

<address> untuk Informasi Kontak

<address> Tag HTML mendefinisikan informasi kontak untuk penulis/pemilik dokumen atau artikel. Informasi kontak dapat berupa alamat email, URL, alamat fisik, nomor telepon, pegangan media sosial, dll. Teks dalam <address> elemen biasanya dirender dalam huruf miring, dan browser akan selalu menambahkan jeda baris sebelum dan sesudah <address> elemen.

Contoh :

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
<address>
Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
</address>
Cobalah sendiri

<cite> untuk Judul Karya

<cite> tag HTML mendefinisikan judul karya kreatif (misalnya buku, puisi, lagu, film, lukisan, patung dll.).

Catatan: Nama seseorang bukanlah gelar sebuah karya.

Teks dalam <cite> elemen biasanya dirender dalam huruf miring.

Contoh :

The Scream oleh Edvard Munch. Dilukis pada tahun 1893.
<p><cite>The Scream</cite> oleh Edvard Munch. Dilukis pada tahun 1893.</p>
Cobalah sendiri

<bdo> untuk merubah Arah teks

BDO adalah singkatan dari Bi-Directional Override. <bdo> Tag digunakan untuk mengganti arah teks saat ini.

Contoh :

Teks ini akan ditulis dari kanan ke kiri
<bdo dir="rtl">Teks ini akan ditulis dari kanan ke kiri</bdo>
Cobalah sendiri

Tag Komentar HTML

Komentar HTML tidak ditampilkan di browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda. Anda dapat menambahkan komentar ke sumber HTML Anda dengan menggunakan Syntax berikut:
Perhatikan bahwa ada tanda seru ! di tag awal, tetapi tidak di tag akhir.

Syntax :
<!-- Tulis komentar anda disini -->

Catatan: Komentar tidak ditampilkan oleh browser, tetapi dapat membantu mendokumentasikan kode sumber HTML Anda.

Tambahkan Komentar

Dengan komentar Anda dapat menempatkan pemberitahuan dan pengingat dalam kode HTML Anda.
Contoh :

Ini adalah sebuah paragraf.

<!-- Ini adalah komentar -->
<p>Ini adalah sebuah paragraf.</p>
<!-- Komentar tidak ditampilkan di browser -->
Cobalah sendiri

Sembunyikan Konten

Komentar dapat digunakan untuk menyembunyikan konten. Yang dapat membantu jika Anda menyembunyikan konten untuk sementara.
Contoh :

Ini adalah sebuah paragraf.

Ini juga paragraf.

<p>Ini adalah sebuah paragraf.</p>

<!-- <p>Ini adalah paragraf lain </p> -->

<p>Ini juga paragraf.</p>
Cobalah sendiri

Anda juga dapat menyembunyikan lebih dari satu baris, semua yang ada di antara <!-- dan --> akan disembunyikan dari tampilan.
Contoh :

Sembunyikan bagian kode HTML:

Ini adalah sebuah paragraf.

Ini juga paragraf.

<p>Ini adalah sebuah paragraf.</p>
<!--
<p>Lihat gambar keren ini:</p>
<img border="0" src="pic_trulli.jpg" alt="Trulli">
-->
<p>Ini juga paragraf.</p>
Cobalah sendiri

Komentar juga bagus untuk men-debug HTML, karena Anda dapat mengomentari baris kode HTML, satu per satu, untuk mencari kesalahan.

Sembunyikan Konten Sebaris

Komentar dapat digunakan untuk menyembunyikan bagian di tengah kode HTML.
Contoh :

Sembunyikan bagian dari paragaph:

Ini adalah sebuah paragraf.

<p>Ini <!-- teks yang bagus --> adalah sebuah paragraf.</p>
