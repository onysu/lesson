Rumus Array

 

Tanpa Rumus Array | Dengan Rumus Array | Kunci F9

Bab ini membantu Anda memahami rumus array di Excel . Rumus array sel tunggal melakukan beberapa perhitungan dalam satu sel.

Tanpa Rumus Array

Tanpa menggunakan rumus array, kami akan menjalankan langkah-langkah berikut untuk menemukan kemajuan terbesar.

1. Pertama, kita akan menghitung kemajuan setiap siswa.

2. Selanjutnya, kita akan menggunakan fungsi MAX untuk menemukan kemajuan terbesar.

Dengan Rumus Array

Kita tidak perlu menyimpan range di kolom D. Excel bisa menyimpan range ini di memorinya. Rentang yang disimpan dalam memori Excel disebut konstanta array .

1. Kita sudah mengetahui bahwa kita dapat mencari kemajuan siswa pertama dengan menggunakan rumus di bawah ini.

2. Untuk mencari progres terbesar (jangan kewalahan), kita tambahkan fungsi MAX, ganti C2 dengan C2:C6 dan B2 dengan B2:B6.

3. Selesai dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) disimpan dalam memori Excel, bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{19;33;63;48;13}

Konstanta array ini digunakan sebagai argumen untuk fungsi MAX, memberikan hasil 63.

Kunci F9

Saat bekerja dengan rumus array, Anda dapat melihat sendiri konstanta array ini.

1. Pilih C2:C6-B2:B6 dalam rumus.

2. Tekan F9.

Itu terlihat bagus. Elemen dalam konstanta array vertikal dipisahkan oleh titik koma. Elemen dalam konstanta array horizontal dipisahkan dengan koma.

Hitung Kesalahan

 

Contoh ini menunjukkan cara membuat rumus array yang menghitung jumlah kesalahan dalam rentang.

1. Kami menggunakan IF dan ISERROR untuk memeriksa kesalahan.

Penjelasan: fungsi IF mengembalikan 1, jika ditemukan kesalahan. Jika tidak, ia mengembalikan string kosong.

2. Untuk menghitung error (jangan kewalahan), kita tambahkan fungsi COUNT dan ganti A1 dengan A1:C3.

3. Selesai dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi IF disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{1,"",1;"","","";"","",1}

Konstanta array ini digunakan sebagai argumen untuk fungsi COUNT, memberikan hasil 3.

4. Untuk menghitung kesalahan tertentu, gunakan fungsi COUNTIF . Misalnya, hitung jumlah sel yang berisi #DIV/0! kesalahan.

Hitung Nilai Unik

 

Contoh ini menunjukkan cara membuat rumus array yang menghitung nilai unik .

1. Kami menggunakan fungsi COUNTIF . Misalnya, untuk menghitung jumlah 5, gunakan fungsi berikut.

2. Untuk menghitung nilai unik (jangan kewalahan), kita tambahkan fungsi SUM, 1/, dan ganti 5 dengan A1:A6.

3. Selesai dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi COUNTIF disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{3;1;1;1;3;3} - (tiga 7, satu matahari, satu bulan, satu 5, tiga 7, tiga 7)

Ini dikurangi menjadi:
{1/3;1/1;1/1;1/1;1/3;1/3}

Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 1/3+1+1+1+1+1/3+1/3 = 4.

4. Rumus array di bawah ini menghitung jumlah nilai yang muncul tepat satu kali.

Penjelasan: fungsi IF mengubah konstanta array {3;1;1;1;3;3} menjadi {0;1;1;1;0;0}. Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 3.

5. Jika Anda memiliki Excel 365, cukup gunakan fungsi UNIQUE ajaib untuk mengekstrak nilai unik.

Catatan: fungsi larik dinamis ini, yang dimasukkan ke dalam sel C3, mengisi beberapa sel. Wow! Perilaku ini di Excel 365 disebut menumpahkan.

6. Selanjutnya, tambahkan fungsi COUNTA untuk menghitung nilai unik di Excel 365.

7. Rumus di bawah ini menghitung jumlah nilai yang muncul tepat satu kali.

Catatan: fungsi UNIK memiliki 2 argumen opsional. Nilai default 0 (argumen kedua) memberi tahu fungsi UNIQUE untuk mengekstrak nilai dari larik vertikal. Nilai 1 (argumen ketiga) memberi tahu fungsi UNIQUE untuk mengekstrak nilai yang muncul tepat satu kali.

Hitung dengan Atau Kriteria

 

Menghitung dengan kriteria Or di Excel bisa jadi rumit. Artikel ini menunjukkan beberapa contoh yang mudah diikuti.

1. Kita mulai dari yang sederhana. Misalnya, kita ingin menghitung jumlah sel yang berisi Google atau Facebook (satu kolom).

2a. Namun, jika kita ingin menghitung jumlah baris yang berisi Google atau Stanford (dua kolom), kita tidak bisa begitu saja menggunakan fungsi COUNTIF dua kali (lihat gambar di bawah). Baris yang berisi Google dan Stanford dihitung dua kali, tetapi seharusnya hanya dihitung sekali. 4 adalah jawaban yang kita cari.

2b. Yang kita butuhkan adalah rumus array. Kami menggunakan fungsi IF untuk memeriksa apakah Google atau Stanford terjadi.

Penjelasan: TRUE = 1, FALSE = 0. Untuk baris 1, fungsi IF dievaluasi menjadi IF(TRUE+TRUE,1,0), IF(2,1,0), 1. Jadi baris pertama akan dihitung. Untuk baris 2, fungsi IF dievaluasi menjadi IF(FALSE+FALSE,1,0), IF(0,1,0), 0. Jadi baris kedua tidak akan dihitung. Untuk baris 3, fungsi IF dievaluasi menjadi IF(FALSE+TRUE,1,0), IF(1,1,0), 1. Jadi baris ketiga akan dihitung, dst.

2c. Yang kita butuhkan hanyalah fungsi SUM yang menghitung 1 ini. Untuk mencapai ini (jangan kewalahan), kami menambahkan fungsi SUM dan mengganti A1 dengan A1:A8 dan B1 dengan B1:B8.

2d. Akhiri dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi IF disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{1;0;1;0;1;0;1;0}

Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 4.

3. Kita bisa melangkah lebih jauh. Misalnya, kami ingin menghitung jumlah baris yang berisi (Google dan Stanford) atau Columbia.

Jumlahkan Setiap Baris ke-n

 

Contoh ini menunjukkan cara membuat rumus array yang menjumlahkan setiap baris ke-n di Excel . Kami akan menunjukkannya untuk n = 3, tetapi Anda dapat melakukannya untuk nomor berapa pun.

1. Fungsi ROW mengembalikan nomor baris sel.

2. Fungsi MOD memberikan sisa pembagian. Misalnya, untuk baris pertama, MOD(1,3) = 1 karena 1 dibagi 3 sama dengan 0 dengan sisa 1. Untuk baris ketiga, MOD(3,3) = 0 karena 3 dibagi 3 sama dengan 1 dengan a sisa 0. Akibatnya, rumus mengembalikan 0 untuk setiap baris ke-3.

Catatan: ubah 3 menjadi 4 untuk menjumlahkan setiap baris ke-4, menjadi 5 untuk menjumlahkan setiap baris ke-5, dst.

3. Ubah sedikit rumus seperti gambar di bawah ini.

4. Untuk mendapatkan jumlah hasil kali kedua rentang ini (FALSE=0, TRUE=1), gunakan fungsi SUM dan selesaikan dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Produk dari dua rentang ini (konstanta array) disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut.

{0;0;5;0;0;66;0;0;21}

Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 92.

Contoh ini menunjukkan Anda bagaimana untuk membuat formula array yang merangkum yang jumlah terbesar dalam kisaran.

1. Kami menggunakan fungsi LARGE. Misalnya, untuk menemukan bilangan terbesar kedua, gunakan fungsi berikut.

2. Untuk menjumlahkan 4 bilangan terbesar (jangan kewalahan), tambahkan fungsi SUM dan ganti 2 dengan {1,2,3,4}.

3. Selesai dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi LARGE disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut.

{22,10,8,6}

Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 46.

 

Contoh ini menunjukkan Anda bagaimana untuk membuat formula array yang meringkas sebuah rentang dengan kesalahan . Anda juga dapat menggunakan fungsi AGGREGATE untuk menjumlahkan rentang dengan kesalahan.

1. Kami menggunakan fungsi IFERROR untuk memeriksa kesalahan.

Penjelasan: fungsi IFERROR mengembalikan 0, jika ditemukan kesalahan. Jika tidak, itu mengembalikan nilai sel.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi IFERROR disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut:

Konstanta array ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 13.

Jumlahkan dengan Atau Kriteria

 

Menjumlahkan dengan kriteria Or di Excel bisa jadi rumit. Artikel ini menunjukkan beberapa contoh yang mudah diikuti.

1. Kita mulai dari yang sederhana. Misalnya, kami ingin menjumlahkan sel yang memenuhi kriteria berikut: Google atau Facebook (satu rentang kriteria).

2a. Namun, jika kita ingin menjumlahkan sel yang memenuhi kriteria berikut: Google atau Stanford (dua rentang kriteria), kita tidak bisa begitu saja menggunakan fungsi SUMIF dua kali (lihat gambar di bawah). Sel yang memenuhi kriteria Google dan Stanford ditambahkan dua kali, tetapi hanya boleh ditambahkan satu kali. 10 adalah jawaban yang kita cari.

2b. Kita membutuhkan rumus array. Kami menggunakan fungsi IF untuk memeriksa apakah Google atau Stanford terjadi.

Penjelasan: TRUE = 1, FALSE = 0. Untuk baris 1, fungsi IF dievaluasi menjadi IF(TRUE+TRUE,1,0)*3, IF(2,1,0)*3, 3. Jadi nilai 3 akan ditambahkan. Untuk baris 2, fungsi IF dievaluasi menjadi IF(FALSE+FALSE,1,0)*5, IF(0,1,0)*5, 0. Jadi nilai 5 tidak akan ditambahkan. Untuk baris 3, fungsi IF dievaluasi menjadi IF(FALSE+TRUE,1,0)*2, IF(1,1,0)*2, 2. Jadi nilai 2 akan ditambahkan, dst.

2c. Yang kita butuhkan hanyalah fungsi SUM yang menjumlahkan nilai-nilai ini. Untuk mencapai ini (jangan kewalahan), kami menambahkan fungsi SUM dan mengganti A1 dengan A1:A8, B1 dengan B1:B8 dan C1 dengan C1:C8.

2d. Akhiri dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi IF disimpan dalam memori Excel , bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{1;0;1;0;1;0;1;0}

dikalikan dengan C1:C8 ini menghasilkan:

{3;0;2;0;4;0;1;0}

Konstanta array yang terakhir ini digunakan sebagai argumen untuk fungsi SUM, memberikan hasil 10.

3. Kita bisa melangkah lebih jauh. Misalnya, kami ingin menjumlahkan sel yang memenuhi kriteria berikut: (Google dan Stanford) atau Columbia.

Kata yang Paling Sering Muncul

 

Contoh ini mengajarkan Anda bagaimana menemukan kata yang paling sering muncul di Excel .

Anda dapat menggunakan fungsi MODE untuk menemukan nomor yang paling sering muncul. Namun, fungsi MODE hanya bekerja dengan angka.

Anda dapat menggunakan fungsi COUNTIF untuk menghitung jumlah kemunculan setiap kata. Namun, kami mencari satu formula yang mengembalikan kata yang paling sering muncul (lingkaran dalam contoh kami).

Untuk menemukan kata yang paling sering muncul, lakukan langkah-langkah berikut.

1. Fungsi MATCH mengembalikan posisi nilai dalam rentang tertentu.

Penjelasan: lingkaran (A7) terdapat pada posisi 2 pada range A1:A7. Setel argumen ketiga ke 0 untuk mengembalikan kecocokan yang sama persis.

2. Untuk mencari posisi kata yang paling sering muncul (jangan kewalahan), kita tambahkan fungsi MODE dan ganti A7 dengan A1:A7.

3. Selesai dengan menekan CTRL + SHIFT + ENTER.

Catatan: Bilah rumus menunjukkan bahwa ini adalah rumus larik dengan mengapitnya dalam kurung kurawal {}. Jangan mengetik ini sendiri. Mereka akan hilang saat Anda mengedit rumus.

Penjelasan: Rentang (konstanta array) yang dibuat oleh fungsi MATCH disimpan dalam memori Excel, bukan dalam rentang. Konstanta array terlihat sebagai berikut:

{1;2;2;2;5;5;2} - (segitiga ditemukan di posisi 1, lingkaran ditemukan di posisi 2, lingkaran ditemukan di posisi 2, dll.)

Konstanta array ini digunakan sebagai argumen untuk fungsi MODE, memberikan hasil 2 (posisi kata yang paling sering muncul).

4. Gunakan hasil ini dan fungsi INDEX untuk mengembalikan kata ke-2 dalam rentang A1:A7, kata yang paling sering muncul.
