Cobalah sendiri

Untuk membuat elemen HTML sebagai grid container, Anda harus mengatur properti display (tampilan) ke grid atau inline-grid. Grid container terdiri dari item grid, ditempatkan di dalam kolom dan baris.

Properti grid-template-columns

Properti grid-template-columns mendefinisikan jumlah kolom dalam tata letak grid Anda, dan dapat menentukan lebar setiap kolom. Nilainya adalah daftar yang dipisahkan spasi, di mana setiap nilai menentukan lebar kolom masing-masing. Jika Anda ingin tata letak kisi Anda berisi 4 kolom, tentukan lebar 4 kolom, atau auto jika semua kolom harus memiliki lebar yang sama.

Contoh :

Buat grid dengan 4 kolom :
Code :
1
2
3
4
 
.grid-container {
  display: grid;
  grid-template-columns: auto auto auto auto;
}

Cobalah sendiri

Catatan: Jika Anda memiliki lebih dari 4 item dalam grid 4 kolom, grid akan secara otomatis Tambahkan baris baru untuk memasukkan item.

Properti grid-template-columns juga dapat digunakan untuk menentukan ukuran (lebar) kolom.

Contoh :

Tetapkan ukuran untuk 4 kolom :
Code :
1
2
3
4
 
.grid-container {
  display: grid;
  grid-template-columns: 80px 200px auto 40px;
}

Cobalah sendiri

Properti grid-template-rows

Properti grid-template-rows mendefinisikan ketinggian setiap baris.

Nilainya adalah space-separated-list, di mana setiap nilai mendefinisikan ketinggian baris masing-masing:

Code :
1
2
3
4
 
.grid-container {
  display: grid;
  grid-template-rows: 80px 200px;
}

Cobalah sendiri

Properti justify-content

Properti justify-content digunakan untuk menyelaraskan seluruh grid di dalam container.

Code :
1
2
3
4
 
.grid-container {
  display: grid;
  justify-content: space-evenly;
}

Cobalah sendiri

Catatan: Lebar total grid harus kurang dari lebar container agar properti justify-content memiliki efek apapun.

Contoh :

.grid-container {
  display: grid;
  justify-content: space-evenly;
}

The align-content Property

The align-content property is used to vertically align the whole grid inside the container.

Note: The grid's total height has to be less than the container's height for the align-content property to have any effect.

Contoh :

.grid-container {
  display: grid;
  height: 400px;
  align-content: center;
}
