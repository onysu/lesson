Buka File yang sudah Ada | Tutup File | Buat File New | Matikan Opsi Mulai

Buku kerja adalah kata lain untuk nama file Excel. Saat Anda memulai Excel, klik Blank workbook untuk membuat file Excel dari awal.

Buka File yang sudah Ada

Untuk membuka file yang telah Anda buat sebelumnya, jalankan langkah-langkah berikut.

  1. Pada tab File, klik Open.

  2. Terbaru menampilkan daftar file yang baru saja Anda gunakan. Anda dapat dengan cepat membuka file dari sini.

  3. Klik Telusuri untuk membuka file yang tidak ada dalam daftar.

Tutup File

Untuk menutup file (dan Excel), klik kanan atas X. Jika Anda memiliki beberapa file yang terbuka, mengklik kanan atas X akan menutup file yang aktif.

Buat File Baru

Terkadang Anda ingin memulai dari awal lagi. Untuk membuat file baru, jalankan langkah-langkah berikut.

  1. Pada tab File, klik New.

  2. Klik Blank workbook.

Matikan Opsi mulai (Start up options)

Saat Anda memulai Excel, ini menunjukkan layar mulai yang mencantumkan file dan templat Excel yang baru saja digunakan. Untuk melewati layar mulai dan selalu memulai dengan file kosong, jalankan langkah-langkah berikut.

  1. Pada tab File, klik Options.

  2. Di bawah Start up options, hapus centang Show the Start screen when this application starts.

  3. Klik Oke.

Tema

Excel menawarkan tema untuk mengubah tampilan file Anda dengan mengklik tombol. Setiap tema terdiri dari 12 warna, dua font (Headings dan Body) dan efek untuk bentuk dan SmartArt.

  1. Pada tab Page Layout, di grup Themes, Anda dapat melihat bahwa lembar kerja di bawah ini menggunakan tema Office standar.

  2. Pada tab Home, dalam grup Font, Anda dapat melihat Theme Colors dan Theme Fonts (Calibri Light dan Calibri) dari tema ini.

  3. Pada tab Page Layout, di grup Themes, klik Themes dan pilih tema Badge.

  4. Pada tab Home, di grup Font, Anda dapat melihat Theme Colors dan Font Tema (Impact dan Gill Sans MT) dari tema ini.

    Catatan: pada tab Page Layout Halaman, dalam grup Tema, klik Colors, Kustomisasi Warna untuk membuat warna tema baru atau klik Font, Kustomisasi Font untuk membuat font tema baru. Selanjutnya, klik Tema, Simpan Tema Saat Ini untuk menyimpan tema Anda di folder Tema Dokumen. Anda sekarang dapat menggunakan tema ini di semua file Anda. Anda bahkan dapat menggunakan tema ini di Word dan PowerPoint!

Lihat Beberapa File (View Multiple Workbooks)

Jika Anda ingin melihat beberapa file Excel secara bersamaan, jalankan langkah-langkah berikut.

  1. Pertama, buka dua atau lebih file.

  2. Pada tab View, di grup Window, klik Arrange All.

  3. Pilih pengaturan susunan yang diinginkan. Misalnya, klik Horizontal.

  4. Klik Oke.

  5. Pada tab View, dalam grup Window, klik View Side by Side (secara default, Synchronous Scrolling diaktifkan), untuk menggulir kedua file secara bersamaan.

Pulihkan Otomatis (AutoRecover)

File yang tidak pernah disimpan | Disimpan setidaknya sekali | Versi | Pilihan

Excel secara berkala menyimpan salinan file Excel Anda. Pelajari cara memulihkan file yang tidak pernah disimpan dan cara memulihkan file yang telah disimpan setidaknya sekali.

Jika Excel mengalami crash, itu menampilkan Panel Pemulihan Dokumen (Document Recovery), pertama kali Anda membuka Excel lagi. Ini adalah cara cepat untuk memulihkan file yang disimpan otomatis terakhir.

File yang tidak pernah disimpan

Jika Anda belum pernah menyimpan file dan secara tidak sengaja mengklik Jangan Simpan di bawah saat Anda menutup Excel (atau Excel crash), jalankan langkah-langkah berikut untuk memulihkan file yang terakhir disimpan secara otomatis.

Catatan: Excel memberi tahu Anda apakah salinan terbaru akan tersedia.

  1. Pada tab File, klik Info.

  2. Klik Manage Workbook, Recover Unsaved Workbooks.

  3. Klik file yang terakhir disimpan secara otomatis.

Disimpan setidaknya sekali

Jika Anda telah menyimpan file setidaknya sekali dan Anda secara tidak sengaja mengklik Jangan Simpan di bawah saat Anda menutup Excel (atau Excel crash), jalankan langkah-langkah berikut untuk memulihkan file yang terakhir disimpan secara otomatis.

Catatan: Excel memberi tahu Anda apakah salinan terbaru akan tersedia.

  1. Open file Excelnya!
  2. Pada tab File, klik Info.

  3. Di bawah Manage Workbook, klik file yang terakhir disimpan secara otomatis.

Versi

Saat Anda mengerjakan file Excel, Excel menyimpan semua file yang disimpan otomatis sebelumnya di bawah Kelola File.

  1. Pada tab File, klik Info.

  2. Kapan saja, Anda dapat kembali ke versi sebelumnya dari file Excel Anda.

    Catatan: Excel menghapus semua file yang disimpan otomatis sebelumnya saat Anda menutup file Excel.

Pilihan

Untuk mengubah opsi AutoRecover. jalankan langkah-langkah berikut.

Pada tab File, klik Options, Save.

Catatan: Anda dapat menyimpan informasi AutoRecover setiap x menit, mengubah lokasi file AutoRecover, dll.

Simpan dalam Format 97-2003

Jika Anda menyimpan Anda file Excel di 97-2003 Format (.xls bukan format.xlsx baru), pengguna yang tidak memiliki Excel 2007 atau kemudian diinstal pada komputer mereka, masih bisa membuka file Excel Anda.

  1. Open file.
  2. Pada tab File, klik Save As.

  3. Pilih Excel 97-2003 Workbook (*.xls) dari daftar drop-down.

  4. Klik Simpan.

    Untuk secara otomatis menyimpan file Excel Anda dalam format 97-2003, jalankan langkah-langkah berikut.

  5. Pada tab File, klik Options

  6. Di sisi kiri, klik Save dan pilih Excel 97-2003 Workbook dari daftar drop-down.

  7. Klik Oke.

    Catatan: Excel akan selalu memberi Anda peringatan jika fitur tidak didukung dalam format 97-2003, sehingga Anda dapat menyimpan file dalam format.xlsx baru (File Excel).
