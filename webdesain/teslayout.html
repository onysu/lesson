HTML

HTML adalah bahasa markup standar untuk halaman Web. Dengan HTML Kalian dapat membuat Website Kalian sendiri. HTML mudah dipelajari - You will enjoy it!

Apa itu HTML?

HTML singkatan dari Hyper Text Markup Language. HTML adalah bahasa markup standar untuk membuat halaman Web HTML menggambarkan struktur halaman Web HTML terdiri dari serangkaian elemen Elemen HTML memberi tahu browser cara menampilkan konten Elemen HTML memberi label potongan konten seperti ini adalah judul, ini adalah paragraf, ini adalah tautan, dll.

File HTML diakhiri dengan ekstensi .html atau .htm. Ekstensi file ini bisa dilihat dengan mengunakan web browser apa pun (seperti Google Chrome, Safari, atau Mozila Firefox). Browser tersebut membaca file HTML dan merender kontennya sehingga user internet bisa melihat dan membacanya.

Sejarah HTML dimulai oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertamanya dirilis pada tahun 1991, dengan 18 tag. Sejak saat itu, setiap kali ada versi barunya, pasti akan selalu ada tag dan attribute (tag modifier) yang juga baru.

Berkat popularitasnya yang terus meningkat, bahasa markup ini kini dianggap sebagai standar web resmi. Spesifikasi HTML dikelola dan dikembangkan oleh World Wide Web Consortium (W3C). Berdasarkan HTML Element Reference milik Mozilla Developer Network, saat ini ada 140 tag HTML, meskipun sebagiannya sudah tidak lagi didukung oleh beberapa versi terbaru browser. Kemudian, pada tahun 2014, HTML5 mulai diperkenalkan. Dengan diperkenalkannya HTML5, terdapat semantic baru seperti <article>, <header> dan <footer> yang memperjelas bagian-bagian dalam konten.

Apa perbedaan HTML dengan HTML5?

HTML4 (dikenal sebagai HTML) dirilis pada tahun 1999, kemudian HTML5 diperkenalkan pada publik pada tahun 2014, dengan menambahkan lebih banyak fitur baru ke bahasa mark up. Salah satu fitur canggih di HTML5 adalah dukungan untuk embed audio dan video. Jadi, daripada menggunakan Flash player, kita bisa menyematkan file video dan audio ke halaman website dengan memanfaatkan tag <audio></audio> dan <video></video>. Fitur tersebut juga memiliki dukungan bawaan untuk SVG dan MathML untuk rumus matematika dan ilmiah. HTML5 juga memperkenalkan beberapa perbaikan pada semantic. Tag semantic baru mengirimkan informasi pada browser tentang arti konten, yang tak hanya bermanfaat bagi pembaca, tapi juga mesin pencari. Tag semantic yang paling banyak digunakan adalah <article></article>, <section></section>, <aside> </aside>, <header></header> dan <footer></footer>.

Kelebihan dan Kekurangan HTML

Kelebihan HTML :
  • Digunakan secara luas dan memiliki banyak sumber serta komunitas yang besar.
  • Dijalankan secara alami di setiap web browser.
  • Mudah dipelajari.
  • Open-source dan sepenuhnya gratis.
  • Rapi dan konsisten.
  • Menjadi standar resmi web, dikelola oleh (W3C).
  • Integrasi mudah dengan bahasa backend, seperti PHP dan Node.js.
Kekurangan HTML :
  • Umumnya digunakan untuk halaman web statis. Untuk fitur dinamis, Kalian bisa menggunakan JavaScript atau bahasa backend, seperti PHP.
  • Tidak bisa menjalankan logic. Alhasil, semua halaman web harus dibuat terpisah meskipun menggunakan elemen yang sama, seperti header dan footer.
  • Fitur baru tidak bisa digunakan dengan cepat di sebagian browser.
  • Perilaku browser terkadang sulit diprediksi. Misalnya, browser lama tidak selalu bisa merender tag yang lebih baru.

Hubungan HTML, CSS dan JavaScript

HTML dinyatakan sebagai bahasa markup yang canggih dan mudah, bahasa ini bisa digunakan untuk menambah elemen dan membuat struktur konten. Bahasa ini juga bisa bekerja secara maksimal dengan dua bahasa frontend: CSS (Cascading Style Sheets) dan JavaScript. Jika digabungkan, kedua bahasa frontend ini bisa meningkatkan pengalaman user dan memberikan fungsi yang lebih canggih.

  • CSS erat kaitannya dengan styling, seperti background, warna, layout, spacing dan animiasi.
  • JavaScript memungkinkan Kalian menambahkan fungsionalitas yang dinamis, seperti slider, pop-up, dan galeri foto.

Berdasarkan cara kerja HTML, Kalian bisa menggunakannya untuk membuat struktur konten pada website dan Aplikasi Web. HTML merupakan level teknologi frontend dan berfungsi sebagai dasar styling yang bisa Kalian tambahkan dengan CSS dan fungsionalitas yang dapat dijalakan menggunakan JavaScript.

Contoh Dokumen HTML Sederhana

Contoh :

<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link href="#" rel="stylesheet">
  <title>Judul Halaman</title>
</head>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
</body>

</html>
Penjelasannya :
<!DOCTYPE html> Deklarasi mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
<html> Elemen akar (root) halaman HTML
<head> Elemen berisi informasi meta tentang halaman HTML.
<title> Elemen menentukan judul untuk halaman HTML (yang ditampilkan dalam bar judul browser atau tab halaman ini).
<body> Elemen mendefinisikan tubuh dokumen, dan merupakan wadah untuk semua isi terlihat, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll.
<h1> Elemen mendefinisikan judul.
<p> Elemen mendefinisikan sebuah paragraf.

Apa itu Elemen HTML?

Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir:

Syntax :
<tagname>Konten berada di sini...</tagname>

Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:

<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>

Mulai tag Konten elemen Tag akhir
<h1> Judul pertama saya </h1>
<p> Paragraf pertama saya. </p>
<br> none none

Catatan: Beberapa elemen HTML tidak memiliki konten seperti elemen <br>, elemen ini disebut elemen kosong. Elemen kosong tidak memiliki tag akhir!.

Nested HTML Elements (Elemen HTML Bersarang)

Elemen HTML dapat bersarang (ini berarti elemen dapat berisi elemen lain). Semua dokumen HTML terdiri dari elemen HTML bersarang. Contoh berikut berisi empat elemen HTML (<html>, <body>, <h1> dan <p>).

Contoh :

<!DOCTYPE html>
<html>
<body>

  <h1>Judul Pertama Saya</h1>
  <p>Paragraf pertama saya.</p>

</body>
</html>

Penjelasannya :

  • <html> adalah elemen root elemen dan mendefinisikan dokumen HTML secara keseluruhan. Ini memiliki tag awal <html> dan tag akhir </html>. Kemudian, di dalam <html> elemen ada <body> elemen:

    <body>

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

    </body>

  • <body> mendefinisikan tubuh dokumen. Ini memiliki tag awal <body> dan tag akhir </body>. Kemudian, di dalam <body> elemen ada dua elemen lainnya: <h1> dan <p>:

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

  • <h1> elemen mendefinisikan heading (judul). Ini memiliki tag awal <h1> dan tag akhir </h1>:

    <h1>Judul Pertama Saya</h1>

  • <p> elemen mendefinisikan sebuah paragraf. Ini memiliki tag awal <p> dan tag akhir </p>:

    <p>Paragraf pertama saya.</p>

Ada dua cara untuk menentukan URL dalam src atribut:

Atribut Keterangan
Absolute URL link ke gambar eksternal yang dihosting di situs web lain. Contoh: src="https://onysu.github.io/lesson/assets/img/icons/html5.png".

Catatan: Gambar eksternal mungkin berada di bawah hak cipta. Jika Kalian tidak mendapatkan izin untuk menggunakannya, Kalian mungkin melanggar undang-undang hak cipta. Selain itu, Kalian tidak dapat mengontrol gambar eksternal; itu bisa tiba-tiba dihapus atau diubah.
Relative URL link ke gambar yang dihosting di dalam situs web. Di sini, URL tidak menyertakan nama domain. Jika URL dimulai tanpa garis miring, itu akan relatif terhadap halaman saat ini. Contoh: src="img_girl.jpg. Jika URL dimulai dengan garis miring, itu akan menjadi relatif terhadap domain. Contoh: src="img_girl.jpg.

Tip: Hampir selalu yang terbaik adalah menggunakan URL relatif. Mereka tidak akan rusak jika Kalian mengubah domain.

Kutipan Tunggal atau Ganda?

Tanda kutip ganda   di sekitar nilai atribut adalah yang paling umum dalam HTML, tetapi tanda kutip tunggal '' juga dapat digunakan. Dalam beberapa situasi, ketika nilai atribut itu sendiri berisi tanda kutip ganda, perlu menggunakan tanda kutip tunggal:

<p title='John "ShotGun" Nelson'>

Atau sebaliknya:

<p title="John 'ShotGun' Nelson">

Heading / Judul HTML

Heading HTML adalah Judul atau SubJudul yang ingin Kalian tampilkan di halaman web.
Contoh :

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Heading HTML didefinisikan dengan tag <h1> - <h6>. <h1> mendefinisikan judul yang paling penting. <h6> mendefinisikan heading yang bisa dibilang tidak penting. 

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Cobalah sendiri

Catatan: Browser secara otomatis menambahkan beberapa spasi (margin) sebelum dan sesudah heading.


Fungsi Heading

Mesin pencari menggunakan judul untuk mengindeks struktur dan konten halaman web Kalian. Pengguna sering membaca sepintas halaman berdasarkan judulnya. Penting untuk menggunakan heading untuk menunjukkan struktur dokumen. <h1> heading sebaiknya digunakan untuk heading utama, diikuti <h2> heading, kemudian heading yang kurang penting <h2>, dan seterusnya.

Catatan: Gunakan heading HTML untuk heading saja. Jangan gunakan heading untuk membuat teks menjadi BESAR atau tebal.


Heading bisa lebih besar

Setiap heading HTML memiliki ukuran default. Namun, Kalian dapat menentukan ukuran untuk setiap heading dengan style atribut tersebut, menggunakan font-size properti CSS :
Contoh :

Heading 1

<h1 style="font-size:60px;">Heading 1</h1>

Warna RGB & RGBA

Nilai warna RGB mewakili sumber cahaya RED, GREEN dan BLUE.
Nilai warna RGBA adalah perpanjangan dari RGB dengan Alpha channel (opacity).

Nilai Warna RGB

Dalam HTML, warna dapat ditentukan sebagai nilai RGB, menggunakan rumus ini:

Syntax :
rgb(red, green, blue)

Setiap parameter (merah, hijau, dan biru) mendefinisikan intensitas warna dengan nilai antara 0 dan 255. Ini berarti ada 256 x 256 x 256 = 16777216 kemungkinan warna!. Misalnya, rgb(255, 0, 0) ditampilkan sebagai merah, karena merah diatur ke nilai tertinggi (255), dan dua lainnya (hijau dan biru) diatur ke 0.
Contoh : lain, rgb(0, 255, 0) ditampilkan sebagai hijau, karena hijau diatur ke nilai tertinggi (255), dan dua lainnya (merah dan biru) diatur ke 0. Untuk menampilkan hitam, atur semua parameter warna ke 0, seperti ini: rgb(0, 0, 0). Untuk menampilkan putih, atur semua parameter warna ke 255, seperti ini: rgb(255, 255, 255).

Nilai Warna RGBA

Nilai warna RGBA adalah perpanjangan dari nilai warna RGB dengan saluran Alpha - yang menentukan opasitas untuk warna. Nilai warna RGBA ditentukan dengan:

Syntax :

Parameter alpha adalah angka antara 0.0 (sepenuhnya transparan) dan 1.0 (tidak transparan sama sekali).

CSS

CSS adalah singkatan dari Cascading Style Sheets.
CSS menghemat banyak pekerjaan. Mengontrol tata letak beberapa halaman web sekaligus.

Apa itu CSS?

Cascading Style Sheets (CSS) digunakan untuk memformat tata letak halaman web. Dengan CSS, Kalian dapat mengontrol warna, font, ukuran teks, jarak antar elemen, bagaimana elemen diposisikan dan ditata, gambar latar belakang atau warna latar apa yang akan digunakan, tampilan berbeda untuk perangkat dan ukuran layar yang berbeda, dan lebih banyak!

Tip: Kata cascading berarti bahwa style yang diterapkan ke elemen induk juga akan diterapkan ke semua elemen turunan di dalam induk. Jadi, jika Kalian mengatur warna teks isi ke blue, semua judul, paragraf, dan elemen teks lain di dalam badan juga akan mendapatkan warna yang sama (kecuali jika Kalian menentukan yang lain)!.

Menggunakan CSS

CSS dapat ditambahkan ke dokumen HTML dengan 3 cara:

# Model Keterangan
1 Inline dengan menggunakan <style> atribut di dalam elemen HTML
2 Internal dengan menggunakan <style> elemen di bagian <head>
3 Eksternal dengan menggunakan <link> elemen untuk menautkan ke file CSS eksternal.

Penjelasannya :

CSS sebaris digunakan untuk menerapkan <style> unik ke satu elemen HTML. CSS sebaris menggunakan <style> atribut elemen HTML.

Hyperlink

Links HTML adalah hyperlink. Kalian dapat mengklik link dan melompat ke dokumen lain. Saat Kalian menggerakkan mouse di atas link, panah mouse akan berubah menjadi tangan kecil.

Catatan: Links tidak harus berupa teks. Links dapat berupa gambar atau elemen HTML lainnya!.

Syntax

<a> Tag HTML mendefinisikan hyperlink. Ini memiliki sintaks berikut:

Syntax :
<a href="url">link text</a>

Atribut terpenting dari <a> elemen adalah href atribut, yang menunjukkan tujuan link. link teks adalah bagian yang akan terlihat pembaca. Mengklik teks link, akan mengirim pembaca ke alamat URL yang ditentukan.

Contoh ini menunjukkan cara membuat link ke O-Learning.io :

Kunjungi O-Learning.io!

<a href="https://onysu.github.io/lesson/">Kunjungi O-Learning.io!</a>

Cobalah sendiri

Secara default, link akan muncul sebagai berikut di semua browser:

  • Links yang belum dikunjungi digaris bawahi dan berwarna biru.
  • Links yang dikunjungi digaris bawahi dan berwarna ungu
  • Links aktif digaris bawahi dan merah

Tip: Links tentu saja dapat ditata dengan CSS, untuk mendapatkan tampilan lain!

Target Atribut

Secara default, halaman tertaut akan ditampilkan di jendela browser saat ini. Untuk mengubahnya, Kalian harus menentukan target lain untuk link tersebut. target menspesifikasikan atribut mana untuk membuka dokumen terkait. target atribut dapat memiliki salah satu dari nilai berikut:

_self Default. Membuka dokumen di jendela/tab yang sama dengan yang diklik.
_blank Membuka dokumen di jendela atau tab baru.
_parent Membuka dokumen di bingkai induk.
_top Membuka dokumen di seluruh badan jendela.

Gunakan target="_blank" untuk membuka link tab baru browser :

Syntax Gambar HTML

<img> Tag HTML digunakan untuk menyematkan gambar di halaman web.

Gambar secara teknis tidak dimasukkan ke dalam halaman web, gambar ditautkan ke halaman web. <img> tag menciptakan ruang holding untuk gambar direferensikan. <img> tag kosong, berisi atribut saja, dan tidak memiliki tag penutup.

<img> tag memiliki dua atribut yang diperlukan :

src Menentukan jalur ke gambar
alt Menentukan teks alternatif untuk gambar

Syntax

<img src="url" alt="alternatetext">

src atribut

src atribut yang diperlukan menentukan jalur (URL) ke gambar.

Catatan: Saat halaman web dimuat, browserlah yang pada saat itu mengambil gambar dari server web dan menyisipkannya ke halaman. Oleh karena itu, pastikan bahwa gambar tersebut benar-benar berada di tempat yang sama dengan halaman web, jika tidak pengunjung Anda akan mendapatkan ikon link yang rusak. Ikon link rusak dan altteks ditampilkan jika browser tidak dapat menemukan gambar.

Contoh :

<img src="img_chania.jpg" alt="Bunga di Chania">

Dengan cara ini browser dapat memilih gambar yang paling sesuai dengan tampilan dan/atau perangkat saat ini. Setiap <source> elemen memiliki mediaatribut yang menentukan kapan gambar paling cocok.

Elemen gambar



Ubah ukuran browser untuk melihat versi berbeda dari pemuatan gambar pada ukuran viewport yang berbeda. Browser mencari elemen sumber pertama tempat kueri media cocok dengan lebar area pandang pengguna saat ini, dan mengambil gambar yang ditentukan di atribut srcset.

Elemen img diperlukan sebagai tag anak terakhir dari blok deklarasi gambar. Elemen img digunakan untuk menyediakan kompatibilitas mundur untuk browser yang tidak mendukung elemen gambar, atau jika tidak ada tag sumber yang cocok.
