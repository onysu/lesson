Blok HTML dan Elemen Inline

Setiap elemen HTML memiliki nilai tampilan default, tergantung pada jenis elemen apa itu.

Ada dua nilai tampilan: blok dan inline.

Elemen Level Blok.

Elemen blok-level selalu dimulai pada baris baru.

Elemen blok-level selalu membutuhkan lebar penuh (membentang ke kiri dan kanan sejauh yang bisa).

Elemen blok level memiliki margin atas dan bawah, sedangkan elemen inline tidak.

<div> Elemen adalah elemen blok-level.

Elemen Inline.

Elemen inline tidak dimulai pada baris baru.

Elemen inline hanya memakan lebar sebanyak yang diperlukan.

<div> Elemen

<div> Elemen sering digunakan sebagai wadah untuk elemen HTML lainnya.

<div> elemen tidak memiliki atribut yang diperlukan, tetapi style, class dan id adalah umum.

Ketika digunakan bersama dengan CSS, <div> Elemen dapat digunakan untuk gaya blok konten:

Contoh

<div style="background-color:black;color:white;padding:20px;">
  <h2>London</h2>
  <p>London adalah ibu kota Inggris.Ini adalah kota terpadat di Inggris, dengan wilayah metropolitan lebih dari 13 juta penduduk.</p>
</div>

<span> Elemen

<span> Elemen adalah wadah inline yang digunakan untuk menandai bagian dari teks, atau bagian dari dokumen.

<span> elemen tidak memiliki atribut yang diperlukan, tetapi style, class dan id adalah umum.

Ketika digunakan bersama dengan CSS, <span> Elemen dapat digunakan untuk membuat bagian dari teks:

Contoh

<p>Ibu saya memiliki mata <span style="color:blue;font-weight:bold">biru</span> dan ayahku punya mata <span style="color:darkolivegreen;font-weight:bold">hijau tua </span>.</p>

Ringkasan Bab.

  • Ada dua nilai tampilan: blok dan inline
  • Elemen blok-level selalu dimulai pada baris baru dan mengambil lebar penuh yang tersedia
  • Elemen inline tidak dimulai pada baris baru dan hanya membutuhkan lebar sebanyak yang diperlukan
  • <div> Elemen adalah level blok dan sering digunakan sebagai wadah untuk elemen HTML lainnya
  • <span> Elemen adalah wadah inline yang digunakan untuk menandai bagian dari teks, atau bagian dari dokumen

HTML Tag

Tag Keterangan
<div> Mendefinisikan bagian dalam dokumen (level blok)
<span> Menentukan bagian dalam dokumen (online)
