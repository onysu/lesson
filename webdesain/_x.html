Style HTML

style Atribut HTML digunakan untuk menambahkan gaya ke elemen, seperti warna, font, ukuran dan lainnya.

Contoh :

Teks normal

Teks Merah

Teks Biru

Teks Besar

4
 
<p>Teks normal</p>
<p style="color:red;">Teks Merah</p>

Atribut Style HTML

Pengaturan style suatu elemen HTML, dapat dilakukan dengan style atribut. style Atribut HTML memiliki Syntax berikut:

Syntax : 
<tagname style="property:value;">

property = Properti CSS.
value = Nilai CSS.

kalian akan belajar lebih banyak tentang CSS dalam tutorial ini.

Ringkasan Bab

  • Gunakan background-color untuk warna background

  • Gunakan text-align untuk perataan teks
