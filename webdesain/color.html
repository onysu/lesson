Warna HTML

Warna HTML ditentukan dengan nama warna yang telah ditentukan sebelumnya, atau dengan nilai RGB, HEX, HSL, RGBA atau HSLA.

Nama Warna

Dalam HTML, warna dapat ditentukan dengan menggunakan nama warna:

Tomato
Orange
DodgerBlue
MediumSeaGreen
Gray
SlateBlue
Violet
LightGray
<h1 style="background-color:Tomato;">Tomato</h1>
<h1 style="background-color:Orange;">Orange</h1>
<h1 style="background-color:DodgerBlue;">DodgerBlue</h1>
<h1 style="background-color:MediumSeaGreen;">MediumSeaGreen</h1>
<h1 style="background-color:Gray;">Gray</h1>
<h1 style="background-color:SlateBlue;">SlateBlue</h1>
<h1 style="background-color:Violet;">Violet</h1>
<h1 style="background-color:LightGray;">LightGray</h1>

Nilai Warna

Dalam HTML, warna juga dapat ditentukan menggunakan nilai RGB, nilai HEX, nilai HSL, nilai RGBA, dan nilai HSLA. Tiga elemen <div> berikut memiliki warna latar belakang yang disetel dengan nilai RGB, HEX, dan HSL.
Contoh :

Sama dengan nama warna "Tomato":

rgb(255, 99, 71)
#ff6347
hsl(9, 100%, 64%)

Dua elemen <div> berikut memiliki warna latar belakang yang disetel dengan nilai RGBA dan HSLA, yang menambahkan saluran Alpha ke warna (di sini kita memiliki transparansi 50%):

Sama seperti nama warna "Tomato", tapi 50% transparan:

rgba(255, 99, 71, 0.5)
hsla(9, 100%, 64%, 0.5)
<h1 style="background-color:rgb(255, 99, 71);">rgb(255, 99, 71)</h1>
<h1 style="background-color:#ff6347;">#ff6347</h1>
<h1 style="background-color:hsl(9, 100%, 64%);">hsl(9, 100%, 64%)</h1>
<h1 style="background-color:rgba(255, 99, 71, 0.5);">rgba(255, 99, 71, 0.5)</h1>
<h1 style="background-color:hsla(9, 100%, 64%, 0.5);">hsla(9, 100%, 64%, 0.5)</h1>

Selain nama warna yang telah ditentukan sebelumnya, warna dapat ditentukan menggunakan RGB, HEX, HSL, atau bahkan warna transparan menggunakan nilai warna RGBA atau HSLA.


Warna RGB & RGBA

Nilai warna RGB mewakili sumber cahaya RED, GREEN dan BLUE.
Nilai warna RGBA adalah perpanjangan dari RGB dengan Alpha channel (opacity).


Nilai Warna RGB

Dalam HTML, warna dapat ditentukan sebagai nilai RGB, menggunakan rumus ini:

Syntax :
rgb(red, green, blue)

Setiap parameter (merah, hijau, dan biru) mendefinisikan intensitas warna dengan nilai antara 0 dan 255. Ini berarti ada 256 x 256 x 256 = 16777216 kemungkinan warna!. Misalnya, rgb(255, 0, 0) ditampilkan sebagai merah, karena merah diatur ke nilai tertinggi (255), dan dua lainnya (hijau dan biru) diatur ke 0.
Contoh : lain, rgb(0, 255, 0) ditampilkan sebagai hijau, karena hijau diatur ke nilai tertinggi (255), dan dua lainnya (merah dan biru) diatur ke 0. Untuk menampilkan hitam, atur semua parameter warna ke 0, seperti ini: rgb(0, 0, 0). Untuk menampilkan putih, atur semua parameter warna ke 255, seperti ini: rgb(255, 255, 255).

Percobaan dengan mencampur Nilai RGB di bawah ini:

rgb(255, 99, 71)

RED

255

GREEN

0

BLUE

0

Contoh :

rgb(255, 0, 0)
rgb(0, 0, 255)
rgb(60, 179, 113)
rgb(238, 130, 238)
rgb(255, 165, 0)
rgb(106, 90, 205) 
<h1 style="background-color:rgb(255, 0, 0);">rgb(255, 0, 0)</h1>
<h1 style="background-color:rgb(0, 0, 255);">rgb(0, 0, 255)</h1>
<h1 style="background-color:rgb(60, 179, 113);">rgb(60, 179, 113)</h1>
<h1 style="background-color:rgb(238, 130, 238);">rgb(238, 130, 238)</h1>
<h1 style="background-color:rgb(255, 165, 0);">rgb(255, 165, 0)</h1>
<h1 style="background-color:rgb(106, 90, 205);">rgb(106, 90, 205)</h1>

Nilai Warna RGBA

Nilai warna RGBA adalah perpanjangan dari nilai warna RGB dengan saluran Alpha - yang menentukan opasitas untuk warna. Nilai warna RGBA ditentukan dengan:

Syntax :
rgba(red, green, blue, alpha)

Parameter alpha adalah angka antara 0.0 (sepenuhnya transparan) dan 1.0 (tidak transparan sama sekali).

Percobaan dengan mencampur nilai RGBA di bawah ini:

rgba(255, 99, 71, 0.5)

255

GREEN

0

BLUE

0

ALPHA

0

rgba(255, 99, 71, 0)
rgba(255, 99, 71, 0.2)
rgba(255, 99, 71, 0.4)
rgba(255, 99, 71, 0.6)
rgba(255, 99, 71, 0.8)
rgba(255, 99, 71, 1)

Warna HEX

Warna Heksadesimal ditentukan dengan: #RRGGBB, di mana bilangan bulat Heksadesimal RR (merah), GG (hijau) dan BB (biru) menentukan komponen warna.


Nilai Warna HEX

Dalam HTML, warna dapat ditentukan menggunakan Nilai Heksadesimal dalam bentuk:

#rrggbb

Dimana rr (merah), gg (hijau) dan bb (biru) adalah Nilai Heksadesimal antara 00 dan ff (sama dengan desimal 0-255).
Misalnya, #ff0000 ditampilkan sebagai merah, karena merah diatur ke Nilai tertinggi (ff), dan dua lainnya (hijau dan biru) diatur ke 00. #00ff00 ditampilkan sebagai hijau, karena hijau diatur ke Nilai tertinggi (ff), dan dua lainnya (merah dan biru) diatur ke 00. Untuk menampilkan hitam, setel semua parameter warna ke 00, seperti ini: #000000. Untuk menampilkan putih, setel semua parameter warna ke ff, seperti ini: #ffffff.

Percobaan dengan mencampur Nilai HEX di bawah ini:

#ff6347

MERAH

ff

HIJAU

63

BIRU

47

Contoh :

#ff0000
#0000ff
#3cb371
#ee82ee
#ffa500
#6a5acd

<h1 style="background-color:#0000ff;">#0000ff</h1>

<h1 style="background-color:#ffa500;">#ffa500</h1>

Warna HSL & HSLA

HSL adalah singkatan dari Hue, Saturation, dan Lightness. Nilai warna HSLA merupakan perpanjangan dari HSL dengan saluran Alpha (opacity).


Nilai Warna HSL

Dalam HTML, sebuah warna dapat ditentukan menggunakan hue, saturation, dan lightness (HSL) dalam bentuk:

hsl(hue, saturation, lightness)
Warna Keterangan
Hue adalah derajat pada roda warna dari 0 hingga 360. 0 adalah merah, 120 adalah hijau, dan 240 adalah biru.
Saturation adalah nilai persentase, 0% berarti bayangan abu-abu, dan 100% adalah warna penuh.
Lightness juga merupakan nilai persentase, 0% hitam, dan 100% putih.

Percobaan dengan mencampur nilai HSL di bawah ini:

hsl(0, 100%, 50%)

HUE

0

SATURATION

100%

LIGHTNESS

50%

Contoh :

hsl(0, 100%, 50%)
hsl(240, 100%, 50%)
hsl(147, 50%, 47%)
hsl(300, 76%, 72%)
hsl(39, 100%, 50%)
hsl(248, 53%, 58%)
<h1 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h1>
<h1 style="background-color:hsl(240, 100%, 50%);">hsl(240, 100%, 50%)</h1>
<h1 style="background-color:hsl(147, 50%, 47%);">hsl(147, 50%, 47%)</h1>
<h1 style="background-color:hsl(300, 76%, 72%);">hsl(300, 76%, 72%)</h1>
<h1 style="background-color:hsl(39, 100%, 50%);">hsl(39, 100%, 50%)</h1>
<h1 style="background-color:hsl(248, 53%, 58%);">hsl(248, 53%, 58%)</h1>

Saturation

Saturation dapat digambarkan sebagai intensitas warna.

Value Keterangan
100% adalah warna murni, tidak ada nuansa abu-abu.
50% adalah 50% abu-abu, tetapi Anda masih dapat melihat warnanya.
0% benar-benar abu-abu, Anda tidak bisa lagi melihat warnanya.

Contoh :

hsl(0, 100%, 50%)
hsl(0, 80%, 50%)
hsl(0, 60%, 50%)
hsl(0, 40%, 50%)
hsl(0, 20%, 50%)
hsl(0, 0%, 50%)

Dengan warna HSL, lebih sedikit Saturasi berarti lebih sedikit warna. 0% benar-benar Abu-abu.

<h1 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h1>
<h1 style="background-color:hsl(0, 80%, 50%);">hsl(0, 80%, 50%)</h1>
<h1 style="background-color:hsl(0, 60%, 50%);">hsl(0, 60%, 50%)</h1>
<h1 style="background-color:hsl(0, 40%, 50%);">hsl(0, 40%, 50%)</h1>
<h1 style="background-color:hsl(0, 20%, 50%);">hsl(0, 20%, 50%)</h1>
<h1 style="background-color:hsl(0, 0%, 50%);">hsl(0, 0%, 50%)</h1>

Lightness

Lightness warna dapat digambarkan sebagai seberapa banyak cahaya yang ingin Anda berikan pada warna, di mana 0% berarti tidak ada cahaya (hitam), 50% berarti 50% terang (tidak gelap atau terang) 100% berarti terang penuh (putih).

Contoh :

hsl(0, 100%, 0%)
hsl(0, 100%, 25%)
hsl(0, 100%, 50%)
hsl(0, 100%, 75%)
hsl(0, 100%, 90%)
hsl(0, 100%, 100%)

Dengan warna HSL, 0% Lightness berarti Hitam, dan 100 Lightness berarti Putih.

<h1 style="background-color:hsl(0, 100%, 0%);">hsl(0, 100%, 0%)</h1>
<h1 style="background-color:hsl(0, 100%, 25%);">hsl(0, 100%, 25%)</h1>
<h1 style="background-color:hsl(0, 100%, 50%);">hsl(0, 100%, 50%)</h1>
<h1 style="background-color:hsl(0, 100%, 75%);">hsl(0, 100%, 75%)</h1>
<h1 style="background-color:hsl(0, 100%, 90%);">hsl(0, 100%, 90%)</h1>
<h1 style="background-color:hsl(0, 100%, 100%);">hsl(0, 100%, 100%)</h1>

hsla(hue, saturation, lightness, alpha)

Percobaan dengan mencampur nilai HSLA di bawah ini:

hsla(0, 100%, 50%, 0.5)

0.5

hsla(9, 100%, 64%, 0.8)
