Tabel HTML - Padding Sel

Cell padding adalah ruang antara tepi sel dan isi sel.

Secara default padding diatur ke 0.

Untuk menambahkan padding pada sel tabel, gunakan paddingproperti CSS :

Contoh :

th, td {
padding: 15px;
}

The <colgroup>elemen harus digunakan sebagai wadah untuk spesifikasi kolom.

Setiap grup ditentukan dengan <col>elemen.

The spanAtribut menentukan berapa banyak kolom yang mendapat gaya.

Catatan: Ada pilihan yang sangat terbatas dari properti CSS legal untuk colgroups .

Contoh :

<table>
<colgroup>
<col span="2" style="background-color: #D6EEEE">
</colgroup>
<tr>
<th>MON</th>

<th>THU</th>

Cobalah sendiri "

Catatan: The <colgroup>tag harus menjadi anak dari <table>elemen dan harus ditempatkan sebelum elemen meja lainnya, seperti <thead>, <tr>, <td> dll, tapi setelah <caption>elemen, jika ada.

Properti CSS Legal

Hanya ada pilihan properti CSS yang sangat terbatas yang diizinkan untuk digunakan dalam colgroup:

widthproperti
visibilityproperti
backgroundproperti
borderproperti

Semua properti CSS lainnya tidak akan berpengaruh pada tabel Anda.
