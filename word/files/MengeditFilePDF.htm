Mengedit file PDF

Sebuah file PDF adalah jenis file yang dirancang untuk menjadi dilihat di komputer manapun . Ini berguna ketika Anda ingin mengirim dokumen ke seseorang dan Anda tidak yakin apakah orang itu memiliki Microsoft Word. File PDF dirancang untuk dilihat daripada diedit, sehingga ideal untuk situasi di mana Anda memiliki versi final yang tidak memerlukan perubahan apa pun. Namun, jika Anda perlu mengedit file PDF karena alasan apa pun, Word 2013 memungkinkan Anda mengeditnya dengan mengubahnya menjadi dokumen Word .

Mengonversi file PDF menjadi dokumen Word dapat menyebabkan beberapa grafik dan konten terlihat berbeda atau hilang sama sekali.

Untuk mengedit file PDF:

Pilih tab File .

Tampilan belakang panggung akan muncul. Klik Buka .

Cari dan buka file PDF yang ingin Anda edit.

Kotak dialog akan muncul mengonfirmasi apakah Anda ingin mengonversi file PDF menjadi dokumen Word. Klik Oke .

Jika file PDF memiliki konten yang tidak dapat dikonversi oleh Word, kotak dialog lain mungkin muncul. Klik Oke .

File PDF akan muncul di jendela Word sebagai dokumen yang dapat diedit. Anda sekarang dapat membuat perubahan apa pun yang Anda inginkan.

Saat Anda siap untuk menyimpan, cari dan pilih perintah Simpan di toolbar Akses Cepat .

Kotak dialog Simpan Sebagai akan muncul. Pilih lokasi di mana Anda ingin menyimpan dokumen. Jika mau, masukkan nama file baru .

Klik kotak tarik-turun Simpan sebagai jenis :, lalu pilih jenis file . Anda dapat memilih untuk menyimpan dokumen sebagai Dokumen Word atau sebagai file PDF .

Klik Simpan .

Jenis file mana yang harus Anda pilih?

Setelah Anda selesai mengonversi dan mengedit file PDF, Anda harus memutuskan apakah akan menyimpannya sebagai file PDF atau dokumen Word. Bergantung pada situasi Anda, satu opsi mungkin berfungsi lebih baik daripada yang lain:

PDF: Jika Anda tidak ingin membuat perubahan lagi pada dokumen—atau jika Anda ingin memastikan bahwa file dapat dibuka di komputer mana pun—simpan sebagai file PDF.

Dokumen Word: Jika Anda ingin terus membuat perubahan pada dokumen—atau jika orang lain perlu mengeditnya—simpan sebagai dokumen Word. Anda selalu dapat menyimpannya sebagai file PDF setelah selesai membuat perubahan.
