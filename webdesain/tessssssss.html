Elemen Semantik HTML

Elemen semantik = elemen dengan makna.

Apa itu elemen semantik?

Elemen semantik dengan jelas menjelaskan artinya bagi browser dan pengembang.
Contoh elemen non-semantik: <div> dan <span> - Tidak memberi tahu apa pun tentang kontennya.
Contoh elemen semantik: <form>, <table>, dan <article> - Mendefinisikan isinya dengan jelas.

Elemen semantik dalam HTML

Banyak situs web yang berisi kode HTML seperti: <div id="nav"> <div class="header"> <div id="footer"> untuk menunjukkan navigasi, header, dan footer. Dalam HTML ada beberapa elemen semantik yang dapat digunakan untuk mendefinisikan berbagai bagian halaman Web:

  • <article>
  • <aside>
  • <details>
  • <figcaption>
  • <figure>
  • <footer>
  • <header>
  • <main>
  • <mark>
  • <nav>
  • <section>
  • <summary>
  • <time>

<section> Element HTML

Elemen <section> mendefinisikan bagian dalam dokumen. Menurut dokumentasi HTML W3C: Bagian adalah pengelompokan konten secara tematik, biasanya dengan judul..
Contoh di mana elemen <section> dapat digunakan.

  • Bab
  • Pengantar
  • Item Berita
  • Kontak Informasi

Halaman web biasanya dapat dibagi menjadi beberapa bagian untuk informasi pengenalan, konten, dan kontak.

<article> Element HTML

Elemen <article> menentukan konten mandiri atau self-contained content.
Sebuah artikel harus masuk akal dengan sendirinya, dan harus memungkinkan untuk mendistribusikannya secara independen dari situs web lainnya.
Contoh di mana elemen <article> dapat digunakan:

  • Posting Forum.
  • Posting Blog.
  • Komentar Pengguna.
  • Kartu Produk.
  • Artikel Koran.

Nesting/Bersarang <article> di <section> atau sebaliknya?

Elemen <article> menentukan konten mandiri dan mandiri. Elemen <section> mendefinisikan bagian dalam dokumen. Bisakah kita menggunakan definisi untuk memutuskan bagaimana menyusun elemen-elemen itu? Tidak, kita tidak bisa! Jadi, Anda akan menemukan halaman HTML dengan elemen <section> yang mengandung elemen <article>, dan elemen <article> yang mengandung elemen <section>.

<header> Elemen

Elemen <header> mewakili wadah untuk konten pengantar atau kumpulan tautan navigasi.
Elemen <header> biasanya berisi:

  • satu atau lebih elemen heading (<h1> - <h6>)
  • logo atau ikon
  • informasi pengarang

Catatan: Anda dapat memiliki beberapa elemen <header> dalam satu dokumen HTML. Namun, <header> tidak dapat ditempatkan di dalam elemen <footer>, <address> atau <header> lainnya.

Contoh :

Header untuk <article>:

Apa yang dilakukan WWF?

Misi WWF:

Misi WWF adalah menghentikan degradasi lingkungan alam planet kita, dan membangun masa depan di mana manusia hidup selaras dengan alam.

<footer> Elemen

Elemen <footer> mendefinisikan footer untuk dokumen atau bagian.
Elemen <footer> biasanya berisi:

  • Informasi Kepenulisan
  • Informasi Hak Cipta
  • Kontak Informasi
  • Sitemap.
  • Kembali Ke Top Tautan
  • Dokumen Terkait

Anda dapat memiliki beberapa elemen <footer> dalam satu dokumen.

Contoh :

Bagian footer dalam dokumen:

Penulis: Ony Darrell

onydarrell@gmail.com

<nav> Elemen

Elemen <nav> mendefinisikan sekumpulan tautan navigasi.

Perhatikan bahwa TIDAK semua tautan dokumen harus berada di dalam elemen <nav>. Elemen <nav> ditujukan hanya untuk blok utama tautan navigasi.
Browser, seperti pembaca layar untuk pengguna yang dinonaktifkan, dapat menggunakan elemen ini untuk menentukan apakah akan menghilangkan rendering awal konten ini.

Contoh :

Satu set tautan navigasi:

Penulis: Ony Darrell

onydarrell@gmail.com

Code :
1
2
3
4
5
6
 
<nav>
  <a href="/html/">HTML</a> |
  <a href="/css/">CSS</a> |
  <a href="/js/">JavaScript</a> |
  <a href="/jquery/">jQuery</a>
</nav>

<figure> & <figcaption> Elemen

Tag <figure> menentukan konten mandiri, seperti ilustrasi, diagram, foto, daftar kode, dll.
Tag <figcaption> mendefinisikan keterangan untuk elemen <figure>. Elemen <figcaption> dapat ditempatkan sebagai yang pertama atau sebagai anak terakhir dari elemen <figure>.
Elemen <img> mendefinisikan gambar/ilustrasi yang sebenarnya.

Contoh :

Tempat untuk dikunjungi

Pemandangan Puglia yang paling terkenal adalah rumah kerucut yang unik (Trulli) yang ditemukan di daerah sekitar Alberobello, sebuah situs yang dinyatakan warisan dunia oleh UNESCO.

Code :

2
3
4
 
          <figure>
            <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
            <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
          </figure>

Mengapa Elemen Semantik?

Menurut W3C: Web semantik memungkinkan data untuk dibagikan dan digunakan kembali di seluruh aplikasi, perusahaan, dan komunitas.

Elemen Semantik dalam HTML

Di bawah ini adalah daftar beberapa elemen semantik dalam HTML.

Tag Keterangan
<article> Mendefinisikan konten mandiri yang mandiri
<aside> Tetapkan Konten Selain dari Konten Halaman
<details> Mendefinisikan detail tambahan yang dapat dilihat atau disembunyikan oleh pengguna
<figcaption> Mendefinisikan keterangan untuk A & LT; Gambar & GT;elemen
<figure> Menentukan konten mandiri, seperti ilustrasi, diagram, foto, daftar kode, dll.
<footer> Mendefinisikan footer untuk dokumen atau bagian
<header> Menentukan header untuk dokumen atau bagian
<main> Menentukan konten utama dokumen
<mark> Mendefinisikan teks yang ditandai / disorot
<nav> Mendefinisikan tautan navigasi
<section> Mendefinisikan bagian dalam dokumen
<summary> Mendefinisikan heading yang terlihat untuk elemen <details>
<time> Mendefinisikan tanggal / waktu
