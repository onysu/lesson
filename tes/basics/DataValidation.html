Gunakan Data Validation di Excel untuk memastikan bahwa pengguna memasukkan nilai tertentu ke dalam sel.

Contoh Data Validation

Dalam contoh ini, kami membatasi pengguna untuk memasukkan bilangan bulat antara 0 dan 10.

Buat Aturan Data Validation

Untuk membuat aturan Data Validation, jalankan langkah-langkah berikut.

  1. Pilih sel C2.

  2. Pada tab Data, dalam grup Data Tools, klik Data Validation.

    Pada tab Settings:

  3. Dalam daftar Allow, klik Whole number.
  4. Dalam daftar Data, klik di between.

  5. Masukkan nilai Minimum dan Maksimum.

Pesan Masukan

Pesan input muncul ketika pengguna memilih sel dan memberi tahu pengguna apa yang harus dimasukkan.

Pada tab Input Message:

  1. Centang 'Show input message when cell is selected'.
  2. Masukkan Title.

  3. Masukkan Input message.

Peringatan Kesalahan

Jika pengguna mengabaikan pesan input dan memasukkan nomor yang tidak valid, Anda dapat menunjukkan peringatan kesalahan kepada mereka.

Pada tab Error Alert:

  1. Centang 'Show error alert after invalid data is entered'.
  2. Masukkan Title.

  3. Masukkan Error message.

  4. Klik Oke.

Hasil Data Validation

  1. Pilih sel C2.

  2. Coba masukkan angka yang lebih tinggi dari 10.

    Hasil:

    Catatan: untuk menghapus Data Validation dari sel, pilih sel, pada tab Data, dalam grup Data Tools, klik Data Validation, lalu klik Hapus Semua. Anda dapat menggunakan fitur Go To Special Excel untuk memilih semua sel dengan cepat dengan Data Validation.
