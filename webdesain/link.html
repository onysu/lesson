Link HTML

Link / Tautan ditemukan di hampir semua halaman web. Tautan memungkinkan pengguna mengeklik dari halaman ke halaman.

Hyperlink

Links HTML adalah hyperlink. Anda dapat mengklik link dan melompat ke dokumen lain. Saat Anda menggerakkan mouse di atas link, panah mouse akan berubah menjadi tangan kecil.

Catatan: Links tidak harus berupa teks. Links dapat berupa gambar atau elemen HTML lainnya!.


Syntax

<a> Tag HTML mendefinisikan hyperlink. Ini memiliki sintaks berikut:

Syntax :
<a href="url">link text</a>

Atribut terpenting dari <a> elemen adalah href atribut, yang menunjukkan tujuan link. link teks adalah bagian yang akan terlihat pembaca. Mengklik teks link, akan mengirim pembaca ke alamat URL yang ditentukan.

Contoh ini menunjukkan cara membuat link ke O-Learning.io :

Kunjungi O-Learning.io!

<a href="https://onysu.github.io/lesson/">Kunjungi O-Learning.io!</a>

Cobalah sendiri

Secara default, link akan muncul sebagai berikut di semua browser:

  • Links yang belum dikunjungi digaris bawahi dan berwarna biru.
  • Links yang dikunjungi digaris bawahi dan berwarna ungu
  • Links aktif digaris bawahi dan merah

Tip: Links tentu saja dapat ditata dengan CSS, untuk mendapatkan tampilan lain!


Atribut Target

Secara default, halaman tertaut akan ditampilkan di jendela browser saat ini. Untuk mengubahnya, Anda harus menentukan target lain untuk link tersebut. target menspesifikasikan atribut mana untuk membuka dokumen terkait. target atribut dapat memiliki salah satu dari nilai berikut:

  • _self - Default. Membuka dokumen di jendela/tab yang sama dengan yang diklik.
  • _blank - Membuka dokumen di jendela atau tab baru.
  • _parent - Membuka dokumen di bingkai induk.
  • _top - Membuka dokumen di seluruh badan jendela.

Contoh :
Gunakan target="_blank" untuk membuka link tab baru browser :

Kunjungi O-Learning!

<a href="https://onysu.github.io/lesson/webdesain/" target="_blank">Kunjungi O-Learning!</a>

Cobalah sendiri


URL Absolut vs. URL Relatif

Kedua contoh di atas menggunakan URL absolut (alamat web lengkap) dalam href atribut. Links lokal (link ke halaman dalam situs web yang sama) ditentukan dengan URL relatif (tanpa bagian "https://www").

Contoh :

Absolute URLs

O-Learning
Google

Relative URLs

Halaman Lokal

<h2>Absolute URLs</h2>
<p><a href="https://onysu.github.io/lesson/webdesain/">O-Learning</a></p>
<p><a href="https://www.google.com/">Google</a></p>

<h2>Relative URLs</h2>
<p><a href="link-lokal.html">HTML Images</a></p>

Cobalah sendiri

Links ke Alamat Email

Contoh :

Tautan ke Alamat Email

Untuk membuat tautan yang terbuka di program email pengguna (agar mereka dapat mengirim email baru), gunakan mailto: di dalam atribut href:

Kirim email

Tombol sebagai Links

Untuk menggunakan tombol HTML sebagai link, Anda harus menambahkan beberapa kode JavaScript.
JavaScript memungkinkan Anda untuk menentukan apa yang terjadi pada peristiwa tertentu, seperti mengklik tombol:

Contoh :

<button onclick="document.location='link-lokal.html'">Link lokal</button>

Ringkasan Bab

  • Gunakan <a> elemen untuk menentukan link.
  • Gunakan href atribut untuk menentukan alamat link.
  • Gunakan target atribut untuk menentukan tempat membuka dokumen tertaut.
  • Gunakan <img> elemen (di dalam <a>) untuk menggunakan gambar sebagai link.
  • Gunakan mailto:skema di dalam hrefatribut untuk membuat link yang membuka program email pengguna.

Warna Link

Links HTML ditampilkan dalam warna yang berbeda tergantung pada apakah telah dikunjungi, belum dikunjungi, atau aktif.

Warna Links HTML

Secara default, link akan muncul seperti ini (di semua browser):

  • Links yang belum dikunjungi digarisbawahi dan berwarna biru.
  • Links yang dikunjungi digarisbawahi dan berwarna ungu.
  • Links aktif digarisbawahi dan merah.

Anda dapat mengubah warna status link, dengan menggunakan CSS:

Di sini, link yang belum dikunjungi akan berwarna hijau tanpa garis bawah. Links yang dikunjungi akan berwarna merah muda tanpa garis bawah. Links aktif akan berwarna kuning dan digarisbawahi. Selain itu, saat mengarahkan mouse ke link (a:hover) itu akan menjadi merah dan digarisbawahi:

Buat Bookmark

Links HTML dapat digunakan untuk membuat bookmark, sehingga pembaca dapat melompat ke bagian tertentu dari halaman web.

Membuat Bookmark di HTML

Bookmark dapat berguna jika halaman web sangat panjang. Untuk membuat bookmark - buat bookmark terlebih dahulu, lalu tambahkan link ke sana. Ketika link diklik, halaman akan bergulir ke bawah atau ke atas ke lokasi dengan bookmark.

Contoh :

Pertama, gunakan id atribut untuk membuat bookmark:

Bab 4

<h2 id="C4">Bab 4</h2>

Kemudian, tambahkan link ke bookmark ("Langsung ke Bab 4"), dari dalam halaman yang sama:

Langsung ke Bab 4

<a href="#C4">Langsung ke Bab 4</a>

Cobalah sendiri


Ringkasan Bab
  • Gunakan id atribut (id="value") untuk menentukan bookmark di halaman.
  • Gunakan href atribut (href="#value") untuk menautkan ke bookmark
