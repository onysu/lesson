Style HTML

style Atribut HTML digunakan untuk menambahkan gaya ke elemen, seperti warna, font, ukuran, dan lainnya.
Contoh :

Teks normal

Teks merah

Teks biru

Teks besar

<p>Teks normal</p>
<p style="color:red;">Teks merah</p>
<p style="color:blue;">Teks biru</p>
<p style="font-size:50px;">Teks besar</p>

Cobalah sendiri

Atribut Style HTML

Pengaturan style suatu elemen HTML, dapat dilakukan dengan style atribut. style Atribut HTML memiliki sintaks berikut:

<tagname style="property:value;">

property adalah properti CSS. value adalah nilai CSS.

Anda akan belajar lebih banyak tentang CSS nanti dalam tutorial ini.

Warna latar belakang

background-color properti CSS mendefinisikan warna latar belakang untuk elemen HTML.
Contoh

Atur warna latar belakang halaman menjadi powderblue:

<body style="background-color:powderblue;">

<h1>Ini adalah judul</h1>
<p>Ini adalah paragraf</p>

</body>

Cobalah sendiri

Contoh :

Atur warna latar belakang untuk dua elemen berbeda:

<body>

<h1 style="background-color:powderblue;">Ini adalah judul</h1>
<p style="background-color:tomato;">Ini adalah paragraf</p>

</body>

Warna teks

color properti CSS mendefinisikan warna teks untuk elemen HTML:

Contoh

<h1 style="color:blue;">Ini adalah judul</h1>
<p style="color:red;">Ini adalah paragraf</p>

Cobalah sendiri

Font

font-family properti CSS mendefinisikan font yang akan digunakan untuk elemen HTML:

Contoh

<h1 style="font-family:verdana;">Ini adalah judul</h1>
<p style="font-family:courier;">Ini adalah paragraf</p>

Cobalah sendiri

Ukuran teks

font-size properti CSS mendefinisikan ukuran teks untuk elemen HTML:

Contoh

<h1 style="font-size:300%;">Ini adalah judul</h1>
<p style="font-size:160%;">Ini adalah paragraf</p>

Cobalah sendiri

Perataan Teks

text-align properti CSS mendefinisikan perataan teks horizontal untuk elemen HTML:

Contoh

<h1 style="text-align:center;">Judul ditengah</h1>
<p style="text-align:center;">Paragraph ditengah.</p>

Cobalah sendiri

Ringkasan Bab

  • Gunakan style atribut untuk menata elemen HTML
  • Gunakan background-color untuk warna latar belakang
  • Gunakan color untuk warna teks
  • Gunakan font-family untuk font teks
  • Gunakan font-size untuk ukuran teks
  • Gunakan text-align untuk perataan teks
