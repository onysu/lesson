Dasar HTML

Dalam bab ini kita akan menunjukkan beberapa contoh HTML dasar. Jangan khawatir jika kami menggunakan tag yang belum Anda pelajari.

Editor HTML

Anda perlu Editor teks untuk mempelajari HTML. Halaman web dapat dibuat dan dimodifikasi dengan menggunakan editor HTML profesional.

Beberapa contoh text editor terbaik saat ini :

Atau dengan Code Editor sederhana seperti Notepad bawaan dari Windows.

Belajar menggunakan Notepad :

Ikuti langkah-langkah di bawah ini untuk membuat halaman web pertama Anda dengan Notepad :


Langkah 1: Buka Notepad

Windows 8 atau lebih baru:
Buka Layar Mulai (simbol jendela di kiri bawah layar Anda). Ketik Notepad.
Windows 7 atau sebelumnya:
Buka Start > Programs > Accessories > Notepad


Langkah 2: Tulis Beberapa HTML

Tulis atau salin kode HTML berikut ke Notepad:

<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Judul Halaman</title>
</head>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
</body>

</html>

Langkah 3: Simpan Halaman HTML

Save As di menu Notepad. Beri nama file "index.htm" dan atur pengkodean ke UTF-8 (yang merupakan pengkodean pilihan untuk file HTML).

Tip: Anda dapat menggunakan .htm atau .html sebagai ekstensi file. Tidak ada perbedaan, terserah Anda.


Langkah 4: Lihat Halaman HTML di Browser

Buka file HTML yang disimpan di browser favorit Anda (klik dua kali pada file, atau klik kanan - dan pilih Open with).

Dengan editor online sederhana ini, Anda dapat mengedit kode HTML dan melihat hasilnya di browser Anda. Ini adalah alat ketika Anda ingin menguji kode dengan cepat.

</head>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
</body>

</html>
