HTML Iframes.

HTML Iframe digunakan untuk menampilkan halaman web dalam halaman web.

Sintaks Iframe HTML.

HTML <iframe> tag menentukan bingkai inline. Bingkai inline digunakan untuk menanamkan dokumen lain dalam dokumen HTML saat ini.

<iframe src="url" title="description"> </iframe>

Tip: Ini adalah praktik yang baik untuk selalu menyertakan atribut title untuk <iframe>. Ini digunakan oleh pembaca layar untuk membacakan apa isi iframe.

Iframe - Set Tinggi & Lebar

Gunakan height dan width atribut untuk menentukan ukuran iframe. Tinggi dan Lebar secara default ditentukan dalam piksel :

Contoh :

<iframe src="demo_iframe.htm" height="200" width="300" title ="Iframe Example"></iframe > Try it Yourself »

atau anda dapat menambahkan atribut style dan menggunakan css height dan width properti:

Contoh :

<iframe src="demo_iframe.htm" style="height:200px;width:300px;" title ="Iframe Example"></iframe > Try it Yourself »

Iframe - Hapus Border

Secara default, iframe memiliki border di sekitarnya. Untuk menghapus border, tambahkan atribut style dan gunakan properti border CSS:

Contoh :

<iframe src="demo_iframe.htm" style="border:none;" title ="Iframe Example"></iframe > Try it Yourself »

Dengan CSS, Anda juga dapat mengubah ukuran, gaya dan warna border iFrame:

Contoh :

<iframe src="demo_iframe.htm" style="border:2px solid red;" title ="Iframe Example"></iframe > Try it Yourself »

Iframe - Target untuk Tautan

Sebuah iframe dapat digunakan sebagai frame target untuk sebuah link. Atribut target tautan harus merujuk ke atribut name iframe:

Contoh :

<iframe src="demo_iframe.htm" name="iframe_a" title="Iframe Example"></iframe>

<p><a href="https://www.w3schools.com" target="iframe_a" > W3Schools.com </a></p> Try it Yourself »

Ringkasan Bab

  • Tag HTML <iframe> menentukan bingkai sebaris.
  • Atribut src mendefinisikan URL halaman yang akan disematkan.
  • Selalu sertakan atribut title (untuk pembaca layar).
  • Atribut height dan width menentukan ukuran iframe.
  • Gunakan border:none; tidak ada; untuk menghapus batas di sekitar iframe.
