Apa Itu Pemrograman?

Pemrograman = memberi instruksi ke komputer agar melakukan tugas tertentu.

Bahasa pemrograman umum:
  • PHP
  • JavaScript
  • Python
  • Java
  • C++
  • Ruby
  • ...
Konsep dasar:
  • Variabel
  • Input/Output
  • Struktur kontrol ( if, loop )

Cara Kerja Sebuah Website

Alur kerja:
Browser → Server → PHP → Database → PHP → Browser kembali

Peran masing-masing:

  • HTML = struktur tampilan
  • CSS = styling / desain tampilan halaman website
  • JavaScript = interaksi
  • PHP = logika di server
  • MySQL = penyimpanan data

Apa Itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang berjalan di sisi server (server-side).

Dikembangkan oleh Rasmus Lerdorf pada tahun 1994.

Digunakan untuk membuat halaman web dinamis dan interaktif.

File PHP memiliki ekstensi .php.

Menjalankan PHP di Local Server

Untuk menjalankan PHP secara lokal, kita perlu menginstal web server seperti:

  • XAMPP: Paket software yang mencakup Apache (web server), MySQL (database), PHP, dan Perl.
  • Laragon: Alternatif ringan untuk XAMPP dengan fitur tambahan.

Setelah instalasi, file PHP disimpan di folder khusus:

  • XAMPP: htdocs
  • Laragon: www

Akses file melalui browser dengan URL: http://localhost/nama_folder/namafile.php
