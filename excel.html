Tentang Microsoft Excel

Microsoft Excel adalah sebuah Aplikasi Spreadsheet yang berfungsi sebagai Aplikasi Pengolah Angka. Apabila di dalam komputer yang Anda gunakan saat ini sudah terdapat Microsoft Word atau Powerpoint maka kemungkinan besar Microsoft Excel juga sudah terinstall dan dapat digunakan pada komputer Anda. Office Excel merupakan salah satu bagian dari Paket Instalasi Microsoft Office.

Apabila anda menggunakan Regional setting English maka menggunakan titik koma (,) dan Regional setting Indonesia maka menggunakan koma (;) yang ada pada setiap Rumus Excel.

Pengertian Rumus Excel

Rumus Excel adalah persamaan matematika yang umumnya melibatkan fungsi excel, operator excel, konstanta dan atau referensi sel atau range untuk menghitung nilai-nilai tertentu dengan tujuan untuk mendapatkan hasil yang diharapkan pada aplikasi microsoft excel.

Tutorial Excel ini menggunakan istilah Fungsi Excel dan Rumus Excel secara berbeda. Fungsi Excel merujuk pada Fungsi Preset yang disediakan oleh Microsoft Excel, sedangkan istilah Rumus Excel lebih kepada cara menggunakan gabungan dari Fungsi-fungsi Excel tersebut dimana cara menggunakan Rumus Excel pada sebuah Cell Excel selalu diawali dengan tanda sama dengan (=). Dengan kata lain satu atau beberapa fungsi excel dapat digunakan dalam sebuah rumus excel, namun sebuah rumus excel tidak harus memuat fungsi excel.

  1. Fungsi IF

    Fungsi IF digunakan untuk menghasilkan Nilai berdasarkan Uji Logika (Benar/Salah) tertentu.

    IF(TesLogika; KondisiTerpenuhi; [KondisiTidakTerpenuhi])

  2. Fungsi IFERROR

    Fungsi IFERROR digunakan untuk menghasilkan sebuah Nilai tertentu jika Nilai Rumus Excel yang diuji menghasilkan Nilai Error/Kesalahan. Sebaliknya jika Nilai yang diuji tidak menghasilkan Error maka IFERROR akan mengembalikan Nilai yang diuji.

    IFERROR(value; value_if_error)

  3. Fungsi AND

    Fungsi AND digunakan untuk melakukan Uji Logika yang menghasilkan Nilai TRUE jika semua argumen yang diuji bernilai TRUE.

    AND(Logika1; [Logika2]; ...)

  4. Fungsi OR

    Fungsi OR digunakan untuk melakukan Uji Logika yang menghasilkan Nilai TRUE jika salah satu argumen yang diuji bernilai TRUE.

    OR(Logika1; [Logika2]; ...)

  5. Fungsi NOT

    Fungsi NOT digunakan untuk membalikkan Nilai Logika TRUE menjadi FALSE dan membalikkan Nilai Logika FALSE menjadi TRUE.

    NOT(logical)

  1. Fungsi LOWER

    Fungsi LOWER digunakan untuk mengubah SEMUA HURUF pada sebuah teks menjadi huruf KECIL semua.

    LOWER(Teks)

  2. Fungsi UPPER

    Fungsi UPPER digunakan untuk mengubah kapitalisasi SEMUA HURUF pada sebuah teks menjadi huruf BESAR semua.

    UPPER(Teks)

  3. Fungsi PROPER

    Fungsi PROPER digunakan untuk mengubah HURUF AWAL setiap kata pada teks menjadi huruf besar.

    PROPER(Teks)

  4. Fungsi LEFT

    Fungsi LEFT digunakan untuk mengambil beberapa karakter teks dari arah KIRI atau awal sebuah teks.

    LEFT(Teks; [JumlahKarakter])

  5. Fungsi RIGHT

    Fungsi RIGHT digunakan untuk mengambil beberapa karakter teks dari arah KANAN atau akhir sebuah teks.

    RIGHT(Teks; [JumlahKarakter])

  6. Fungsi MID

    Fungsi MID digunakan untuk mengambil beberapa karakter teks dari posisi tertentu (AWAL/TENGAH) sebanyak beberapa karakter tertentu.

    MID(Teks; AwalKarakter; JumlahKarakter)

  7. Fungsi FIND

    Fungsi FIND digunakan untuk mencari teks pada teks lain (Case Sensitive) dan menghasilkan sebuah angka yang menunjukkan posisi teks yang dicari.

    FIND(find_text; within_text; [start_num])

  8. Fungsi SEARCH

    Fungsi SEARCH digunakan untuk mencari teks pada teks lain (Non Case Sensistive) dan menghasilkan sebuah angka yang menunjukkan posisi teks yang dicari.

    SEARCH(find_text, within_text, [start_num])

  9. Fungsi LEN

    Fungsi LEN digunakan untuk mengetahui jumlah karakter sebuah teks.

    LEN(Teks)

  10. Fungsi TRIM

    Fungsi TRIM digunakan untuk menghapus spasi berlebihan (spasi ganda, spasi pada awal teks dan spasi pada akhir teks).

    TRIM(Teks)

  11. Fungsi SUBSTITUTE

    Fungsi SUBSTITUTE digunakan untuk mengubah teks tertentu dengan teks lain.

    SUBSTITUTE(text; old_text; new_text; [instance_num])

  12. Fungsi REPLACE

    Fungsi REPLACE digunakan untuk mengganti teks pada posisi tertentu dengan teks lain.

    REPLACE(old_text; start_num; num_chars; new_text)

  13. Fungsi CONCATENATE

    Fungsi CONCATENATE digunakan untuk menggabungkan beberapa teks pada pada excel ke dalam satu sel.

    CONCATENATE(Teks1; [Teks2]; ...)

  14. Fungsi TEXT

    Fungsi TEXT digunakan untuk mengkonversi angka(numeric) termasuk format tanggal dan waktu menjadi teks(string) dalam format yang ditentukan.

    TEXT(Angka;"KodeFormat")

  15. Fungsi EXACT

    Fungsi EXACT digunakan untuk membandingkan 2 teks apakah identik sama persis atau tidak.

    EXACT(Teks1; Teks2)

  1. Fungsi NOW

    Fungsi NOW digunakan untuk mendapatkan nomor seri TANGGAL dan WAKTU saat ini.

    NOW()

  2. Fungsi TODAY

    Fungsi TODAY digunakan untuk mendapatkan nomor seri TANGGAL saat ini.

    TODAY()

  3. Fungsi DATE

    Fungsi DATE digunakan untuk mendapatkan format (nomor seri) TANGGAL berdasarkan informasi tanggal, bulan dan tahun yang Anda tentukan.

    DATE(Tahun;Bulan;Tanggal)

  4. Fungsi DATEVALUE

    Fungsi DATEVALUE digunakan untuk mengonversi tanggal dalam bentuk teks menjadi nomor seri tanggal excel.

    DATEVALUE(TeksTanggal)

  5. Fungsi DATEDIF

    Fungsi DATEDIF digunakan untuk menghasilkan selisih tangal, bulan atau tahun diantara dua tanggal tertentu. Biasanya digunakan untuk menghitung umur atau masa kerja.

    DATEDIF(TglAwal;TglAkhir;KodeSatuan)

  6. Fungsi DAY

    Fungsi DAY digunakan untuk mendapatkan nilai angka TANGGAL(1-31) dari format tanggal(DATE).

    DAY(Tanggal)

  7. Fungsi DAYS (Excel 2013+)

    Fungsi DAYS digunakan untuk menghitung jumlah hari diantara dua tanggal.

    DAYS(TanggalAkhir; TanggalAwal)

  8. Fungsi DAYS360

    Fungsi DAYS360 digunakan untuk menghitung jumlah hari diantara dua tanggal dengan asumsi semua bulan dalam setahun adalah 30 hari atau 360 hari dalam setahun.

    DAYS360(TanggalAwal; TanggalAkhir; [Metode])

  9. Fungsi MONTH

    Fungsi MONTH digunakan untuk mendapatkan nilai angka BULAN (1-12) dari sebuah format tanggal(DATE).

    MONTH(Tanggal)

  10. Fungsi YEAR

    Fungsi YEAR digunakan untuk mendapatkan nilai angka TAHUN dari sebuah format tanggal(DATE).

    YEAR(Tanggal)

  11. Fungsi TIME

    Fungsi TIME digunakan untuk mendapatkan format WAKTU berdasarkan informasi jam, menit dan detik tertentu.

    TIME(Jam;Menit;Detik)

  12. Fungsi TIMEVALUE

    Fungsi TIMEVALUE digunakan untuk menkonversi TEKS yang menunjukkan waktu menjadi nilai desimal atau format waktu yang dikenal oleh excel.

    TIMEVALUE(TeksWaktu)

  13. Fungsi SECOND

    Fungsi SECOND digunakan untuk mendapatkan nilai angka DETIK dari sebuah format waktu(TIME).

    SECOND(Waktu)

  14. Fungsi MINUTE

    Fungsi MINUTE digunakan untuk mendapatkan nilai MENIT dari sebuah format waktu (TIME).

    MINUTE(Waktu)

  15. Fungsi HOUR

    Fungsi HOUR digunakan untuk mendapatkan nilai JAM dari sebuah format waktu (TIME).

    HOUR(Waktu)

  16. Fungsi EDATE

    Fungsi EDATE digunakan untuk menghasilkan nomor seri tanggal yang menunjukkan jumlah bulan sebelum atau sesudah tanggal tertentu.

    EDATE(TanggalAwal; Bulan)

  17. Fungsi EOMONTH

    Fungsi EOMONTH digunakan untuk menghasilkan nomor seri dari hari terakhir bulan sebelum atau sesudah jumlah bulan yang ditentukan.

    EOMONTH(TanggalAwal; Bulan)

  18. Fungsi NETWORKDAYS

    Fungsi NETWORKDAYS digunakan untuk menghasilkan jumlah hari kerja di antara dua tanggal (Mengecualikan akhir pekan dan hari libur).

    NETWORKDAYS(Tanggal_Awal; Tanggal_Akhir, [Hari_Libur])

  19. Fungsi NETWORKDAYS.INTL (Excel 2010+)

    Fungsi NETWORKDAYS.INTL digunakan untuk menghasilkan jumlah hari kerja di antara dua tanggal (Dengan mengatur parameter akhir pekan dan hari libur).

    NETWORKDAYS.INTL(Tanggal_Awal; Tanggal_Akhir; [Akhir_Pekan]; [Hari_Libur])

  20. Fungsi WORKDAY

    Fungsi WORKDAY digunakan untuk menghasilkan tanggal selesai kerja (Mengecualikan akhir pekan dan hari libur).

    WORKDAY(Tanggal_Awal;Jumlah_Hari; [Hari_Libur])

  21. Fungsi WORKDAY.INTL (Excel 2010+)

    Fungsi WORKDAY.INTL digunakan untuk menentukan tanggal selesai kerja (Dengan mengatur parameter akhir pekan dan hari libur).

    WORKDAY.INTL(Tanggal_Awal;Jumlah_Hari; [Akhir_Pekan]; [Hari_Libur])

  1. Fungsi LOOKUP

    Fungsi LOOKUP untuk melakukan pencarian data pada satu baris (row) atau satu kolom (column) tabel data atau sebuah array.

    LOOKUP(Lookup_value; Lookup_vector; [Result_vector])

  2. Fungsi VLOOKUP

    Fungsi VLOOKUP untuk mencari sebuah nilai pada kolom(column) pertama sebuah data tabel untuk mendapatkan sebuah nilai data tertentu pada kolom lain yang berada pada baris(row) yang sama dengan hasil pencarian. Data disusun secara vertikal.

    VLOOKUP(NilaiYangDiCari; TabelReferensi; NomorKolom; [RangeLookup])

  3. Fungsi HLOOKUP

    Fungsi HLOOKUP untuk mencari sebuah nilai pada baris(row) pertama sebuah data tabel untuk mendapatkan nilai data tertentu pada baris lain yang berada pada kolom(column) yang sama dengan hasil pencarian. Data disusun secara horizontal.

    HLOOKUP(NilaiYangDiCari; TabelReferensi; NomorBaris; [RangeLookup])

  4. Fungsi INDEX

    Fungsi INDEX untuk mendapatkan sebuah nilai data berdasarkan input/informasi kolom dan baris tertentu.

    INDEX(array; Nomor_Baris; [Nomor_Kolom])

  5. Fungsi MATCH

    Fungsi MATCH untuk mendapatkan angka urut posisi sebuah nilai data pada sebuah array. Biasanya dikombinasikan dengan Fungsi INDEX dalam penggunannya.

    MATCH(lookup_value; lookup_array; [match_type])

  8. Fungsi INDIRECT

    INDIRECT(TeksReferensi; JenisReferensi)

  9. Fungsi ROW

    Fungsi ROW untuk mendapatkan posisi baris(Row) sebuah sel.

    ROW([Referensi])

  10. Fungsi ROWS

    Fungsi ROWS untuk mendapatkan jumlah baris sebuah array atau referensi.

    ROWS(Array)

  11. Fungsi COLUMN

    Fungsi COLUMN untuk mendapatkan posisi kolom(Column) sebuah sel.

    COLUMN([Referensi])

  12. Fungsi COLUMNS

    Fungsi COLUMNS untuk mendapatkan jumlah kolom sebuah array atau referensi.

    COLUMNS(Array)

  13. Fungsi CHOOSE

    Fungsi CHOOSE digunakan untuk memilih nilai dari daftar nilai yang disediakan.

    CHOOSE(IndexNumber; Nilai1; [Nilai2]; ...)

  14. Fungsi TRANSPOSE

    Fungsi TRANSPOSE digunakan untuk mengubah orientasi data vertikal menjadi horizontal atau sebaliknya.

    TRANSPOSE(Array)

  1. Fungsi SUBTOTAL
