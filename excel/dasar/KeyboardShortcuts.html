unduh file latihan

Pintasan keyboard memungkinkan Anda melakukan berbagai hal dengan keyboard daripada mouse untuk meningkatkan kecepatan.

Dasar

Pilih sel B2.

1. Untuk memilih seluruh range, tekan Ctrl+A (jika Anda menekan Ctrl+A sekali lagi Excel memilih seluruh lembar).

2. Untuk menyalin range, tekan Ctrl+C (untuk memotong range, tekan Ctrl+X).

3. Pilih sel A6 dan tekan Ctrl+V untuk menempelkan range ini.

4. Untuk membatalkan operasi ini, tekan Ctrl+Z

Memindah (Moving)

Pilih sel B2.

1. Untuk berpindah ke bagian bawah range dengan cepat, tahan Ctrl dan tekan ↓

2. Untuk bergerak cepat ke kanan range, tahan Ctrl dan tekan →

Cobalah sendiri. Tahan Ctrl dan tekan tombol panah untuk berpindah dari ujung ke ujung.

Memilih (Selecting)

Pilih sel A1.

1. Untuk memilih sel saat bergerak ke bawah, tahan Shift dan tekan ↓ beberapa kali.

2. Untuk memilih sel sambil bergerak ke kanan, tahan Shift dan tekan → beberapa kali.

Rumus (formulas)

Pilih sel F2.

1. Untuk memasukkan fungsi SUM dengan cepat, tekan Alt+++=, dan tekan Enter.

2. Pilih sel F2, tahan Shift dan tekan ↓ dua kali.

3. Untuk mengisi rumus ke bawah, tekan Ctrl+D (bawah).

Catatan: dengan cara yang sama, Anda dapat mengisi rumus dengan benar dengan menekan Ctrl+R (kanan).

Memformat (Formatting)

Pilih range B2:F4.

1. Untuk meluncurkan kotak dialog 'Format sel', tekan Ctrl+1

2. Tekan TAB dan tekan ↓ dua kali untuk memilih format Currency (Mata Uang).

3. Tekan TAB dan tekan ↓ dua kali untuk mengatur jumlah tempat desimal ke 0.

4. Tekan Enter.

Hasil:

5. Untuk menebalkan range dengan cepat, pilih range dan tekan Ctrl+B

Fungsi Tombol (Function Keys)

F1 | F2 | F3 | F4 | F5 | F6 | F7 | F8 | F9 | F10 | F11 | F12

Fungsi Tombol memungkinkan Anda melakukan berbagai hal dengan keyboard daripada mouse untuk meningkatkan kecepatan.

F1

Membuka Bantuan Excel.

F2

Memindahkan titik penyisipan ke akhir konten sel yang aktif. Misalnya, pilih sel B6 di bawah ini dan tekan F2.

Catatan: matikan Izinkan pengeditan langsung di sel (File, Opsi, Tingkat Lanjut, Opsi Pengeditan) dan menekan F2 akan memindahkan titik penyisipan ke rumus bar.

F3

Menampilkan kotak dialog Tempel Nama. Misalnya, pilih sel E2 di bawah ini, ketik =SUM(, tekan F3 dan pilih nama.

Catatan: tentu saja, pertama-tama buat setidaknya satu nama range.

F4

Siklus melalui semua 4 jenis referensi sel (absolut, referensi campuran (2x) dan relatif). Misalnya, pilih sel B5 di bawah ini, klik di rumus bar, pindahkan titik penyisipan ke dalam atau ke kanan G2, dan tekan F4.

Catatan: jika Anda tidak mengedit sel, F4 mengulangi tindakan terakhir, jika memungkinkan.

F5

Menampilkan kotak dialog Go To. Misalnya, untuk memilih sel C15, dalam kotak Referensi, ketik C15, dan klik OK.

Catatan: Anda juga dapat memilih nama range, atau klik Khusus untuk memilih semua sel dengan cepat dengan rumus, komentar, pemformatan bersyarat, konstanta, validasi data, dll.

F6

Pindah ke panel berikutnya di lembar kerja yang telah dipisah.

F7

Menampilkan kotak dialog Ejaan (sama seperti mengklik Ejaan pada tab Review).

F8

Mengaktifkan/menonaktifkan mode Perpanjang. Jika mode Perluas diaktifkan, pilih sel A1 dan tekan → dan beberapa kali.

Catatan: jika mode Perpanjang dimatikan, tahan Shift dan tekan → dan beberapa kali.

F9

Menghitung buku kerja. Secara default, setiap kali Anda mengubah nilai, Excel secara otomatis menghitung buku kerja. Aktifkan Perhitungan manual (pada tab Rumus, dalam grup Perhitungan, klik Opsi Perhitungan, Manual) dan ubah nilai di sel A1 dari 5 menjadi 6.

Tekan F9.

Catatan: jika Anda mengedit sel, F9 mengganti rumus dengan hasilnya sendiri.

.kbd>F10/div>

Menunjukkan tip kunci (sama seperti menekan Alt). Tips Kunci memungkinkan Anda dengan cepat melakukan tugas apa pun yang tersedia di Ribbon tanpa menggunakan mouse.

.kbd>F11/div>

Membuat lembar bagan dari range yang dipilih.

.kbd>F12/div>

Menampilkan kotak dialog Save As.

Catatan: untuk mengubah lokasi file default, pada tab File, klik Opsi, Simpan.

Simpan Sebagai (Save As)

Simpan Sebagai | Menyimpan

Gunakan pintasan F12 untuk menampilkan kotak dialog Simpan Sebagai di Excel. Tekan Ctrl+S untuk menyimpan buku kerja yang ada. Ini adalah praktik yang baik untuk menyimpan secara berkala saat Anda sedang mengerjakan file Excel Anda.

Simpan Sebagai

Untuk menampilkan kotak dialog Save As, jalankan langkah-langkah berikut.

1. Pada tab File, klik Simpan Sebagai.

2. Klik Telusuri.

Hasil:

Catatan: daripada menjalankan langkah 1 dan langkah 2, cukup gunakan pintasan F12.

3. Pilih lokasi file.

4. Masukkan nama file.

5. Klik Simpan.

Menyimpan

Sekali lagi, ini adalah praktik yang baik untuk menyimpan secara berkala saat Anda mengerjakan file Excel Anda. Sering menabung dapat mencegah pekerjaan Anda hilang.

1. Mengedit buku kerja yang ada.

3. Tekan F12 (Excel membuka kotak dialog Save As lagi) untuk membuat salinan buku kerja ini sambil menyimpan aslinya.

Kunci Scrol (Scroll Lock)

Apa itu Scroll Lock? | Matikan Scroll Lock

Saat Scroll Lock dihidupkan, Anda dapat menggunakan tombol panah untuk menggulir lembar kerja tanpa mengubah sel yang aktif. Excel menampilkan Scroll Lock di status bar.

Apa itu Scroll Lock?

Pertama, ketika Scroll Lock dimatikan, Anda dapat menggunakan tombol panah untuk mengubah sel yang aktif.

1. Misalnya, pilih sel A1.

2. Tekan ↓ dua kali.

Saat Scroll Lock dihidupkan, Anda dapat menggunakan tombol panah untuk menggulir lembar kerja tanpa mengubah sel yang aktif.

3. Pilih sel A1 lagi.

4. Sekali lagi, tekan ↓ dua kali.

Catatan: tekan Ctrl+Backspace untuk menggulir kembali ke sel yang aktif.

Matikan Scroll Lock

Untuk mematikan scroll lock, jalankan langkah-langkah berikut.

1. Tekan tombol Scroll Lock (Scroll Lock atau ScrLk) pada keyboard Anda. Selesai.

Jika keyboard Anda tidak memiliki tombol Scroll Lock, luncurkan keyboard di layar.

2. Klik Start > Settings > Ease of Access > Keyboard > Gunakan Papan Ketik di Layar (atau tekan tombol logo Windows+Ctrl+O).

3. Klik tombol ScrLk.

Catatan: untuk mengaktifkan scroll lock, cukup ulangi langkah 1 atau langkah 2 dan 3.

Saat Scroll Lock dihidupkan, Excel menampilkan Scroll Lock di status bar (hanya jika status bar dikustomisasi untuk menampilkan status Scroll Lock).

4. Klik kanan status bar untuk menampilkan atau menyembunyikan status Scroll Lock.

Poin Bullet (Bullet Points)

Halaman ini menggambarkan tiga cara untuk menyisipkan poin-poin di Excel. Anda dapat menggunakan pintasan keyboard, menyisipkan simbol, atau membuat format angka khusus.

Jika Anda memiliki papan angka di sisi kanan keyboard Anda, jalankan langkah 1 atau langkah 2.

1. Untuk menyisipkan titik peluru bulat yang terisi, tekan Alt+Numpad 7.

2. Untuk menyisipkan titik peluru bulat berongga, tekan Alt+Numpad 9.

Jika Anda tidak memiliki papan angka di sisi kanan keyboard, jalankan langkah 3 dan langkah 4.

3. Pada tab Insert, di grup Symbol, klik Symbol.

4. Pilih font dari daftar drop-down, ketik 2022 di kotak Kode Character dan klik Insert.

Hasil.

5. Untuk menyisipkan beberapa poin bullet dalam satu sel, tekan Alt+Enter untuk memulai baris baru di sel Excel.

Buat format angka khusus dan memasukkan daftar berpoin di Excel akan menjadi hal yang mudah.

6. Pilih range A1:A3.

7. Klik kanan, lalu klik Format Cells.

8. Pilih Kustom.

9. Pada kotak Type, tekan Alt+Numpad 7 untuk memasukkan bullet point (atau copy/paste bullet point), masukkan spasi dan ketik @

10. Klik Oke.

11. Anda sekarang dapat membuat daftar berpoin hanya dengan mengetik item daftar. Excel secara otomatis menambahkan poin-poin!

Catatan: gunakan Format Painter untuk menyalin format poin-poin dengan cepat ke sel lain.

Jeda Baris (Line Break)

Masukkan jeda baris di Excel dengan menekan Alt+Enter. Saat menggabungkan string, gunakan CHAR(10) untuk menyisipkan jeda baris.

Untuk menyisipkan jeda baris, jalankan langkah-langkah berikut.

1. Misalnya, masukkan string teks panjang di sel A1.

2. Klik dua kali sel A1.

3. Tempatkan kursor Anda di lokasi di mana Anda ingin garis putus.

4. Tekan Alt+Enter.

5. Klik pada batas kanan header kolom A dan seret pemisah untuk menambah lebar kolom.

6. Klik dua kali batas bawah header baris 1 untuk menyesuaikan tinggi baris secara otomatis.

Hasil:

Catatan: untuk menghapus jeda baris, klik dua kali sel, tempatkan kursor Anda di awal baris dan tekan Backspace.

7. Saat menggabungkan string, gunakan CHAR(10) untuk menyisipkan jeda baris. Misalnya, alamat di bawah ini berisi dua jeda baris. Gunakan operator & untuk menggabungkan string.

Catatan: jangan lupa untuk mengaktifkan pembungkusan teks.

Tampilkan Rumus (Show Formulas)

Secara default, Excel memperlihatkan hasil rumus. Untuk menampilkan rumus daripada hasilnya, tekan Ctrl+` (Anda dapat menemukan kunci ini di atas tombol tab).

1. Saat Anda memilih sel, Excel memperlihatkan rumus sel di rumus bar.

2. Untuk menampilkan semua rumus, di semua sel, tekan Ctrl+` (Anda dapat menemukan tombol ini di atas tombol tab).

3. Tekan ↓ dua kali.

Catatan: seperti yang Anda lihat, Excel menyoroti semua sel yang direferensikan oleh rumus.

4. Untuk menyembunyikan semua rumus, tekan Ctrl+` lagi.

5. Jika Anda tidak dapat menemukan aksen grave (`) pada keyboard Anda, pada tab Rumus, dalam grup Audit Rumus, klik Perlihatkan Rumus.

Catatan: untuk menyembunyikan semua rumus, klik lagi Perlihatkan Rumus. Kunjungi halaman kami tentang audit formula di Excel untuk mempelajari lebih lanjut tentang melacak preseden, melacak tanggungan, pemeriksaan kesalahan, dll.
