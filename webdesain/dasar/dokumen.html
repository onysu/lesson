Dasar HTML

Dalam bab ini kita akan menunjukkan beberapa contoh HTML dasar. Jangan khawatir jika kami menggunakan tag yang belum Anda pelajari.

Semua dokumen HTML harus dimulai dengan deklarasi tipe dokumen: <!DOCTYPE html>. Dokumen HTML itu sendiri dimulai dengan <html> dan diakhiri dengan </html>. Bagian yang terlihat dari dokumen HTML adalah antara <body> dan </body>.


Struktur HTML5 :

<!DOCTYPE html>
<html lang="en">

  <head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Judul Halaman</title>
  </head>

  <body>
    <h1>Hello, world!</h1>
    <p>Ini adalah paragraf</p>
  </body>

</html>

Penjelasannya :

  • <html> adalah elemen root elemen dan mendefinisikan dokumen HTML secara keseluruhan. Ini memiliki tag awal <html> dan tag akhir </html>. Kemudian, di dalam <html> elemen ada <body> elemen:

    <body>

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

    </body>

  • <body> mendefinisikan tubuh dokumen. Ini memiliki tag awal <body> dan tag akhir </body>. Kemudian, di dalam <body> elemen ada dua elemen lainnya: <h1> dan <p>:

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

  • <h1> elemen mendefinisikan heading (judul). Ini memiliki tag awal <h1> dan tag akhir </h1>:

    <h1>Judul Pertama Saya</h1>

  • <p> elemen mendefinisikan sebuah paragraf. Ini memiliki tag awal <p> dan tag akhir </p>:

    <p>Paragraf pertama saya.</p>


Jangan abaikan tag akhir!

Beberapa elemen HTML akan ditampilkan dengan benar, bahkan jika Anda lupa tag akhir:
Contoh : 

<html>
<body>

  <p>Ini adalah paragraf
  <p>Ini adalah paragraf

</body>
</html>

Cobalah sendiri

Warning!!!, jangan pernah mengandalkan ini! Hasil dan kesalahan yang tidak terduga dapat terjadi jika Anda lupa tag akhir!.


<!DOCTYPE html> Deklarasi mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
<html> Elemen akar (root) halaman HTML
<head> Elemen berisi informasi meta tentang halaman HTML.
<title> Elemen menentukan judul untuk halaman HTML (yang ditampilkan dalam bar judul browser atau tab halaman ini).
<body> Elemen mendefinisikan tubuh dokumen, dan merupakan wadah untuk semua isi terlihat, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll.
<h1> Elemen mendefinisikan judul.
<p> Elemen mendefinisikan sebuah paragraf.

Visualisasi Struktur HTML5:

<html>
<head>
<title>Judul halaman</title>
</head>
<body>
<h1>Judul halaman</h1>
<p>Ini adalah paragraf.</p>
<p>Ini adalah paragraf yang lain.</p>
</body>
</html>

Catatan: Konten di dalam bagian <body> (area putih di atas) akan ditampilkan di browser. Konten di dalam elemen <title> akan ditampilkan di bilah judul browser atau di tab halaman.
