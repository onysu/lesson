IfError

unduh file latihan

Gunakan fungsi IFERROR di Excel untuk mengembalikan hasil alternatif, seperti teks, saat rumus mengevaluasi kesalahan.

  1. Misalnya, Excel mengembalikan #DIV/0! kesalahan saat rumus mencoba membagi angka dengan 0.

  2. Jika rumus mengevaluasi kesalahan, fungsi IFERROR di bawah ini mengembalikan pesan ramah.

  3. Jika rumus tidak mengevaluasi kesalahan, fungsi IFERROR hanya mengembalikan hasil rumus.

  4. Misalnya, Excel mengembalikan kesalahan #N/A saat fungsi VLOOKUP tidak dapat menemukan kecocokan.

  5. Jika fungsi VLOOKUP mengevaluasi kesalahan, fungsi IFERROR di bawah ini mengembalikan pesan ramah.

  6. Jika fungsi VLOOKUP tidak mengevaluasi kesalahan, fungsi IFERROR hanya mengembalikan hasil fungsi VLOOKUP.

Catatan: fungsi IFERROR menangkap kesalahan berikut: #DIV/0!, #N/A, #VALUE!, #REF!, #NUM!, #NAME? dan #NULL!. Misalnya, fungsi IFERROR yang ditunjukkan di atas juga menangkap #NAME? kesalahan jika Anda tidak sengaja salah mengeja kata VLOOKUP. Gunakan fungsi IFNA di Excel 2013 atau yang lebih baru untuk hanya menangkap kesalahan #N/A.

IsError

unduh file latihan

Contoh ini menggambarkan fungsi ISERROR di Excel.

  1. Misalnya, Excel menampilkan #DIV/0! kesalahan saat rumus mencoba membagi angka dengan 0.

  2. Fungsi ISERROR di Excel memeriksa apakah suatu nilai adalah kesalahan dan mengembalikan TRUE atau FALSE.

  3. Tambahkan fungsi IF. Jika sel berisi kesalahan, nilai 5 dikembalikan. Jika tidak, nilai 100 dikembalikan.

  4. Gunakan fungsi IFERROR untuk mengembalikan hasil rumus (jika tidak ada kesalahan) atau hasil alternatif, seperti teks (jika ada kesalahan).

  5. Gunakan COUNT, IF dan ISERROR untuk menghitung jumlah total kesalahan dalam rentang sel.

Catatan: ini adalah rumus array. Anda dapat menyelesaikan rumus array dengan menekan Ctrl+Shift+Enter. Excel menambahkan kurung kurawal {}. Kunjungi halaman kami tentang Menghitung Kesalahan untuk petunjuk terperinci tentang cara membuat rumus array ini.

Aggregate

unduh file latihan

Fungsi Excel seperti SUM, COUNT, LARGE, dan MAX tidak berfungsi jika rentang menyertakan kesalahan. Namun, Anda dapat dengan mudah menggunakan fungsi AGGREGATE untuk memperbaikinya.

  1. Misalnya, Excel mengembalikan kesalahan jika Anda menggunakan fungsi SUM untuk menjumlahkan rentang dengan kesalahan.

  2. Gunakan fungsi AGGREGATE untuk menjumlahkan rentang dengan kesalahan.

    Penjelasan: argumen pertama (9) memberitahu Excel bahwa Anda ingin menggunakan fungsi SUM. Argumen kedua (6) memberi tahu Excel bahwa Anda ingin mengabaikan nilai kesalahan.

  3. Tidak mudah untuk mengingat argumen mana yang termasuk dalam fungsi yang mana. Untungnya, fitur AutoComplete di Excel membantu Anda dalam hal ini.

  4. Fungsi AGGREGATE di bawah ini menemukan angka terbesar kedua dalam rentang dengan kesalahan.

    Penjelasan: argumen pertama (14) memberi tahu Excel bahwa Anda ingin menggunakan fungsi LARGE. Argumen kedua (6) memberi tahu Excel bahwa Anda ingin mengabaikan nilai kesalahan. Argumen keempat (2) memberi tahu Excel bahwa Anda ingin menemukan angka terbesar kedua.

  5. Fungsi AGGREGATE di bawah ini menemukan nilai maksimum dengan mengabaikan nilai kesalahan dan baris tersembunyi.

Penjelasan: argumen pertama (4) memberitahu Excel bahwa Anda ingin menggunakan fungsi MAX. Argumen kedua (7) memberi tahu Excel bahwa Anda ingin mengabaikan nilai kesalahan dan baris tersembunyi. Dalam contoh ini, baris 2 disembunyikan.

Circular Reference

Referensi Edaran Langsung | Referensi Edaran Tidak Langsung | Temukan Referensi Edaran

unduh file latihan

Sebuah referensi melingkar di Excel terjadi ketika formula secara langsung atau tidak langsung mengacu ke sel sendiri. Ini tidak mungkin.

Referensi Edaran Langsung (Direct Circular Reference)

Mari kita mulai dengan sederhana. Referensi melingkar langsung di Excel terjadi saat rumus secara langsung merujuk ke selnya sendiri.

  1. Misalnya, rumus di sel A3 di bawah ini langsung merujuk ke selnya sendiri (A3). Ini tidak mungkin.

  2. Tekan Enter. Pesan kesalahan referensi melingkar muncul.

  3. Klik Oke. Excel mengembalikan 0.

Referensi Edaran Tidak Langsung (Indirect Circular Reference)

Referensi melingkar tidak langsung di Excel terjadi ketika rumus secara tidak langsung merujuk ke selnya sendiri. Mari kita lihat contoh sederhana.

  1. Misalnya, sel D2 di bawah ini berisi nilai 25.

  2. Sel F4 mengacu pada sel D2.

  3. Sel C5 mengacu pada sel F4.

  4. Sejauh ini, semuanya baik-baik saja. Sekarang ganti nilai 25 di sel D2 dengan rumus yang ditunjukkan di bawah ini.

  5. Tekan Enter. Pesan kesalahan referensi melingkar muncul.

  6. Klik Oke. Excel mengembalikan 0.

  7. Pada tab Rumus, dalam grup Audit Rumus, klik Perlihatkan Rumus.

  8. Sel D2 mengacu pada sel C5. Sel C5 mengacu pada sel F4. Sel F4 merujuk kembali ke sel D2.

Kesimpulan: rumus di sel D2 secara tidak langsung mengacu pada selnya sendiri (D2). Ini tidak mungkin. Rumus di sel C5 dan sel F4 juga merujuk kembali ke sel mereka sendiri (itu lingkaran).

Temukan Referensi Edaran (Find Circular References)

Untuk menemukan referensi melingkar di Excel, dalam grup Audit Rumus, klik panah bawah di samping Pemeriksaan Kesalahan dan klik Referensi Sel.

  1. Lembar kami berisi 2 referensi melingkar. 1 referensi melingkar langsung (A3) dan 1 referensi melingkar tidak langsung (D2, C5 dan F4).

  2. Pada tab Rumus, dalam grup Audit Rumus, klik panah bawah di samping Pemeriksaan Kesalahan.

  3. Klik Referensi Edaran. Excel menunjukkan alamat sel dari satu referensi melingkar.

  4. Anda juga dapat melihat bilah status. Ini jauh lebih mudah.

  5. Perbaiki referensi melingkar ini.

    Untuk menemukan lebih banyak referensi melingkar, ulangi langkah-langkah ini.

  6. Pada tab Rumus, di grup Audit Rumus, klik panah bawah di sebelah Pemeriksaan Kesalahan.

  7. Klik Referensi Edaran. Excel menunjukkan alamat sel referensi melingkar tidak langsung.

  8. Status bar hanya menampilkan satu alamat sel.

Catatan: perbaiki referensi melingkar ini dengan mengganti salah satu rumus dengan nilai. Akibatnya, peringatan di bilah status akan hilang.

Audit Rumus (Formula Auditing)

Trace Precedents | Hapus Panah | Trace Dependents | Tampilkan Rumus | Cek Error | Evaluasi Rumus

Audit rumus di Excel memungkinkan Anda menampilkan hubungan antara rumus dan sel. Contoh di bawah ini membantu Anda menguasai Formula Auditing dengan cepat dan mudah.

Lacak Preseden (Trace Precedents)

Anda harus membayar $96.00. Untuk memperlihatkan panah yang menunjukkan sel mana yang digunakan untuk menghitung nilai ini, jalankan langkah-langkah berikut.

  1. Pilih sel C13.

  2. Pada tab Rumus, dalam grup Audit Rumus, klik Lacak Preseden.

    Hasil:

    Seperti yang diharapkan, Total biaya dan ukuran Grup digunakan untuk menghitung Biaya per orang.

  3. Klik Jejak Preseden lagi.

    Seperti yang diharapkan, biaya yang berbeda digunakan untuk menghitung Total biaya.

Hapus Panah (Remove Arrows)

Untuk menghapus panah, jalankan langkah-langkah berikut.

Pada tab Rumus, dalam grup Audit Rumus, klik Hapus Panah.

Melacak Tanggungan (Trace Dependents)

Untuk memperlihatkan panah yang menunjukkan sel mana yang bergantung pada sel yang dipilih, jalankan langkah-langkah berikut.

  1. Pilih sel C12.

  2. Pada tab Rumus, dalam grup Audit Rumus, klik Lacak Tanggungan.

    Hasil:

    Seperti yang diharapkan, Biaya per orang tergantung pada ukuran Grup.

Tampilkan Rumus (Show Formulas)

Secara default, Excel memperlihatkan hasil rumus. Untuk menampilkan rumus alih-alih hasilnya, jalankan langkah-langkah berikut.

Pada tab Rumus, dalam grup Audit Rumus, klik Perlihatkan Rumus.

Hasil:

Catatan: daripada mengklik Tampilkan Rumus, tekan Ctrl+` (Anda dapat menemukan kunci ini di atas tombol tab).

Cek Error (Error Checking)

Untuk memeriksa kesalahan umum yang terjadi dalam rumus, jalankan langkah-langkah berikut.

  1. Masukkan nilai 0 ke dalam sel C12.

  2. Pada tab Rumus, dalam grup Audit Rumus, klik Pemeriksaan Kesalahan.

    Hasil. Excel menemukan kesalahan di sel C13. Rumus mencoba membagi angka dengan 0.

Evaluasi Rumus (Evaluate Formula)

Untuk men-debug rumus dengan mengevaluasi setiap bagian rumus satu per satu, jalankan langkah-langkah berikut.

  1. Pilih sel C13.

  2. Pada tab Rumus, dalam grup Audit Rumus, klik Evaluasi Rumus.

  3. Klik Evaluasi empat kali.

    Excel menunjukkan hasil rumus.

Kesalahan Titik Mengambang (Floating Point Errors)

unduh file latihan

Excel menyimpan dan menghitung angka floating point. Terkadang, hasil dari sebuah rumus adalah perkiraan yang sangat dekat.

  1. Sebagai contoh, perhatikan rumus di bawah ini. Sekilas, semuanya terlihat baik-baik saja.

  2. Namun, jika kita menunjukkan 16 tempat desimal, kita dapat melihat bahwa salah satu hasilnya adalah perkiraan yang sangat dekat.

    Anda tidak perlu khawatir tentang kesalahan floating point. Mereka langka.

  3. Bahkan jika lembar kerja Anda berisi kesalahan floating point, dalam banyak kasus, ini tidak menyebabkan masalah. Namun, jika Anda membandingkan nilai di sel C8 dengan nilai lain, masalah berikut dapat terjadi.

  4. Gunakan fungsi ROUND untuk memperbaikinya.

IFNA

unduh file latihan

Gunakan fungsi IFNA di Excel untuk mengganti kesalahan #N/A dengan pesan ramah. Fungsi IFNA hanya menangkap kesalahan #N/A.

  1. Misalnya, Excel mengembalikan kesalahan #N/A saat fungsi VLOOKUP tidak dapat menemukan kecocokan.

  2. Gunakan fungsi IFNA untuk mengganti kesalahan #N/A dengan pesan ramah.

  3. Jika fungsi VLOOKUP tidak mengevaluasi kesalahan #N/A, fungsi IFNA hanya mengembalikan hasil fungsi VLOOKUP.

    Catatan: fungsi IFNA diperkenalkan di Excel 2013. Jika Anda menggunakan Excel 2010 atau Excel 2007, cukup ganti IFNA dengan IFERROR. Ingat, fungsi IFERROR juga menangkap kesalahan lain. Misalnya, #NAME? kesalahan jika Anda tidak sengaja salah mengeja kata VLOOKUP.
