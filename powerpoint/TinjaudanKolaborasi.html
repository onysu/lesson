edit hotspots

Review Changes in the Slides Pane

Some changes, like Inserted or Deleted slides, will show up on the Slides pane. Click the revisions icon to see the change, then click the check box if you want to accept it.

Accepting Multiple Changes

In addition to reviewing changes individually, you can accept multiple changes at once. Click the Accept drop-down arrow, then choose either Accept All Changes to This Slide or Accept All Changes to the Presentation.

Compare

To compare your presentation with another version, click the Compare command. Next, locate and select the desired file.

Heading

Next Change

Use the Next command to view the next change in the presentation. You can also click Previous to go back to the previous one.

Ending the Review

When you're satisfied with the changes you've reviewed, click End Review. Only the changes you've accepted will be applied to the final presentation.

Revisions Pane: Slides

The Slides tab actually lets you preview the revised slide, so you can see the changes in action (as opposed to the Details tab, which only lists the changes).

Revisions Pane: Details

The Details tab lists every change that has been made to the current slide, as well as any comments. Click a change to see its location on the slide.

Review Changes on the Slide

Click the revisions icon anywhere it appears on a slide to see the changes that were made in that spot. To accept a revision, click the check box beside it. To reject a revision, leave the box unchecked.

Untuk menjalankan pemeriksaan ejaan:

1.Dari tab Tinjau, klik perintah Ejaan.

2.The Spelling pane akan muncul di sebelah kanan. Untuk setiap kesalahan dalam presentasi Anda, PowerPoint akan mencoba menawarkan satu atau beberapa saran. Anda dapat memilih saran dan klik Ubah untuk memperbaiki kesalahan.

3.PowerPoint akan menelusuri setiap kesalahan hingga Anda meninjau semuanya. Setelah kesalahan terakhir ditinjau, kotak dialog akan muncul yang mengonfirmasi bahwa pemeriksaan ejaan selesai. Klik Oke.

Jika tidak ada saran yang diberikan, Anda dapat mengetik ejaan yang benar secara manual di slide.

Mengabaikan ejaan "kesalahan"

Pemeriksaan ejaan tidak selalu benar. Kadang-kadang mungkin berpikir sebuah kata dieja dengan tidak benar padahal sebenarnya tidak. Ini sering terjadi dengan nama orang dan kata benda, yang mungkin tidak ada dalam kamus. Jika PowerPoint mengatakan ada sesuatu yang salah, Anda dapat memilih untuk tidak mengubahnya menggunakan salah satu dari tiga opsi:

Abaikan : Ini akan melewati kata tanpa mengubahnya.

Abaikan Semua : Ini akan melewati kata tanpa mengubahnya, dan juga akan melewati semua contoh kata lainnya dalam presentasi Anda.

Add : Ini menambahkan kata ke kamus sehingga tidak akan pernah muncul sebagai kesalahan lagi. Pastikan kata dieja dengan benar sebelum memilih opsi ini.

Pemeriksaan ejaan otomatis

Secara default, PowerPoint secara otomatis memeriksa presentasi Anda untuk kesalahan ejaan, jadi Anda mungkin tidak perlu menjalankan pemeriksaan terpisah menggunakan perintah Ejaan. Kesalahan ini ditunjukkan dengan garis bergelombang merah.

Untuk menggunakan fitur pemeriksa ejaan otomatis:

1.Klik kanan kata yang digarisbawahi. Sebuah menu akan muncul.

2.Klik ejaan yang benar dari daftar saran.

3.Koreksi akan muncul dalam presentasi.

Anda juga dapat memilih untuk Mengabaikan kata yang digarisbawahi atau Tambahkan ke Kamus.

Memodifikasi opsi pemeriksaan

PowerPoint memungkinkan Anda untuk mengubah opsi pemeriksaan, memberi Anda lebih banyak kontrol atas cara meninjau teks Anda. Misalnya, Anda dapat menyesuaikan pemeriksaan ejaan otomatis untuk mengubah cara PowerPoint menandai kesalahan ejaan. Anda juga dapat mengaktifkan opsi pemeriksaan tata bahasa untuk membantu Anda menemukan dan memperbaiki kesalahan ejaan kontekstual (seperti mereka vs. mereka ).

Untuk mengubah opsi pemeriksaan:

1.Klik tab File untuk mengakses tampilan Backstage.

2.Pilih Opsi. Sebuah kotak dialog akan muncul.

3.Pilih Pemeriksaan, lalu sesuaikan opsi sesuai kebutuhan. Setelah selesai, klik OK.

Mengomentari presentasi

Saat merevisi atau berkolaborasi pada presentasi, Anda mungkin ingin membuat catatan atau saran tanpa benar-benar mengubah slide. Meninggalkan komentar memungkinkan Anda mencatat sesuatu tanpa mengubah slide itu sendiri. Komentar dapat ditambahkan dan dibaca oleh penulis asli atau pengulas lainnya.

Untuk menambahkan komentar:

1.Pilih teks atau objek (atau klik area slide) tempat Anda ingin komentar muncul.

2.Masuk ke tab Review, lalu klik perintah New Comment.

3.The Komentar pane akan muncul. Dalam contoh kami, ini berisi komentar yang ada oleh ulasan lain ( Javier ), ditambah ruang untuk komentar Anda.

4.Ketik komentar Anda di dalam kotak, lalu tekan Enter atau klik di mana saja di luar kotak untuk menyimpan komentar.

5.Komentar akan diwakili oleh ikon kecil di slide.

Melihat komentar

Anda dapat melihat atau membalas komentar apa pun�termasuk komentar pengulas lain�dengan kembali ke panel Komentar. Cukup klik ikon komentar pada slide, atau klik perintah Tampilkan Komentar pada tab Tinjau.

Untuk mengedit komentar:

1.Di panel Komentar, pilih komentar yang ingin Anda edit.

2.Ketik perubahan yang Anda inginkan, lalu tekan Enter atau klik di mana saja di luar kotak komentar. Perubahan Anda akan diterapkan.

Untuk membalas komentar:

1.Di panel Komentar, klik Balas di bawah komentar yang ingin Anda tanggapi.

2.Ketik tanggapan Anda, lalu tekan Enter atau klik di mana saja di luar kotak. Komentar Anda akan muncul di bawah komentar asli, dan ikon tambahan akan ditambahkan ke slide.

Menghapus komentar

Untuk menghapus komentar, pilih komentar yang ingin Anda hapus, lalu buka tab Review dan klik perintah Delete.

Untuk menghapus beberapa komentar, klik panah tarik-turun di bawah perintah Hapus. Anda dapat menghapus komentar dari slide Anda saat ini atau dari seluruh presentasi.

Membandingkan presentasi

Ada situasi di mana Anda mungkin berakhir dengan lebih dari satu versi presentasi yang sama. Misalnya, Anda dapat membuat beberapa draf, atau kolaborator atau rekan kerja dapat menyimpan salinan unik mereka sendiri.

Anda dapat dengan mudah membandingkan dan menggabungkan beberapa versi menggunakan fitur Bandingkan PowerPoint. Ini memungkinkan Anda untuk melihat perbedaan antara dua versi dari presentasi yang sama, sehingga Anda dapat memutuskan perubahan mana yang akan disertakan dalam versi final.

Pemeriksa Dokumen

Setiap kali Anda membuat atau mengedit presentasi, informasi pribadi tertentu dapat ditambahkan ke file secara otomatis�seperti informasi tentang pembuat dokumen. Anda dapat menggunakan Pemeriksa Dokumen untuk menghapus jenis informasi ini sebelum berbagi presentasi dengan orang lain.

Karena beberapa perubahan mungkin bersifat permanen, sebaiknya gunakan Simpan Sebagai untuk membuat salinan cadangan presentasi Anda sebelum menggunakan Pemeriksa Dokumen.

Untuk menggunakan Pemeriksa Dokumen:

1.Klik tab File untuk pergi ke tampilan Backstage.

2.Dari panel Info, klik Periksa Masalah, lalu pilih Periksa Dokumen dari menu tarik-turun.

3.Pemeriksa Dokumen akan muncul. Centang atau hapus centang pada kotak, bergantung pada konten yang ingin Anda tinjau, lalu klik Inspect. Dalam contoh kita, kita akan menggunakan pilihan default.

4.Hasil pemeriksaan akan menunjukkan tanda seru untuk setiap kategori yang menemukan data yang berpotensi sensitif, dan juga akan memiliki tombol Hapus Semua untuk setiap kategori tersebut. Klik Hapus Semua untuk menghapus data.

5.Setelah selesai, klik Tutup.

Melindungi presentasi Anda

Secara default, siapa pun yang memiliki akses ke presentasi Anda akan dapat membuka, menyalin, dan mengedit kontennya kecuali Anda memproteksinya. Ada beberapa cara untuk melindungi presentasi, tergantung pada kebutuhan Anda.

Untuk melindungi dokumen Anda:

1.Klik tab File untuk pergi ke tampilan Backstage.

2.Dari panel Info, klik perintah Lindungi Presentasi.

3.Di menu tarik-turun, pilih opsi yang paling sesuai dengan kebutuhan Anda. Dalam contoh kita, kita akan memilih Mark as Final. Menandai presentasi Anda sebagai final adalah cara yang baik untuk mencegah orang lain mengedit file, sementara opsi lain memberi Anda lebih banyak kontrol jika diperlukan.

4.Sebuah kotak dialog akan muncul, meminta Anda untuk menyimpan. Klik Oke.

5.Kotak dialog lain akan muncul. Klik Oke.

6.Presentasi akan ditandai sebagai final. Setiap kali orang lain membuka file, bilah akan muncul di bagian atas untuk mencegah mereka mengedit dokumen.

Menandai presentasi sebagai final tidak akan menghalangi seseorang untuk mengeditnya karena mereka cukup memilih Edit Anyway. Jika Anda ingin mencegah orang mengeditnya, Anda dapat menggunakan opsi Batasi Akses sebagai gantinya.

2.Gunakan Pemeriksa Dokumen untuk memeriksa dan menghapus informasi tersembunyi apa pun.

3.Lindungi dokumen dengan menandainya sebagai final.
