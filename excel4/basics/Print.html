Bab ini mengajarkan Anda cara mencetak lembar kerja dan cara mengubah beberapa pengaturan cetak penting di Excel.

Cetak Lembar Kerja

Untuk mencetak lembar kerja di Excel, jalankan langkah-langkah berikut.

1. Pada tab File, klik Print.

2. Untuk melihat preview halaman lain yang akan dicetak, klik 'Next Page' atau 'Previous Page' di bagian bawah jendela.

3. Untuk mencetak lembar kerja, klik tombol Print.

Apa yang Harus Dicetak?

Daripada mencetak seluruh lembar kerja, Anda juga dapat mencetak pilihan saat ini.

1. Pertama, pilih range sel yang ingin Anda cetak.

2. Selanjutnya, di bawah Settings, pilih Print Selection.

3. Untuk mencetak pilihan, klik tombol Print.

Catatan: Anda juga dapat mencetak lembar aktif (pertama pilih lembar dengan menahan Ctrl dan mengklik tab lembar) atau mencetak seluruh buku kerja. Gunakan kotak di sebelah Pages (lihat gambar pertama diatas) untuk hanya mencetak beberapa halaman dokumen Anda. Misalnya, 2 hingga 2 hanya mencetak halaman kedua.

Beberapa Salinan

Untuk mencetak banyak salinan, lakukan langkah-langkah berikut.

1. Gunakan panah di sebelah kotak Copies.

2. Jika satu salinan berisi beberapa halaman, Anda dapat beralih antara Collated dan Uncollated. Misalnya, jika Anda mencetak 6 eksemplar, Collated mencetak seluruh salinan pertama, kemudian seluruh salinan kedua, dll. Uncollated mencetak 6 salinan halaman 1, 6 salinan halaman 2, dll.

Orientasi

Anda dapat beralih antara Portrait Orientation (lebih banyak baris tetapi lebih sedikit kolom) dan Landscape Orientation (lebih banyak kolom tetapi lebih sedikit baris).

Margin Halaman

Untuk menyesuaikan margin halaman, jalankan langkah-langkah berikut.

1. Pilih salah satu margin yang telah ditentukan sebelumnya (Normal, Wide atau Narrow) dari daftar drop-down Margins.

2. Atau klik ikon 'Show Margins' di kanan bawah jendela. Sekarang Anda dapat menyeret garis untuk mengubah margin halaman secara manual.

Penskalaan (Scaling)

Jika Anda ingin memasukkan lebih banyak data pada satu halaman, Anda dapat memasukkan lembar tersebut pada satu halaman. Untuk mencapai ini, jalankan langkah-langkah berikut.

1. Pilih 'Fit Sheet on One Page' dari daftar drop-down Scaling.

Catatan: Anda juga dapat mengecilkan hasil cetak menjadi satu halaman lebar atau satu halaman tinggi. Klik Custom Scaling Options untuk memasukkan persentase penskalaan secara manual atau untuk menyesuaikan hasil cetak dengan jumlah halaman tertentu yang lebar dan tinggi. Hati-hati, Excel tidak memperingatkan Anda saat hasil cetak Anda menjadi tidak terbaca.

Normal

Kapan saja, Anda dapat beralih kembali ke tampilan Normal.

1. Pada tab View, dalam group Workbook Views, klik Normal.

Hasil:

Catatan: jika Anda beralih ke tampilan lain dan kembali ke tampilan Normal, Excel menampilkan page breaks (page breaks). Tutup dan buka kembali file Excel untuk menyembunyikan page breaks ini. Untuk selalu menyembunyikan page breaks untuk lembar kerja ini, klik File, Options, Advanced, scroll ke bawah ke Display options untuk lembar kerja ini dan hapus centang Show page breaks.

Pratinjau Hentian Halaman (Page Break Preview)

Pratinjau Hentian Halaman memberi Anda gambaran umum yang bagus tentang di mana hentian halaman saat Anda mencetak dokumen. Gunakan tampilan ini untuk mengeklik dan menyeret page breaks dengan mudah.

1. Pada tab View, dalam group Workbook Views, klik Pratinjau Hentian Halaman.

Hasil:

Catatan: klik dan seret page breaks untuk memasukkan semua informasi pada satu halaman. Hati-hati, Excel tidak memperingatkan Anda saat hasil cetak Anda menjadi tidak terbaca. Secara default, Excel mencetak ke bawah, lalu selesai. Dengan kata lain, ia mencetak semua baris untuk set kolom pertama. Selanjutnya, ia mencetak semua baris untuk kumpulan kolom berikutnya, dll. (lihat nomor halaman pada gambar di atas untuk mendapatkan ide). Untuk beralih ke Print over, lalu down, klik File, Print, Page Setup, pada tab Sheet, di bawah Page order, klik Over, lalu down.

Hentian Halaman (Page Breaks)

Sisipkan page break di Excel untuk menentukan di mana halaman baru akan dimulai dalam salinan cetak.

Untuk menyisipkan page breaks horizontal, jalankan langkah-langkah berikut.

1. Pilih baris pertama halaman baru.

2. Pada tab Page Layout, dalam grup Page Setup, klik Breaks.

3. Klik Insert Page Break.

Excel menampilkan garis solid (page break manual) untuk menunjukkan kepada Anda di mana halaman baru dimulai. Garis putus-putus adalah page break yang disisipkan secara otomatis oleh Excel.

Catatan: dengan cara yang serupa, Anda dapat memilih kolom untuk menyisipkan page breaks vertikal.

4. Pada tab File, klik Print untuk preview cetak.

Catatan: untuk menghapus page break horizontal, pilih sel di bawah page break yang ingin Anda hapus dan klik Breaks, Remove Page Break. Untuk menghapus semua page break manual, klik Breaks, Reset All Page Breaks. Anda tidak dapat menghapus page breaks otomatis.

Header dan Footer

Contoh ini mengajarkan Anda cara menambahkan informasi ke header (atas setiap halaman yang dicetak) atau footer (bawah setiap halaman yang dicetak) di Excel.

1. Pada tab View, dalam group Workbook Views, klik Page Layout, untuk beralih ke tampilan Page Layout.

2. Klik Add header.

Tab Header & Footer Tools diaktifkan.

3. Pada tab Design, dalam grup Header & Footer Elements, klik Current Date untuk menambahkan tanggal saat ini (atau tambahkan current time, file name, sheet name, dll).

Hasil:

Catatan: Excel menggunakan kode untuk memperbarui header atau footer secara otomatis saat Anda mengubah buku kerja.

4. Anda juga dapat menambahkan informasi di bagian kiri dan kanan header. Misalnya, klik bagian kiri untuk menambahkan nama perusahaan Anda.

5. Klik di tempat lain pada lembar untuk melihat header.

6. Pada tab Design, dalam grup Options, Anda dapat menambahkan header/footer halaman pertama yang berbeda dan header/footer berbeda untuk halaman ganjil (odd) dan genap (even).

7. Pada tab View, dalam group Workbook Views, klik Normal, untuk beralih kembali ke tampilan Normal.

4. Ketik off

5. Pada tab Design, di grup Header & Footer Elements, klik Number of Pages untuk menambahkan jumlah halaman.

Hasil:

Catatan: Excel menggunakan kode untuk memperbarui header atau footer secara otomatis saat Anda mengubah buku kerja.

Mencetak Judul (Print Titles)

Anda dapat menentukan baris dan kolom di Excel yang akan dicetak pada setiap halaman yang dicetak. Ini dapat membuat salinan cetak Anda lebih mudah dibaca.

Untuk mencetak judul, jalankan langkah-langkah berikut.

1. Pada tab Page Layout, dalam grup Page Setup, klik Print Titles.

Kotak dialog Page Setup muncul.

2. Untuk mengulang baris 1 di bagian atas setiap halaman yang dicetak, klik di kotak yang sesuai dan pilih baris 1.

3. Klik Oke.

Catatan: dengan cara yang sama, Anda dapat mengulang kolom di sebelah kiri setiap halaman yang dicetak.

4. Pada tab File, klik Print untuk preview cetak.

Label (LastName, Sales, Country dan Quarter) muncul di halaman 1 dan halaman 2.

Cetak Garis Kisi dan Judul (Print Gridlines and Headings)

Cetak garis kisi (garis horizontal dan vertikal pada lembar kerja Anda) dan judul baris/kolom (1, 2, 3 dst. dan A, B, C dst.) untuk membuat salinan cetak Anda lebih mudah dibaca.

Untuk mencetak garis kisi dan judul di Excel, jalankan langkah-langkah berikut.

1. Pada tab Page Layout, di grup Sheet Options, centang Print di bawah Gridlines dan centang Print di bawah Headings.

2. Pada tab File, klik Print untuk preview cetak.

Area Cetak (Print Area)

Jika Anda mengatur area cetak di Excel, hanya area itu yang akan dicetak. Area cetak disimpan saat Anda menyimpan buku kerja.

Untuk mengatur area cetak, jalankan langkah-langkah berikut.

1. Pilih range sel.

2. Pada tab Page Layout, dalam grup Page Setup, klik Print Area.

3. Klik Set Print Area.

4. Simpan, tutup dan buka kembali file Excel.

5. Pada tab File, klik Print.

Hasil. Lihat preview cetak di bawah ini. Hanya area cetak yang akan dicetak.

Catatan: gunakan Name Manager untuk mengedit dan menghapus area cetak. Pada tab Formulas, dalam grup Defined Names, klik Name Manager.
