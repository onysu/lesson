Untuk memasukkan tanggal hari ini di Excel , gunakan fungsi TODAY . Untuk memasukkan tanggal dan waktu saat ini, gunakan fungsi SEKARANG. Untuk memasukkan tanggal dan waktu saat ini sebagai nilai statis, gunakan pintasan keyboard.

Hari ini dan Sekarang

1. Untuk memasukkan tanggal hari ini di Excel, gunakan fungsi TODAY.

Catatan: fungsi TODAY tidak membutuhkan argumen. Tanggal ini akan diperbarui secara otomatis saat Anda membuka buku kerja pada tanggal lain.

2. Untuk memasukkan tanggal dan waktu sekarang, gunakan fungsi SEKARANG.

Catatan: fungsi SEKARANG tidak membutuhkan argumen. Waktu ini akan diperbarui secara otomatis setiap kali lembar dihitung ulang. Ini terjadi saat Anda membuat perubahan ke sel mana pun atau saat Anda membuka buku kerja. Tekan F9 untuk menghitung ulang buku kerja secara manual.

3. Untuk memasukkan waktu saat ini saja, gunakan NOW()-TODAY() dan terapkan format waktu .

Catatan: tanggal disimpan sebagai angka di Excel dan menghitung jumlah hari sejak 0, Januari 1900. Waktu ditangani secara internal sebagai angka antara 0 dan 1. Kunjungi halaman kami tentang format tanggal dan waktu untuk informasi selengkapnya.

Tanggal dan Waktu Statis

1. Untuk memasukkan tanggal saat ini sebagai nilai statis, tekan CTRL + ; (titik koma).

Catatan: tanggal ini tidak akan berubah saat Anda membuka buku kerja pada tanggal lain.

2. Untuk memasukkan waktu saat ini sebagai nilai statis, tekan CTRL + SHIFT + ; (titik koma).

Catatan: waktu ini tidak akan berubah saat Anda membuat perubahan ke sel atau saat Anda membuka buku kerja.

3. Untuk memasukkan tanggal dan waktu saat ini sebagai nilai statis, cukup tekan CTRL + ; (titik koma), masukkan spasi dan tekan CTRL + SHIFT + ; (titik koma).
