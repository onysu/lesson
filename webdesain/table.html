Sebuah tabel dalam HTML terdiri dari sel-sel tabel di dalam baris dan kolom

Sel Tabel

Setiap sel tabel didefinisikan oleh a <td> dan </td> tag.

<td> singkatan dari tabel data.

Segala sesuatu di antara <td> dan </td> adalah konten sel tabel.

Contoh :

Elemen <td> mendefinisikan sel tabel

Emil Tobias Linus

Untuk memahami contoh dengan lebih baik, kami telah menambahkan border ke tabel.


<style>
  table, th, td {
    border:1px solid black;
  }
</style>

<table>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
</table>

Cobalah sendiri

Catatan: elemen data tabel adalah wadah data tabel.
Mereka dapat berisi semua jenis elemen HTML; teks, gambar, daftar, tabel lain, dll.

Baris Tabel

Setiap baris tabel dimulai dengan <tr> dan diakhiri dengan </tr> tag.

<tr> singkatan dari table row (tabel baris).

Contoh :

Elemen <tr> mendefinisikan baris tabel

Emil Tobias Linus
16 14 10

Untuk memahami contoh dengan lebih baik, kami telah menambahkan border ke tabel.


<style>
  table, th, td {
    border:1px solid black;
  }
</style>

<table>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>

Cobalah sendiri

Anda dapat memiliki baris sebanyak yang Anda suka dalam sebuah tabel, pastikan jumlah selnya sama di setiap baris.

Catatan: Ada kalanya baris dapat memiliki lebih sedikit atau lebih banyak sel daripada yang lain. Anda akan mempelajarinya di bab selanjutnya.

Tabel Header

Terkadang Anda ingin sel Anda menjadi tajuk, dalam kasus tersebut gunakan <th> tag alih-alih <td> tag:

Contoh :

Biarkan baris pertama menjadi header tabel:

Elemen <th> mendefinisikan header tabel

Person 1 Person 2 Person 3
Emil Tobias Linus
16 14 10

Untuk memahami contoh dengan lebih baik, kami telah menambahkan border ke tabel.


<style>
  table,
  th,
  td {
    border: 1px solid black;
  }
</style>

<table>

    <th>Person 1</th>
    <th>Person 2</th>
    <th>Person 3</th>
  </tr>
  <tr>
    <td>Emil</td>
    <td>Tobias</td>
    <td>Linus</td>
  </tr>
  <tr>
    <td>16</td>
    <td>14</td>
    <td>10</td>
  </tr>
</table>

Cobalah sendiri

Secara default, teks dalam <th>elemen dicetak tebal dan di tengah, tetapi Anda dapat mengubahnya dengan CSS.

Tag Tabel HTML

Tag Keterangan
<table> Mendefinisikan tabel
<th> Mendefinisikan sel header dalam tabel
<tr> Mendefinisikan baris dalam tabel
<td> Mendefinisikan sel dalam tabel
<caption> Mendefinisikan judul tabel
<colgroup> Menentukan grup dari satu atau lebih kolom dalam tabel untuk pemformatan
<col> Menentukan properti kolom untuk setiap kolom dalam elemen <colgroup>
<thead> Mengelompokkan konten header dalam sebuah tabel
<tbody> Mengelompokkan konten tubuh dalam sebuah tabel
<tfoot> Mengelompokkan konten footer dalam tabel

Border Tabel HTML

Tabel HTML dapat memiliki batas gaya dan bentuk yang berbeda.
Cara Menambahkan Border :

Saat Anda menambahkan border ke tabel, Anda juga menambahkan border di sekitar setiap sel tabel. Untuk menambahkan border, gunakan border properti CSS pada table, th, dan td elemen.

Contoh :

     
     
      

<style>
  table,
  th,
  td {
    border: 1px solid black;
  }
</style>

<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>80</td>
  </tr>
</table>

Cobalah sendiri

Tabel Border Collapse

Untuk menghindari border ganda seperti pada contoh di atas, setel border-collapse properti CSS ke collapse. Ini akan membuat perbatasan runtuh menjadi satu perbatasan:

     
     
      

<style>
  table, th, td {
    border: 1px solid black;
    border-collapse: collapse;
  }
</style>

<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>80</td>
  </tr>
</table>
