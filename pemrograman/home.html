1. Teori
    • Apa itu PHP & sejarah singkat
    • PHP sebagai server-side language
    • Cara kerja Request → Server → PHP → Browser
    • Ekstensi file .php
    • Menjalankan PHP di local server:
      • Menggunakan XAMPP
      • Menggunakan Laragon
    • Struktur folder htdocs atau www
    • Mengakses file via http://localhost/…
  2. Praktik
    • Instalasi XAMPP atau Laragon
    • Membuat project baru
    • File: index.php
    • <?php
    • echo "Hello World!";
    • Menulis komentar
    • Output:
      • echo
      • print
  3. Latihan
    • Buat file biodata.php yang menampilkan nama, umur, alamat.

TUJUAN PEMBELAJARAN

Setelah mengikuti pertemuan ini, diharapkan siswa mampu:

  1. Menjelaskan konsep dasar pemrograman dan web server.
  2. Menginstall dan menjalankan XAMPP dengan benar.
  3. Mengetahui fungsi folder htdocs (root website di XAMPP).
  4. Membuat file .php dan menjalankannya di browser melalui localhost.
  5. Menampilkan teks, HTML, dan komentar menggunakan PHP.
