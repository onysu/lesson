HTML

HTML adalah bahasa markup standar untuk halaman Web. Dengan HTML Anda dapat membuat Website Anda sendiri. HTML mudah dipelajari - You will enjoy it!

Apa itu HTML?

HTML singkatan dari Hyper Text Markup Language. HTML adalah bahasa markup standar untuk membuat halaman Web HTML menggambarkan struktur halaman Web HTML terdiri dari serangkaian elemen Elemen HTML memberi tahu browser cara menampilkan konten Elemen HTML memberi label potongan konten seperti ini adalah judul, ini adalah paragraf, ini adalah tautan, dll.

File HTML diakhiri dengan ekstensi .html atau .htm. Ekstensi file ini bisa dilihat dengan mengunakan web browser apa pun (seperti Google Chrome, Safari, atau Mozila Firefox). Browser tersebut membaca file HTML dan merender kontennya sehingga user internet bisa melihat dan membacanya.

Sejarah HTML dimulai oleh Tim Berners-Lee, seorang ahli fisika di lembaga penelitian CERN yang berlokasi di Swiss. Versi pertamanya dirilis pada tahun 1991, dengan 18 tag. Sejak saat itu, setiap kali ada versi barunya, pasti akan selalu ada tag dan attribute (tag modifier) yang juga baru.

Berkat popularitasnya yang terus meningkat, bahasa markup ini kini dianggap sebagai standar web resmi. Spesifikasi HTML dikelola dan dikembangkan oleh World Wide Web Consortium (W3C). Berdasarkan HTML Element Reference milik Mozilla Developer Network, saat ini ada 140 tag HTML, meskipun sebagiannya sudah tidak lagi didukung oleh beberapa versi terbaru browser. Kemudian, pada tahun 2014, HTML5 mulai diperkenalkan. Dengan diperkenalkannya HTML5, terdapat semantic baru seperti <article>, <header> dan <footer> yang memperjelas bagian-bagian dalam konten.

Apa perbedaan HTML dengan HTML5?

HTML4 (dikenal sebagai HTML) dirilis pada tahun 1999, kemudian HTML5 diperkenalkan pada publik pada tahun 2014, dengan menambahkan lebih banyak fitur baru ke bahasa mark up. Salah satu fitur canggih di HTML5 adalah dukungan untuk embed audio dan video. Jadi, daripada menggunakan Flash player, kita bisa menyematkan file video dan audio ke halaman website dengan memanfaatkan tag <audio></audio> dan <video></video>. Fitur tersebut juga memiliki dukungan bawaan untuk SVG dan MathML untuk rumus matematika dan ilmiah. HTML5 juga memperkenalkan beberapa perbaikan pada semantic. Tag semantic baru mengirimkan informasi pada browser tentang arti konten, yang tak hanya bermanfaat bagi pembaca, tapi juga mesin pencari. Tag semantic yang paling banyak digunakan adalah <article></article>, <section></section>, <aside> </aside>, <header></header> dan <footer></footer>.

Kelebihan dan Kekurangan HTML

Kelebihan HTML :
  • Digunakan secara luas dan memiliki banyak sumber serta komunitas yang besar.
  • Dijalankan secara alami di setiap web browser.
  • Mudah dipelajari.
  • Open-source dan sepenuhnya gratis.
  • Rapi dan konsisten.
  • Menjadi standar resmi web, dikelola oleh (W3C).
  • Integrasi mudah dengan bahasa backend, seperti PHP dan Node.js.
Kekurangan HTML :
  • Umumnya digunakan untuk halaman web statis. Untuk fitur dinamis, Anda bisa menggunakan JavaScript atau bahasa backend, seperti PHP.
  • Tidak bisa menjalankan logic. Alhasil, semua halaman web harus dibuat terpisah meskipun menggunakan elemen yang sama, seperti header dan footer.
  • Fitur baru tidak bisa digunakan dengan cepat di sebagian browser.
  • Perilaku browser terkadang sulit diprediksi. Misalnya, browser lama tidak selalu bisa merender tag yang lebih baru.

Hubungan HTML, CSS dan JavaScript

HTML dinyatakan sebagai bahasa markup yang canggih dan mudah, bahasa ini bisa digunakan untuk menambah elemen dan membuat struktur konten. Bahasa ini juga bisa bekerja secara maksimal dengan dua bahasa frontend: CSS (Cascading Style Sheets) dan JavaScript. Jika digabungkan, kedua bahasa frontend ini bisa meningkatkan pengalaman user dan memberikan fungsi yang lebih canggih.

  • CSS erat kaitannya dengan styling, seperti background, warna, layout, spacing dan animiasi.
  • JavaScript memungkinkan Anda menambahkan fungsionalitas yang dinamis, seperti slider, pop-up, dan galeri foto.

Berdasarkan cara kerja HTML, Anda bisa menggunakannya untuk membuat struktur konten pada website dan Aplikasi Web. HTML merupakan level teknologi frontend dan berfungsi sebagai dasar styling yang bisa Anda tambahkan dengan CSS dan fungsionalitas yang dapat dijalakan menggunakan JavaScript.

Contoh Dokumen HTML Sederhana

Contoh :

<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link href="#" rel="stylesheet">
  <title>Judul Halaman</title>
</head>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>
</body>

</html>
Penjelasannya :
<!DOCTYPE html> Deklarasi mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
<html> Elemen akar (root) halaman HTML
<head> Elemen berisi informasi meta tentang halaman HTML.
<title> Elemen menentukan judul untuk halaman HTML (yang ditampilkan dalam bar judul browser atau tab halaman ini).
<body> Elemen mendefinisikan tubuh dokumen, dan merupakan wadah untuk semua isi terlihat, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll.
<h1> Elemen mendefinisikan judul.
<p> Elemen mendefinisikan sebuah paragraf.

Struktur Halaman HTML

Di bawah ini adalah visualisasi struktur halaman HTML:

<html>
<head>
<title>Judul halaman</title>
</head>
<body>
<h1>Judul halaman</h1>
<p>Ini adalah paragraf.</p>
<p>Ini adalah paragraf yang lain.</p>
</body>
</html>

Catatan: Konten di dalam bagian <body> (area putih di atas) akan ditampilkan di browser. Konten di dalam elemen <title> akan ditampilkan di bilah judul browser atau di tab halaman.

Apa itu Elemen HTML?

Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir:

Syntax :
<tagname>Konten berada di sini...</tagname>

Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:

<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>

Mulai tag Konten elemen Tag akhir
<h1> Judul pertama saya </h1>
<p> Paragraf pertama saya. </p>
<br> none none

Catatan: Beberapa elemen HTML tidak memiliki konten seperti elemen <br>, elemen ini disebut elemen kosong. Elemen kosong tidak memiliki tag akhir!.

Contoh

<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <title>Judul Halaman</title>

<body>
  <h1>Hello, world!</h1>
  <p>Ini adalah paragraf</p>

</html>

<!DOCTYPE>

deklarasi <!DOCTYPE> mewakili jenis dokumen, dan membantu browser untuk tampilan halaman web dengan benar. Itu hanya boleh muncul sekali, di bagian atas halaman (sebelum tag HTML apa pun).

<!DOCTYPE> deklarasi ini tidak case sensitive.

Nested HTML Elements (Elemen HTML Bersarang)

Elemen HTML dapat bersarang (ini berarti elemen dapat berisi elemen lain). Semua dokumen HTML terdiri dari elemen HTML bersarang. Contoh berikut berisi empat elemen HTML ( <html>, <body>, <h1> dan <p>):

Contoh :

<!DOCTYPE html>
                <html>
                <body>
                
                  <h1>Judul Pertama Saya</h1>
                  <p>Paragraf pertama saya.</p>
                
                </body>
                </html>

Penjelasannya :

  • <html> adalah elemen root elemen dan mendefinisikan dokumen HTML secara keseluruhan. Ini memiliki tag awal <html> dan tag akhir </html>. Kemudian, di dalam <html> elemen ada <body> elemen:

    <body>

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

    </body>

  • <body> mendefinisikan tubuh dokumen. Ini memiliki tag awal <body> dan tag akhir </body>. Kemudian, di dalam <body> elemen ada dua elemen lainnya: <h1> dan <p>:

    <h1>Judul Pertama Saya</h1>
    <p>Paragraf pertama saya.</p>

  • <h1> elemen mendefinisikan heading (judul). Ini memiliki tag awal <h1> dan tag akhir </h1>:

    <h1>Judul Pertama Saya</h1>

  • <p> elemen mendefinisikan sebuah paragraf. Ini memiliki tag awal <p> dan tag akhir </p>:

    <p>Paragraf pertama saya.</p>

Jangan abaikan tag akhir

Beberapa elemen HTML akan ditampilkan dengan benar, bahkan jika Anda lupa tag akhir:
Contoh : 

<html>
                <body>
                
                  <p>Ini adalah paragraf
                  <p>Ini adalah paragraf
                
                </body>
                </html>

Cobalah sendiri

Warning!!!, jangan pernah mengandalkan ini! Hasil dan kesalahan yang tidak terduga dapat terjadi jika Anda lupa tag akhir!

<img> tag digunakan untuk menanamkan gambar di halaman HTML. src menspesifikasikan atribut path ke gambar yang akan ditampilkan:

Contoh :

Cobalah sendiri

Ada dua cara untuk menentukan URL dalam src atribut:

Atribut Keterangan
Absolute URL link ke gambar eksternal yang dihosting di situs web lain. Contoh: src="https://onysu.github.io/lesson/assets/img/icons/html5.png".

Catatan: Gambar eksternal mungkin berada di bawah hak cipta. Jika Anda tidak mendapatkan izin untuk menggunakannya, Anda mungkin melanggar undang-undang hak cipta. Selain itu, Anda tidak dapat mengontrol gambar eksternal; itu bisa tiba-tiba dihapus atau diubah.
Relative URL link ke gambar yang dihosting di dalam situs web. Di sini, URL tidak menyertakan nama domain. Jika URL dimulai tanpa garis miring, itu akan relatif terhadap halaman saat ini. Contoh: src="img_girl.jpg. Jika URL dimulai dengan garis miring, itu akan menjadi relatif terhadap domain. Contoh: src="img_girl.jpg.

Tip: Hampir selalu yang terbaik adalah menggunakan URL relatif. Mereka tidak akan rusak jika Anda mengubah domain.

lang atribut

Kode Bahasa

Anda harus selalu menyertakan lang atribut di dalam <html> tag, untuk mendeklarasikan bahasa halaman Web. Ini dimaksudkan untuk membantu mesin pencari dan browser.

Contoh berikut menentukan bahasa Inggris sebagai bahasa:

<!DOCTYPE html>
                <html lang="en">
                <body>
                ...
                </body>
                </html>

Kode Bahasa ISO 639-1 mendefinisikan singkatan untuk bahasa. Dalam HTML dan XHTML mereka dapat digunakan dalam lang dan xml:lang atribut.

Daftar Kode Bahasa

Kode Negara

Kode Negara ISO mendefinisikan singkatan untuk negara. Dalam HTML mereka dapat digunakan sebagai tambahan nilai bahasa di lang atribut. Dua karakter pertama dari kode bahasa mendefinisikan bahasa. dua terakhir mendefinisikan Negara.

Contoh berikut menentukan bahasa Inggris sebagai bahasa dan Amerika Serikat sebagai negara:

<!DOCTYPE html>
                <html lang="en-US">
                <body>
                ...
                </body>
                </html>

Kutipan Tunggal atau Ganda?

Tanda kutip ganda   di sekitar nilai atribut adalah yang paling umum dalam HTML, tetapi tanda kutip tunggal '' juga dapat digunakan. Dalam beberapa situasi, ketika nilai atribut itu sendiri berisi tanda kutip ganda, perlu menggunakan tanda kutip tunggal:
