Pengantar

Microsoft Excel adalah salah satu aplikasi perangkat lunak yang paling banyak digunakan sepanjang masa. Ratusan juta orang di seluruh dunia menggunakan Microsoft Excel. Anda dapat menggunakan Excel untuk memasukkan semua jenis data dan melakukan perhitungan keuangan, matematis, atau statistik.

Range di Excel adalah kumpulan dari dua atau lebih sel. Bab ini memberikan gambaran umum tentang beberapa operasi jangkauan yang sangat penting.

(Rumus dan Fungsi) : Rumus adalah ekspresi yang menghitung nilai sel. Fungsi adalah rumus yang telah ditentukan sebelumnya dan sudah tersedia di Excel.
Dasar

Microsoft Excel adalah salah satu aplikasi perangkat lunak yang paling banyak digunakan sepanjang masa. Ratusan juta orang di seluruh dunia menggunakan Microsoft Excel. Anda dapat menggunakan Excel untuk memasukkan semua jenis data dan melakukan perhitungan keuangan, matematis, atau statistik.

Excel memilih tab Beranda pita saat Anda membukanya. Pelajari cara menggunakan Ribbon.

Workbook adalah kata lain untuk file Excel Anda. Saat Anda memulai Excel, klik Buku kerja kosong untuk membuat buku kerja Excel dari awal.

Lembar kerja adalah kumpulan sel tempat Anda menyimpan dan memanipulasi data. Setiap buku kerja Excel dapat berisi beberapa lembar kerja.

Saat kita memformat sel di Excel, kita mengubah tampilan angka tanpa mengubah angka itu sendiri.

Pelajari cara menggunakan fitur Find, Replace and Go To Special di Excel.

Daripada membuat buku kerja Excel dari awal, Anda bisa membuat buku kerja berdasarkan templat. Ada banyak template gratis yang tersedia, menunggu untuk digunakan.

Gunakan validasi data di Excel untuk memastikan bahwa pengguna memasukkan nilai tertentu ke dalam sel.

Pintasan keyboard memungkinkan Anda melakukan berbagai hal dengan keyboard alih-alih mouse untuk meningkatkan kecepatan.

Bab ini mengajarkan Anda cara mencetak lembar kerja dan cara mengubah beberapa pengaturan cetak penting di Excel.

Pelajari cara berbagi data Excel dengan dokumen Word dan file lainnya.

Mengenkripsi file Excel dengan password sehingga membutuhkan password untuk membukanya.
