Dokumen HTML Sederhana

Contoh :

<!doctype html>
<html
  <head>    
    <title>Judul Halaman</title>
  </head>
  <body>
          
    <h1>Ini adalah judul</h1>
    <p>Ini adalah paragraf</p>
          
  </body>
</html>
Penjelasannya :
<!DOCTYPE html> Deklarasi mendefinisikan bahwa dokumen ini adalah dokumen HTML5.
<html> Elemen akar (root) halaman HTML
<head> Elemen berisi informasi meta tentang halaman HTML.
<title> Elemen menentukan judul untuk halaman HTML (yang ditampilkan dalam bar judul browser atau tab halaman ini).
<body> Elemen mendefinisikan tubuh dokumen, dan merupakan wadah untuk semua isi terlihat, seperti judul, paragraf, gambar, hyperlink, tabel, daftar, dll.
<h1> Elemen mendefinisikan judul.
<p> Elemen mendefinisikan sebuah paragraf.

Apa itu Elemen HTML?

Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir:

<tagname>Konten berada di sini...</tagname>

Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:

<h1>Judul Pertama Saya</h1>
<p>Paragraf pertama saya.</p>

Mulai tag Konten elemen Tag akhir
<h1> Judul pertama saya </h1>
<p> Paragraf pertama saya. </p>
<br> none none

Catatan: Beberapa elemen HTML tidak memiliki konten (seperti elemen <br>). Elemen-elemen ini disebut elemen kosong. Elemen kosong tidak memiliki tag akhir!

Peramban Web

Tujuan browser web (Chrome, Edge, Firefox, Safari) adalah untuk membaca dokumen HTML dan menampilkannya dengan benar. Browser tidak menampilkan tag HTML, tetapi menggunakannya untuk menentukan cara menampilkan dokumen:

Struktur Halaman HTML

Di bawah ini adalah visualisasi struktur halaman HTML:

<html>
<head>
<title>Judul halaman</title>
</head>
<body>
<h1>Judul halaman</h1>
<p>Ini adalah paragraf.</p>
<p>Ini adalah paragraf yang lain.</p>
</body>
</html>

Catatan: Konten di dalam bagian <body> (area putih di atas) akan ditampilkan di browser. Konten di dalam elemen <title> akan ditampilkan di bilah judul browser atau di tab halaman.

Editor HTML

Anda hanya perlu editor teks sederhana untuk mempelajari HTML.

Halaman web dapat dibuat dan dimodifikasi dengan menggunakan editor HTML profesional. Seperti Visual Studio Code yang bisa Anda download pada Link resminya.

Atau dengan Code Editor sederhana seperti Notepad bawaan dari Windows.

Belajar menggunakan Notepad :

Ikuti langkah-langkah di bawah ini untuk membuat halaman web pertama Anda dengan Notepad :

  • Langkah 1: Buka Notepad

    Windows 8 atau lebih baru:
    Buka Layar Mulai (simbol jendela di kiri bawah layar Anda). Ketik Notepad.
    Windows 7 atau sebelumnya:
    Buka Start > Programs > Accessories > Notepad

  • Langkah 2: Tulis Beberapa HTML

    Tulis atau salin kode HTML berikut ke Notepad:

    <!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Judul Halaman</title>
  </head>
  <body>
          
    <h1>Hello, world!</h1>
    <p>Ini adalah paragraf</p>
          
  </body>
</html>

  • Langkah 3: Simpan Halaman HTML

    Save As di menu Notepad. Beri nama file "index.htm" dan atur pengkodean ke UTF-8 (yang merupakan pengkodean pilihan untuk file HTML).

    Tip: Anda dapat menggunakan .htm atau .html sebagai ekstensi file. Tidak ada perbedaan, terserah Anda.

  • Langkah 4: Lihat Halaman HTML di Browser

    Buka file HTML yang disimpan di browser favorit Anda (klik dua kali pada file, atau klik kanan - dan pilih Open with).
